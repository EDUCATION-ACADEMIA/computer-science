Select all the types of secondary storage

(A) RAM

(B) ROM

(C) ** Hard Disk Drive

(D) ** Cloud Storage

(E) ** Solid State Drive

EXPLANATIONS BELOW

Concept note-1: -A solid-state drive (SSD) is a solid-state storage device that uses integrated circuit assemblies to store data persistently, typically using flash memory, and functioning as secondary storage in the hierarchy of computer storage.

Concept note-2: -magnetic storage devices, such as hard disk drives. optical storage devices, such as CD, DVD and Blu-ray discs. solid state storage devices, such as solid state drives and USB memory sticks.

Concept note-3: -Typical secondary storage devices include magnetic tape drives, magnetic disk drives, compact optical disk drives, and archival storage devices such as disk jukeboxes. Figure 1.6.

Concept note-4: -Device types Solid-state storage devices serve as secondary storage components for more complex systems, which may range from embedded and portable devices to large servers and dedicated network-attached storage (NAS) systems.
