What does the term POST stand for?Lesson 8-1

(A) power-on start-up test

(B) power-on system test

(C) power-on software test

(D) ** power-on self test

EXPLANATIONS BELOW

Concept note-1: -A Power-On Self-Test (POST) is an operation initiated by a computer after it has been turned on but before it boots up the OS.

Concept note-2: -A power-on self-test (POST) is a set of routines performed by firmware or software immediately after a computer is powered on, to determine if the hardware is working as expected.

Concept note-3: -The system BIOS provides a basic power-on self-test (POST), during which the BIOS checks the basic devices required for the server to operate.

Concept note-4: -No Post Issues. The term POST refers to Power-On Self-Test, which is a series of checks the computer goes through whenever it starts. If the computer fails any of these tests, it stops the start-up process and report a fault.

Concept note-5: -When the computer is first turned on, all the components must be tested and initialized by the BIOS, and the operating system must be loaded into memory. This is commonly referred to as POST (Power-On Self-Test).
