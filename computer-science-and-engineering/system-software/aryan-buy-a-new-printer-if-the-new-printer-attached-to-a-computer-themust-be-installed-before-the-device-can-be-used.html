Aryan buy a new printer. If the new printer attached to a computer, the ____ must be installed before the device can be used.

(A) ** driver

(B) kernel

(C) platform

(D) manager

EXPLANATIONS BELOW

Concept note-1: -Plug and Play means the operating system automatically configures new devices as you install them.

Concept note-2: -Device driver – a software program that is designed to control a particular hardware device that is attached to a computer. The main purpose of device driver – it acts as a translator between the hardware device and operating systems or applications that use it.

Concept note-3: -A device driver is a specialized software that operates a particular computer-connected device-offering a software interface to the hardware allows operating systems and other computer applications to access hardware functionalities.

Concept note-4: -A multitasking is an operating system allows a single user to work on two or more programs that reside in memory at the same time. Using the example just cited, if you are working with a multitasking operating system, you do not have to quit the graphics program to run the email program.
