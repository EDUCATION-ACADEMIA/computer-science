How can we add trusted sites? (KLO 1.3.5)

(A) Firewall

(B) Router settings

(C) ** Browser

(D) we can not

EXPLANATIONS BELOW

Concept note-1: -Click the Chrome Menu icon on the far right of the Address bar. Click on Settings, scroll to the bottom and click the Show Advanced Settings link. Click on Change proxy settings (under Network) Click the Security tab > Trusted Sites icon, then click Sites. Enter the URL of your Trusted Site, then click Add.

Concept note-2: -To set trusted sites via GPO Open the Group Policy Management Editor. Go to User Configuration > Policies > Administrative Templates > Windows Components > Internet Explorer > Internet Control Panel > Security Page. Select the Site to Zone Assignment List. Select Enabled and click Show to edit the list.

Concept note-3: -Confirm the website has been added to the “Websites” list and select the “Close” button. Note: If the “Add this website to the zone:” is “grayed” out, this means the setting is not available to the user. You must contact your IT Department and request an IT admin to update this setting.
