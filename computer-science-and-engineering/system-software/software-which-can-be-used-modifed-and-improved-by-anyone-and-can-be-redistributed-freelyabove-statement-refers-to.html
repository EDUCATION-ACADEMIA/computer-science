COMPUTER SCIENCE AND ENGINEERING

SYSTEM SOFTWARE

[SOURCES]
“Software which can be used, modifed and improved by anyone and can be redistributed freely"Above statement refers to:

(A) Proprietary System

(B) ** Open source system

(C) Freeware System

(D) Shareware System

EXPLANATIONS BELOW

Concept note-1: -What is “Open Source” software? Generally, Open Source software is software that can be freely accessed, used, changed, and shared (in modified or unmodified form) by anyone. Open source software is made by many people, and distributed under licenses that comply with the Open Source Definition.

Concept note-2: -Open source software refers to computer code that is distributed under a license in which the copyright holder grants users the rights to freely access, modify, and distribute the software for any purpose.

Concept note-3: -Public domain. However, copyright does not apply to works in the public domain-anyone can modify and use such software without any restrictions.

Concept note-4: -Free redistribution: The license shall not restrict any party from selling or giving away the software as a component of an aggregate software distribution containing programs from several different sources. The license shall not require a royalty or other fee for such sale.

Concept note-5: -Open source software is code that is designed to be publicly accessible-anyone can see, modify, and distribute the code as they see fit. Open source software is developed in a decentralized and collaborative way, relying on peer review and community production.
