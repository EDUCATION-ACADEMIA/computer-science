First 32-bit version of Windows designed for high-end workstations and servers.

(A) ** Windows NT

(B) Windows ME

EXPLANATIONS BELOW

Concept note-1: -Windows 95 was the first 32-bit Windows operating system and a major upgrade from Windows 3.1. It used an entirely different user interface that incorporated the now-common Start menu and Taskbar.

Concept note-2: -The first version of Windows NT was Windows NT 3.1 and was produced for workstations and server computers. It was a commercially focused operating system intended to complement consumer versions of Windows that were based on MS-DOS (including Windows 1.0 through Windows 3.1x).

Concept note-3: -NT was the first purely 32-bit version of Windows, whereas its consumer-oriented counterparts, Windows 3.1x and Windows 9x, were 16-bit/32-bit hybrids. It is a multi-architecture operating system.

Concept note-4: -The first true 32-bit operating system developed by Microsoft was Windows 3.1. Windows 95 used cooperative multitasking for 16-bit applications and preemptive multitasking for 32-bit applications. Windows 98 was the first Windows OS to use the Kerberos security system.

Concept note-5: -The correct answer is Operating system.
