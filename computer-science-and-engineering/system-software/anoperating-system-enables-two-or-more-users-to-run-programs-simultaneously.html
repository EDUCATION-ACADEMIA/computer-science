COMPUTER SCIENCE AND ENGINEERING

SYSTEM SOFTWARE

[SOURCES]
A(n) ____ operating system enables two or more users to run programs simultaneously.

(A) multiplied

(B) ** multiuser

(C) engaged

(D) rasterized

EXPLANATIONS BELOW

Concept note-1: -A multi-user operating system is an operating system that allows multiple users to access underlying hardware resources simultaneously. Multiple users access the operating system with the help of many terminals connected to a network. The objective of a multi-user operating system is time-sharing and batch processing.

Concept note-2: -A multiuser operating system enables two or more users to run programs simultaneously. Networks, servers, mainframes, and supercomputers allow hundreds to thousands of users to connect at the same time, and thus are multiuser.

Concept note-3: -UNIX is a multi-user, multitasking operating system, and was designed to be a small, flexible system used by computer programmers.

Concept note-4: -Multi-programming OS The operating system which can run multiple processes on a single processor is called a multiprogramming operating system.
