What lets you do many things, like write book reports and stories?

(A) Antivirus Software

(B) Email

(C) Duct Tape

(D) ** Application Programs

EXPLANATIONS BELOW

Concept note-1: -For example, Microsoft Word or Google Docs are both word processing programs that let you write book reports, essays, stories, and more.

Concept note-2: -Application software includes all the programs that allow you to perform specific tasks on a computer, such as writing a letter, preparing an invoice, viewing a Web page, listening to a music file, checking the inventory of a particular product, playing a game, preparing financial statements, and designing a home.

Concept note-3: -An operating system is the most important software that runs on a computer. It manages the computer’s memory and processes, as well as all of its software and hardware. It also allows you to communicate with the computer without knowing how to speak the computer’s language.

Concept note-4: -Operating system (OS) – a software that controls and coordinates the computer hardware devices and runs other software and applications on a computer. It is the main part of system software and a computer will not function without it.

Concept note-5: -The Web operates within the Internet’s basic client-server format; servers are computer programs that store and transmit documents to other computers on the network when asked to, while clients are programs that request documents from a server as the user asks for them.
