Question 8:Windows, Icons, menus and Pointers are all part of this? (1-4)

(A) ** Graphical User Interface

(B) Command Line Interface

(C) Menu Interface

(D) No User Interface

EXPLANATIONS BELOW

Concept note-1: -WIMP stands for Windows, Icons, Menus and Pointers (or maybe Windows, Icons, Mouse, Pull-down menus). WIMP is the style of graphical user interface that uses the above-mentioned common widgets.

Concept note-2: -Windows, icons, menus and pointing device (WIMP) denotes a style of computer-human interaction involving the aforementioned elements of the graphical user interface (GUI) which is the most common interaction method being used by desktop computers today.

Concept note-3: -A graphics-based operating system interface that uses icons, menus and a mouse (to click on the icon or pull down the menus) to manage interaction with the system. Developed by Xerox, the GUI was popularized by the Apple Macintosh in the 1980s.

Concept note-4: -Windows uses icons to graphically represent items, such as programs and folders. This is part of Windows GUI, or graphical user interface. Icons appear everywhere throughout Windows, and are really just pictures that depict the type of items they represent. The Windows desktop contains a range of desktop icons.
