Exporting is when:

(A) Refers to the process of converting (extracting) a compressed file back to the original size

(B) File Explorer allows you to search for a file based on certain criteria:file size, type of file, partial file name, date accessed and even text in file

(C) ** Saving data om a format that can be read by different version of same application or new application. Menu option ‘Export’ or ‘Save As’.

(D) Opening or ‘reading’ data created in different version of the same application or new application.

EXPLANATIONS BELOW

Concept note-1: -Data export is the extraction and conversion of raw data from their existing format into a format required by another application. Exporting data is also a way of backing up data or moving it between two different versions of programs.

Concept note-2: -save: To save is to commit changes to a permanent state in a format that the application can natively utilize(keeps the file format). To export is to change the data format so that another application can utilize it. (changes the format for example from png to jpg).

Concept note-3: -Go to File > Save As. Choose where you want to save your file. Tip: To go directly to the Save As dialog box, double-click a location (for example, double-click Computer). In the Save As dialog box, under Save as type, select the file format you want.
