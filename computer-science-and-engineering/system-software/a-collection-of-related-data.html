A collection of related data

(A) Data Type

(B) Information

(C) Blog

(D) ** Database

EXPLANATIONS BELOW

Concept note-1: -A collection of related data is called Database. A database is a data structure that stores organized information. Most databases contain multiple tables, which may each include several different fields.

Concept note-2: -The sum total of the database, the DBMS and the associated applications can be referred to as a database system. Often the term “database” is also used loosely to refer to any of the DBMS, the database system or an application associated with the database.

Concept note-3: -You can describe a database as an organised collection of related information (or data) that is stored in a computer system. You can then, in an easy way, access, manage and update that data. Usually, a database is controlled by a Database Management System, also known as a DBSM.
