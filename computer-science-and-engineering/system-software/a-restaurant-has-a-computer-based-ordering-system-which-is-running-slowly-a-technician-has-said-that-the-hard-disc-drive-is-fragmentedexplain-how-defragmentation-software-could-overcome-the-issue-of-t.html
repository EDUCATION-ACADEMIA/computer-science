A restaurant has a computer-based ordering system which is running slowly. A technician has said that the hard disc drive is fragmented.Explain how defragmentation software could overcome the issue of the slow computer system.

(A) ** Files on the hard disc drive are moved (1)

(B) ** Empty spaces collected together (1)

(C) ** Files are moved to be stored together (1)

(D) ** Fewer disc accesses are needed (1)

(E) More disc access are needed (1)

EXPLANATIONS BELOW

Concept note-1: -The purpose of fragmentation is to help an operating system use the available space on a storage device. In the absence of disk fragmentation, application and file storage need to continuously be updating and moving storage blocks, in order to maintain a continuous, non-fragmented file system.

Concept note-2: -Disk fragmentation occurs when files or pieces of files get scattered throughout your disks. Not only do hard disks get fragmented, but removable storage can also become fragmented. This can cause poor disk performance and overall system degradation.

Concept note-3: -Disk Defragmenter is a Windows utility program that located and eliminates unnecessary fragments and rearranges files and unused to disk space to optimise operations.
