The ability to use files on differing operating systems is called ____ Lesson 8-2

(A) cross-platform compliance

(B) cross-platform coexistence

(C) cross-platform communication

(D) ** cross-platform compatibility

EXPLANATIONS BELOW

Concept note-1: -interoperability or platform-independence; the ability for devices or software applications to work with more than one hardware platform or operating system. Broad compatibility with various systems is critical to having a large market for a product (reaching your entire user audience).

Concept note-2: -able to be used with different types of computer systems: The software is truly cross-platform and is available for computers running the Windows, Mac OS X, Linux, and Solaris operating systems.

Concept note-3: -The ability of an operating system to run more than one application at a time is called multitasking. A multitasking operating system (also called multiprocessing operating system) supports two or more active processes simultaneously.

Concept note-4: -Virtualization is the process of running a virtual instance of a computer system in a layer abstracted from the actual hardware. Most commonly, it refers to running multiple operating systems on a computer system simultaneously.
