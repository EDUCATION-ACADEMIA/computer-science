SYSTEM SOFTWARE

[SOURCES]
The operating system and utility programs that control a computer system and allow you to use your computer.

(A) ** System Software

(B) Application Software

EXPLANATIONS BELOW

Concept note-1: -The OS manages all the other programs in a computer. System software is used to manage the computer itself. It runs in the background, maintaining the computer’s basic functions so users can run higher-level application software to perform certain tasks.

Concept note-2: -System software consists of the operating system and utility programs that control a computer system and allow you to use a computer.

Concept note-3: -A computer’s operating system is an example of system software.

Concept note-4: -Operating system (OS) – a software that controls and coordinates the computer hardware devices and runs other software and applications on a computer. It is the main part of system software and a computer will not function without it.

Concept note-5: -Utility software helps users configure, analyse, optimise and maintain a computer. This software normally consists of small programs which are thought of as part of the operating system (OS) as they often come built in with the OS.
