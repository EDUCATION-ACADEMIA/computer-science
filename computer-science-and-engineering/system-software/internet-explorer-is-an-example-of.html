SYSTEM SOFTWARE

[SOURCES]
Internet Explorer is an example of?

(A) System Software

(B) ** Application Software

EXPLANATIONS BELOW

Concept note-1: -A web browser, also known as a “browser, ” is an application software that allows users to find, access, display, and view websites. Microsoft Internet Explorer, Google Chrome, Mozilla Firefox, and Apple Safari are all popular web browsers.

Concept note-2: -Internet Explorer and Safari are examples of internet/web browsers.

Concept note-3: -Microsoft Windows Explorer is a graphical file management utility for the Windows operating system (OS). Windows Explorer was introduced with Windows 95 and later renamed File Explorer. Each time that a computer user accesses a disk or opens a folder containing files, they are using the Windows/File Explorer utility.

Concept note-4: -A web browser is an application for accessing websites. When a user requests a web page from a particular website, the browser retrieves its files from a web server and then displays the page on the user’s screen. Browsers are used on a range of devices, including desktops, laptops, tablets, and smartphones.
