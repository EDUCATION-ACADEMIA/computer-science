A hard disk drive contains

(A) disks

(B) ** platters

(C) CDs

(D) no moving parts

EXPLANATIONS BELOW

Concept note-1: -Most basic hard drives consist of several disk platters–a circular disk made of either aluminum, glass or ceramic–that are positioned around a spindle inside a sealed chamber. The platter spins with a motor that is connected to the spindle.

Concept note-2: -The number of platters in a hard drive depend on the physical size of the hard drive, its capacity, how many sectors, and its manufacturer. Therefore there is no set number of platters in a computer hard drive. However, all modern hard drives have two or more platters and if the drive is an SSD, it has none.

Concept note-3: -A platter is a circular magnetic plate that is used for storing data in a hard disk. It is often made of aluminum, glass substrate or ceramic. A hard disk drive contains several platters that are mounted on the same spindle.

Concept note-4: -Hard drives typically have several platters which are mounted on the same spindle. A platter can store information on both sides, typically requiring two recording heads per platter, one per surface.
