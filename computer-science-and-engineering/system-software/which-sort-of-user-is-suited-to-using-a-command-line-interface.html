Which sort of user is suited to using a command-line interface?

(A) Software Tester

(B) ** Network Technician

(C) Programmer

(D) Web Developer

EXPLANATIONS BELOW

Concept note-1: -Programmers, experienced computer users, or administrators may utilise a command line interface. There are specific situations when typing text into the interface will provide faster results than simply using a GUI. Plus, CLIs can offer greater control over an operating system via succinct commands.

Concept note-2: -The command line remains an important tool for IT professionals, software developers, sys admins, network administrators and many others who prefer a more precise and reproduceable interface to their systems.

Concept note-3: -The various types of user interfaces include: graphical user interface (GUI) command line interface (CLI) menu-driven user interface.
