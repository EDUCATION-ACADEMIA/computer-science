A utility software is used to ____

(A) ** Maintain a computer

(B) Provides options to manage your files and folders

(C) Allows the os to manage the hardware devices

(D) Allows each user to have different accounts on the system

EXPLANATIONS BELOW

Concept note-1: -A computer’s utility software includes components that assist the operating system to optimize, maintain, organize and manage how it functions in all situations. This type of software also supports the computer’s infrastructure, which differs from application software that performs tasks to benefit the user.

Concept note-2: -Utility software helps users configure, analyse, optimise and maintain a computer. This software normally consists of small programs which are thought of as part of the operating system (OS) as they often come built in with the OS.

Concept note-3: -Utility software Utility software is part of the system software and performs specific tasks to keep the computer running.

Concept note-4: -Utility software are programs designed to perform maintenance work on a computer system, so that the system can function smoothly.
