COMPUTER SCIENCE AND ENGINEERING

SYSTEM SOFTWARE

[SOURCES]
Application software performs a useful finction.

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -Application software is used to perform any operation. This type of software can be used by the user if it is installed on the local machine on any system software.

Concept note-2: -The sole purpose of application software is to aid the user in doing specified tasks. Web browsers like Firefox, and Google Chrome, as well as Microsoft Word and Excel, are examples of application software that is used on a personal computer or laptop.

Concept note-3: -Application software, also known as an application or an app, is computer software designed to help the user to perform specific tasks. Examples include enterprise software, accounting software, office suites, graphics software and media players. Many application programs deal principally with documents.

Concept note-4: -Application software are those software which are used for the particular task and runs under the system software . Microsoft office is one of the example of application software . Without the system software application software will never run that’s why the option(b) is correct option.

Concept note-5: -Answer: ‘Operating system is application software’ is a FALSE statement. Explanation: Software can be categorized into two types namely ‘system software’ and application software.
