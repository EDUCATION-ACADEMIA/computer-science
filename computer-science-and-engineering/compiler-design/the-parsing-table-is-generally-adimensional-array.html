The parsing table is generally a ____ dimensional array.

(A) one

(B) ** two

(C) three

(D) four

EXPLANATIONS BELOW

Concept note-1: -LR Parsing Tables are a two-dimensional array in which each entry represents an Action or goto entry. A programming language grammar having a large number of productions has a large number of states or items, i.e., I0, I1 … … In.

Concept note-2: -A compiler maintains two types of symbol tables: a global symbol table which can be accessed by all the procedures and scope symbol tables that are created for each scope in the program.
