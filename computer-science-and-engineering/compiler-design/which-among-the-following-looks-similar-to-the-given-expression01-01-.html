Concept note-1: -Every regular expression over 0, 1 is itself a string over the 8-symbol alphabet 0, 1, +, *, (, ), , Ø. regular expression is a base-9 representation of a unique integer.

Concept note-2: -Which of the following does not represents the given language? Explanation: The given option represents 0, 01 in different forms using set operations and Regular Expressions. The operator like ^, v, etc. are logical operation and they form invalid regular expressions when used.
