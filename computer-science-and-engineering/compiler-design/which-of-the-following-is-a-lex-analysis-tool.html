which of the following is a lex analysis tool

(A) lex

(B) flex

(C) jflex

(D) ** all the above

EXPLANATIONS BELOW

Concept note-1: -Explanation: oolex, flex, lex, jflex, all are lexical analyzer tools which perform the following function. Explanation: Lexical analysis or scanning is the process of parsing the source code into proper syntactic classes.

Concept note-2: -It is a tool or software which automatically generates a lexical analyzer (finite Automata). It takes as its input a LEX source program and produces lexical Analyzer as its output. Lexical Analyzer will convert the input string entered by the user into tokens as its output.

Concept note-3: -FLEX (fast lexical analyzer generator) is a tool/computer program for generating lexical analyzers (scanners or lexers) written by Vern Paxson in C around 1987.

Concept note-4: -Lexical Analysis is the first phase of compiler also known as scanner. It converts the High level input program into a sequence of Tokens.

Concept note-5: -Explanation: A lexical analyzer coverts character sequences to set of tokens.
