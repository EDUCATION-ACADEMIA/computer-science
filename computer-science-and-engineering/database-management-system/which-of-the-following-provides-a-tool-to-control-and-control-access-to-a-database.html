(D) ** The database can be restored when the server station is down and no longer contains student data for the whole school.

EXPLANATIONS BELOW

Concept note-1: -A database management system (DBMS) is system software for creating and managing databases. A DBMS makes it possible for end users to create, protect, read, update and delete data in a database.

Concept note-2: -Introduction to Database Security Oracle uses schemas and security domains to control access to data and to restrict the use of various database resources.

Concept note-3: -Concurrent Execution in DBMS In a multi-user system, multiple users can access and use the same database at one time, which is known as the concurrent execution of the database. It means that the same database is executed simultaneously on a multi-user system by different users.
