The communication line between CPU memory and peripherals is called a ____ (UBI Clerk 2011)

(A) ** Bus

(B) line

(C) media

(D) All the above

EXPLANATIONS BELOW

Concept note-1: -The communication line between the CPU, memory and peripherals is called a. Bus. line.

Concept note-2: -The communication channel between the processor and the peripherals is called a bus.

Concept note-3: -To communicate with a particular device, the processor places a device address on address lines. Each Interface decodes the address and control received from the I/O bus, interprets them for peripherals and provides signals for the peripheral controller.

Concept note-4: -The CPU and RAM of a computer communicate using system of wires, called a bus. This bus usually has three parts, the address bus, the control bus, and the data bus.

Concept note-5: -External devices are not connected directly to the system bus because they have a wide range of control logics, as well as data transfer speeds and formats.
