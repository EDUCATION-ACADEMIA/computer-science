Consider a direct mapped cache of size 32KB with block size 32 bytes. The CPU generates 32 bit addresses. The number of bits needed for cache indexing and the number of tag bits respectively are ____ . ( GATE 2005 )

(A) ** 10, 17

(B) 10, 22

(C) 15, 17

(D) 5, 15

EXPLANATIONS BELOW

Concept note-1: -Number of blocks = cache size/block size = 32KB/32 = 1024 bytes. So, indexing requires 10 bits. Number of OFFSET bits required to access 32 bit block = 5. So, number of TAG bits = 32-10-5 = 17.
