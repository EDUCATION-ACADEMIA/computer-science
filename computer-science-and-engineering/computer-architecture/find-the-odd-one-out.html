Find the odd one out

(A) BCD

(B) XS3

(C) Gray Code

(D) ** Parity Bit

EXPLANATIONS BELOW

Concept note-1: -If the total number of ones in the data plus the parity bit is an odd number of ones, it is called odd parity. If the data already has an odd number of ones, the value of the added parity bit is 0, otherwise it is 1. Parity bits are the simplest form of error detection.

Concept note-2: -odd parity bit. There are two kinds of parity bits: In even parity, the number of bits with a value of one are counted. If that number is odd, the parity bit value is set to one to make the total number of ones in the set (including the parity bit) an even number.

Concept note-3: -It is simpler to count the number of 1s and make them even by adding a 0 or a 1. Here the number of ones in given data unit is 4. So, the parity bit 0 and codeword is 01001011. Therefore, the parity bit for 1001011 is 0.

Concept note-4: -The value of the parity bit depends on the initial parity of the data. For example, the binary value 10000000 has an odd parity. Therefore, a 0 would be added to keep the parity odd and a 1 would be added to give the value an even parity.
