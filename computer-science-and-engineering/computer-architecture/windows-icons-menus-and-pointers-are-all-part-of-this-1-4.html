Windows, Icons, menus and Pointers are all part of this? (1-4)

(A) ** Graphical User Interface

(B) Command Line Interface

(C) Menu Interface

(D) No User Interface

EXPLANATIONS BELOW

Concept note-1: -These are essentially user interfaces that are graphical in nature. They are known as Graphical User Interfaces (GUI) or WIMP interfaces (Windows, Icons, Menus and Pointer).

Concept note-2: -WIMP stands for Windows, Icons, Menus and Pointers (or maybe Windows, Icons, Mouse, Pull-down menus). WIMP is the style of graphical user interface that uses the above-mentioned common widgets.

Concept note-3: -Windows, icons, menus and pointing device (WIMP) denotes a style of computer-human interaction involving the aforementioned elements of the graphical user interface (GUI) which is the most common interaction method being used by desktop computers today.

Concept note-4: -Eg: Windows has GUI. CUI: CUI stands for Character User Interface. This is a type of user interface where user interacts with computer using only keyboard.

Concept note-5: -GUI examples include computer monitors, smartphones, tablets, gaming systems, or other consoles.
