Generally ____ is used to hold the data temporarily.

(A) ROMs

(B) EPROMs

(C) ** Buffers

(D) Magnetic disks

EXPLANATIONS BELOW

Concept note-1: -Explanation: Buffer memory is a temporary storage region in main memory (RAM) that holds data while it is being transferred between two or more devices or between an application and a device.

Concept note-2: -Caches are used to store temporary files, using hardware and software components. An example of a hardware cache is a CPU cache. This is a small chunk of memory on the computerâ€™s processor used to store basic computer instructions that were recently used or are frequently used.

Concept note-3: -Buffer memory is a temporary storage area in the main memory (RAM) that stores data transferring between two or more devices or between an application and a device. Buffering compensates for the difference in transfer speeds between the sender and receiver of the data.

Concept note-4: -computer memory, device that is used to store data or programs (sequences of instructions) on a temporary or permanent basis for use in an electronic digital computer.
