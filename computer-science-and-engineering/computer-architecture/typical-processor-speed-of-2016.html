COMPUTER ARCHITECTURE

[SOURCES]
Typical processor speed of 2016

(A) 16Hz

(B) 100MHz

(C) ** 4GHz

(D) 300GHz

EXPLANATIONS BELOW

Concept note-1: -Processor speed measures (in megahertz or gigahertz; MHz or GHz) the number of instructions per second the computer executes. The need for speed is most evident for schools that offer advanced computing classes including web design, animation and graphic design.

Concept note-2: -The clock speed measures the number of cycles your CPU executes per second, measured in GHz (gigahertz).

Concept note-3: -A good processor speed is between 3.50 to 4.2 GHz, but it is more important to have a single-thread performance. In short, 3.5 to 4.2 GHz is a good speed for processor.

Concept note-4: -A more powerful processor will help with tasks such as streaming or running multiple programs. At the same time, large amounts of RAM will help with multitasking but will primarily improve performance in complex programs and operations.
