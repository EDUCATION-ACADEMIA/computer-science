The Accumulator is a type of what?

(A) ** Register

(B) Buffer

(C) RAM

(D) Cache

EXPLANATIONS BELOW

Concept note-1: -An accumulator is a type of register for short-term, intermediate storage of arithmetic and logic data in a computerâ€™s central processing unit (CPU). However, the term is rarely used in reference to contemporary CPUs, having been replaced around the turn of the millennium by the term register.

Concept note-2: -Therefore, most of the instructions in the instruction set refer to the accumulator (e.g., the 6502 is an accumulator-based microprocessor). The purpose of all the microprocessors internal registers of 6502 is predefined by the manufacturer. Therefore, these are special purpose registers.

Concept note-3: -The Accumulator (AC) register is a general purpose processing register. The instruction read from memory is placed in the Instruction register (IR). The Temporary Register (TR) is used for holding the temporary data during the processing. The Input Registers (IR) holds the input characters given by the user.

Concept note-4: -There are different types of registers, including memory address registers, memory buffer registers, input output address registers, and input output buffer registers and shift registers.
