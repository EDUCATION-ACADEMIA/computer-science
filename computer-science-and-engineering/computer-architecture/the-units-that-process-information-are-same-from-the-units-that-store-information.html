EXPLANATIONS BELOW

Concept note-1: -Decimal units such as kilobyte (KB), megabyte (MB), and gigabyte (GB) are commonly used to express the size of data. Binary units of measurement include kibibyte (KiB), mebibyte (MiB), and gibibyte (GiB).

Concept note-2: -Memory is the unit of a computer dedicated for storage where the data is processed, and the instructions required for data processing are stored in it as well. Memory is generally of two types: Internal, and external.

Concept note-3: -Cache memory. This temporary storage area, known as a cache, is more readily available to the processor than the computerâ€™s main memory source. It is also called CPU memory because it is typically integrated directly into the CPU chip or placed on a separate chip with a bus interconnect with the CPU.

Concept note-4: -Central Processing Unit (CPU) The CPU is the brain of a computer, containing all the circuitry needed to process input, store data, and output results.
