Which type of data transfer is not allowed in interconnection structures

(A) Memory to processor

(B) ** Memory to memory

(C) I/O to processor

(D) Processor to I/O

EXPLANATIONS BELOW

Concept note-1: -The interconnection structure must support the following types of transfers: Memory to processor: The processor reads an instruction or a unit of data from memory. Processor to memory: The processor writes a unit of data to memory. I/O to processor: The processor reads data from an I/O device via an I/O module.

Concept note-2: -Explanation: The switching element controls the interconnections between the processing elements. Explanation: For a large number of processors, the complete interconnection is impractical due to a large number of interconnection paths.

Concept note-3: -system bus connects the CPU (Processor), memory and peripherals devices (input/output device or secondary memory) with each other. The bus system carries data, address and control information . The speed of the system bus is the part of performance of computer system .
