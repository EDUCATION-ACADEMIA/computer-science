(A) cache bus

(B) processor bus

(C) standard I/O bus

(D) ** memory bus

EXPLANATIONS BELOW

Concept note-1: -Second-level system bus that connects the memory subsystem to the chipset and the processor is referred to as . cache bus.

Concept note-2: -The system level bus (as distinct from a CPUâ€™s internal datapath busses) connects the CPU to memory and I/O devices.

Concept note-3: -The System bus connects the CPU and the main memory and the I/O bus connects the CPU to the peripheral devices.

Concept note-4: -Address bus-carries memory addresses from the processor to other components such as primary storage and input/output devices. The address bus is unidirectional .

Concept note-5: -The system bus is the connection between the main groups of components. The system bus is also referred to as the CPU bus and as the front side bus.
