COMPUTER ARCHITECTURE

[SOURCES]
SPEC stands for ____

(A) Standard Performance Evaluation Code

(B) System Processing Enhancing Code

(C) ** System Performance Evaluation Corporation

(D) Standard Processing Enhancement Corporation

EXPLANATIONS BELOW

Concept note-1: -The Standard Performance Evaluation Corporation (SPEC) is a vendor consortium that selects and standardizes benchmark programs submitted by members or others, for the purpose of rating and comparing the performance of processors.

Concept note-2: -The Standard Performance Evaluation Corporation (SPEC) is a non-profit corporation formed to establish, maintain and endorse standardized benchmarks and tools to evaluate performance and energy efficiency for the newest generation of computing systems.

Concept note-3: -Explanation: SPEC is a corporation that started to standardize the evaluation method of a system’s performance. Explanation: In SPEC system of measuring a system’s performance, a system is used as a reference against which other systems are compared and performance is determined.

Concept note-4: -The SPECmark measures the ten programs’ execution rates and SPECthruput, which examines the system’s throughput. Owing to its unsatisfactory results, SPEC89 was replaced by SPEC92 in 1992. The SPEC92 consists of two suites as CINT92 and CFP92. CINT92 consists of six integer intensive C programs.

Concept note-5: -A computer performance evaluation is defined as the process by which a computer system’s resources and outputs are assessed to determine whether the system is performing at an optimal level. It is similar to a voltmeter that a handyman may use to check the voltage across a circuit.
