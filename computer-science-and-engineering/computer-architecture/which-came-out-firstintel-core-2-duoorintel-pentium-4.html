Which came out first?Intel core 2 duoOrIntel pentium 4

(A) Intel core 2 duo

(B) ** Intel Pentium 4

(C) Both came out at the same time

(D) I don’t know.

EXPLANATIONS BELOW

Concept note-1: -The Pentium D was Intel’s first dual-core processor. Still based on Netburst, the first version had the 90 nm Smithfield core (two Northwood cores) and was released as the Pentium D 800 series.

Concept note-2: -The Core 2 processor line was introduced on July 27, 2006, comprising the Duo (dual-core) and Extreme (dual-or quad-core CPUs for enthusiasts), and in 2007, the Quad (quad-core) and Solo (single-core) sub-brands.

Concept note-3: -SANTA CLARA, Calif., Nov. 20, 2000-Intel Corporation today introduced the Intel® Pentium® 4 processor, its newest microprocessor for high-performance desktop computers.

Concept note-4: -Pentium vs Core 2 Duo The Core 2 Duo is a more recent line of microprocessors that succeeded the Core line. Aside from being newer, the Core 2 Duo is also more specific as it only relates to the dual core microprocessors of the Core 2 series.

Concept note-5: -A Core Duo achieves the same performance per core as a whole Pentium 4 with almost only half the clock speed. Which one is better between Duo Core 2 and Pentium 4? The Core 2 Duo had better IPC, two physical cores, more cache depending on model and was more efficient but both are equally obsolete processors.
