BCD stands for ____

(A) ** Binary Coded Decimal

(B) Binary Coded Decoder

(C) Bit coded data

(D) None of these

EXPLANATIONS BELOW

Concept note-1: -In computing and electronic systems, binary-coded decimal (BCD) is a class of binary encodings of decimal numbers where each digit is represented by a fixed number of bits, usually four or eight. Sometimes, special bit patterns are used for a sign or other indications (e.g. error or overflow).

Concept note-2: -Binary Coded Decimal, or BCD, is another process for converting decimal numbers into their binary equivalents. It is a form of binary encoding where each digit in a decimal number is represented in the form of bits. This encoding can be done in either 4-bit or 8-bit (usually 4-bit is preferred).

Concept note-3: -BCD (binary-coded decimal), also called alphanumeric BCD, alphameric BCD, BCD Interchange Code, or BCDIC, is a family of representations of numerals, uppercase Latin letters, and some special and control characters as six-bit character codes.

Concept note-4: -The BCD8421 code is so called because each of the four bits is given a ‘weighting’ according to its column value in the binary system. The least significant bit (lsb) has the weight or value 1, the next bit, going left, the value 2.

Concept note-5: -BCD Code uses four bits to represent the 10 decimal digits of 0 to 9.
