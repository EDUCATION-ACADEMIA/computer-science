An 24 bit address generates an address space of ____ locations.

(A) 1024

(B) 4096

(C) 248

(D) ** 16, 777, 216

EXPLANATIONS BELOW

Concept note-1: -Since we have 24-bit addresses and address resolution is at the byte level, 224 bytes can be addressed. Then, words are there in address space = 2 24 2 4 = 220 = 1M words can be addressed. Hence the correct answer is 1 M words.

Concept note-2: -24-bit addressing involves computers that can locate memory addresses that are 24-bits (3 bytes) wide, allowing for a theoretical maximum of 16 megabytes of memory, though logic board configurations may add other limitations.

Concept note-3: -For an ‘n’ bit address line, we can access 2n memory locations. So, n = 14 bits.

Concept note-4: -On a 32-bit system, virtual addresses are 32 bits, which means that the virtual address space runs from 0 to 0xffff ffff. The size of this address space is 232 bytes, or 4 GiB. On a 64-bit system, the size of the virtual address space is 264 bytes, or 24⋅10246 bytes.
