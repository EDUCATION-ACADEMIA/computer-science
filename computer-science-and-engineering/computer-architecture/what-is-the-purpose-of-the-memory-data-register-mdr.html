What is the purpose of the Memory Data Register (MDR)?

(A) ** Holds data read from memory to be decoded

(B) Hold the location of memory to use in fetch cycle

(C) Holds next memory address to be processed

(D) It does nothing

EXPLANATIONS BELOW

Concept note-1: -A memory buffer register (MBR) or memory data register (MDR) is the register in a computerâ€™s CPU that stores the data being transferred to and from the immediate access storage. It contains a copy of the value in the memory location specified by the memory address register .

Concept note-2: -memory data register (MDR)-holds the contents found at the address held in the MAR, or data which is to be transferred to main memory.

Concept note-3: -Memory Buffer Register (MBR)-a two-way register that holds data fetched from memory (and ready for the CPU to process) or data waiting to be stored in memory. Current Instruction register (CIR)-a temporary holding ground for the instruction that has just been fetched from memory.

Concept note-4: -The Memory Data Register (MDR) holds data that is being transferred to or from memory.
