COMPUTER SCIENCE AND ENGINEERING

COMPUTER ARCHITECTURE

[SOURCES]
The instructions are transferred from the main memory and stored temporarily in small memories within the CPU called registers.

(A) Executing of instructions

(B) Decoding of instructions

(C) Storing of results

(D) ** Fetching of instructions

EXPLANATIONS BELOW

Concept note-1: -The instructions are transferred from the main memory and stored temporarily in small memories within the CPU called registers. The control unit interprets the instruction stored in the instruction register, turning it into a signal that can be understood by the ALU .

Concept note-2: -Push and Pop − The push and pop instructions transfer data between a processor register and memory stack.

Concept note-3: -Temporary memory:-RAM is both used for temporary storage and from a technical perspective, it is only capable of temporary storage. The hard drive is capable of storing information after the computer turns off, but RAM is not. Once your computer turns off, any data that was in your RAM sticks is lost.

Concept note-4: -RAM is used for immediate data storage and retrieval. Your RAM can process information significantly faster than data on a hard disk-twenty to a hundred times faster, depending on the specific hardware and task.

Concept note-5: -Cache memory. This temporary storage area, known as a cache, is more readily available to the processor than the computer’s main memory source. It is also called CPU memory because it is typically integrated directly into the CPU chip or placed on a separate chip with a bus interconnect with the CPU.
