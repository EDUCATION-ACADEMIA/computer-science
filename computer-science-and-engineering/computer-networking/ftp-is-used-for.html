FTP is used for..

(A) .. reading files from the server

(B) .. training people to manage server space

(C) ** .. moving files from one computer to another across the internet

(D) .. following files from the server to your email

EXPLANATIONS BELOW

Concept note-1: -What is FTP (File Transfer Protocol)? FTP (File Transfer Protocol) is a network protocol for transmitting files between computers over Transmission Control Protocol/Internet Protocol (TCP/IP) connections. Within the TCP/IP suite, FTP is considered an application layer protocol.

Concept note-2: -FTP works on a client-server model. The FTP client is a program that runs on the userâ€™s computer to enable the user to talk to and get files from remote computers. It is a set of commands that establishes the connection between two hosts, helps to transfer the files, and then closes the connection.

Concept note-3: -FTP is an acronym for File Transfer Protocol. As the name suggests, FTP is used to transfer files between computers on a network. You can use FTP to exchange files between computer accounts, transfer files between an account and a desktop computer, or access online software archives.

Concept note-4: -File transfer protocol (FTP) is a way to download, upload, and transfer files from one location to another on the Internet and between computer systems. FTP enables the transfer of files back and forth between computers or through the cloud. Users require an Internet connection in order to execute FTP transfers.

Concept note-5: -File transfer protocol (FTP), transmission control protocol (TCP) and hypertext transfer protocol (HTTP) are examples of common standards used today.
