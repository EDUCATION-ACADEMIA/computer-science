What is Hypertext used for?

(A) ** Creating websites

(B) Sending emails

(C) Saving programs

(D) Storing data

EXPLANATIONS BELOW

Concept note-1: -"Hypertext‚Äù refers to links that connect web pages to one another, either within a single website or between websites. Links are a fundamental aspect of the Web. By uploading content to the Internet and linking it to pages created by other people, you become an active participant in the World Wide Web.

Concept note-2: -Hypertext is text displayed on a computer display or other electronic devices with references (hyperlinks) to other text that the reader can immediately access. Hypertext documents are interconnected by hyperlinks, which are typically activated by a mouse click, keypress set, or screen touch.

Concept note-3: -HTML is used to create webpages but does experience limitations when it comes to fully responsive components. Therefore, HTML should only be used to add text elements and structure them within a page. For more complex features, HTML can be combined with cascading style sheets (CSS) and JavaScript (JS).

Concept note-4: -Website development: HTML is used to create web pages that are shown on the internet. HTML code is used by developers to define how browsers display web page elements such as text, hyperlinks, and media files.

Concept note-5: -HyperText Markup Language (HTML) is the basic scripting language used by web browsers to render pages on the world wide web. HyperText allows a user to click a link and be redirected to a new page referenced by that link.
