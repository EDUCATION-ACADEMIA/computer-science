Which of the following topologies would be used if every single computer was to have its own dedicated connection from the central hub?

(A) ** Star Topology

(B) Bus Topology

(C) Ring Topology

EXPLANATIONS BELOW

Concept note-1: -In a star topology, computers arenâ€™t connected to one another but are all connected to a central hub or switch.

Concept note-2: -Ring Topology In this topology, every computer is connected to another computer. Here, the last node is combined with a first one. This topology uses token to pass the information from one computer to another.

Concept note-3: -A mesh topology is defined by overlapping connections between nodes. You can create a full mesh topology, where every node in the network is connected to every other node.

Concept note-4: -Ring topology is a topology in which each computer is connected to exactly two other computers to form the ring. The message passing is unidirectional and circular in nature.
