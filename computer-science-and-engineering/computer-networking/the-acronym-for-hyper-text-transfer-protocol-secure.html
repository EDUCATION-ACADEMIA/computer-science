the acronym for hyper text transfer protocol secure

(A) ** https

(B) http

(C) html

(D) pdf

EXPLANATIONS BELOW

Concept note-1: -Hypertext Transfer Protocol (HTTP)

Concept note-2: -Hypertext transfer protocol secure (HTTPS) is the secure version of HTTP, which is the primary protocol used to send data between a web browser and a website. HTTPS is encrypted in order to increase security of data transfer.

Concept note-3: -HTTPS (Hyper Text Transfer Protocol Secure) appears in the URL when a website is secured by an SSL certificate. The details of the certificate, including the issuing authority and the corporate name of the website owner, can be viewed by clicking on the lock symbol on the browser bar.

Concept note-4: -Highly Hazardous Pesticides (HHPs)
