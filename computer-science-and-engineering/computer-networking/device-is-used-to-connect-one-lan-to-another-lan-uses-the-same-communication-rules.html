Device is used to connect one LAN to another LAN uses the same communication rules

(A) ** Bridge

(B) Gateway

(C) Router

(D) Switches

EXPLANATIONS BELOW

Concept note-1: -"Router” networking device connects one LAN to another LAN using the same protocol.

Concept note-2: -Router: A router is a device that connects two distinct networks.

Concept note-3: -The bridge is used to connect similar LANs with the same protocols. A bridge operates at the data link layer. It is also used for interconnecting two LANs working on the same protocol. It has a single input and single output port, thus making it a 2 port device.

Concept note-4: -A bridge operates on the data link layer. It is a repeater with additional filtering functionality based on reading the source and destination MAC addresses. It is also used to interconnect two LANs that operate under the same protocol. It has a single input and a single output port, thus making it a 2 port device.

Concept note-5: -A bridge is a class of network device that’s designed to connect networks at OSI Level 2, which is the data link layer of a local-area network (LAN).
