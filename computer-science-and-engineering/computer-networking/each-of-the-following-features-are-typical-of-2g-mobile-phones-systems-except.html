Each of the following features are typical of 2G mobile phones systems except:

(A) Caller ID

(B) Internet Access

(C) Multimedia Messaging

(D) ** Video Streaming

EXPLANATIONS BELOW

Concept note-1: -The three main 2G technology standards are Global System for Mobile Communications (GSM), which is based on European Telecommunications Standards Institute (ETSI) standards, time division multiple access (TDMA) IS-136 and code division multiple access (CDMA).

Concept note-2: -2G is short for ‘2nd Generation wireless telephone technology’. 2G lets you send text and picture messages and has some internet capabilities. Because we only sell Smartphones, all our phones work better on 3G than 2G. Sometimes, if you look at your mobile phone or smartphone, you might notice either a letter E or G.

Concept note-3: -Designed to accommodate the growing number of mobile phones, 2G introduced cellular services like SMS, multimedia messaging, and digitally encrypted voice conversations. It also made more efficient use of the radio frequency (RF) spectrum, so more devices could share the same frequency bands.
