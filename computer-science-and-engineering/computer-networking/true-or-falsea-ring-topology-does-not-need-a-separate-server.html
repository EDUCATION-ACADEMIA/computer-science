TRUE OR FALSE:A RING TOPOLOGY does not need a separate server.

(A) ** TRUE

(B) FALSE

EXPLANATIONS BELOW

Concept note-1: -A network server is not needed to control network connectivity between each workstation. Data can transfer between workstations at high speeds. Additional workstations can be added without impacting performance of the network.

Concept note-2: -There is no need of server to control the connectivity among the nodes in the topology. It is cheap to install and expand. Minimum collision. Speed to transfer the data is very high in this type of topology.

Concept note-3: -Advantages of Ring Topology Any network server is not required in the ring topology to network connectivity between each workstation. It has the ability to send data at high speeds. In this network, if you add additional workstations, they do not impact the performance of the network.

Concept note-4: -Ring topology is a type of network topology in which each device is connected to two other devices on either side via an RJ-45 cable or coaxial cable. This forms a circular ring of connected devices which gives it its name. Data is commonly transferred in one direction along the ring, known as a unidirectional ring.

Concept note-5: -Answer: Physically it is star topology but logically its a ring topology. Advantages: Does not require a server.
