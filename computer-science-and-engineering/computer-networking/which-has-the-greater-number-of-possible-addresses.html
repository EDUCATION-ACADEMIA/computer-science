Which has the greater number of possible addresses?

(A) IPv4

(B) ** IPv6

(C) Both the same

EXPLANATIONS BELOW

Concept note-1: -IPv6 uses 128-bit (2128) addresses, allowing 3.4 x 1038 unique IP addresses. This is equal to 340 trillion trillion trillion IP addresses.

Concept note-2: -The number of IPv6 addresses is 1028 (79 228 162 514 264 337 593 543 950 336) times larger than the number of IPv4 addresses.

Concept note-3: -The very large IPv6 address space supports a total of 2128 (about 3.4×1038) addresses – or approximately 5×1028 (roughly 295) addresses for each of the roughly 6.5 billion (6.5×109) people alive today.

Concept note-4: -How many IP addresses can IPv6 create? Picture unique IP addresses as grains of sand. IPv6 = 3.4x1038 addresses-Enough to fill the earth 1 million times. IPv4 = 4 billion addresses-Enough “grains” to fill a wheelbarrow.

Concept note-5: -There are a total of 4, 294, 967, 296 (2^32) or roughly 4.2 billion ipv6/32s (which is also the number of individual IPv4 addresses).
