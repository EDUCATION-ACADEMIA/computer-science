COMPUTER SCIENCE AND ENGINEERING

COMPUTER NETWORKING

[SOURCES]
A series of number used to identify a network device.

(A) ** IP Address

(B) IM Address

(C) MAC Address

(D) OP Address

EXPLANATIONS BELOW

Concept note-1: -An IP address, or Internet Protocol address, is a series of numbers that identifies any device on a network. Computers use IP addresses to communicate with each other both over the internet as well as on other networks.

Concept note-2: -The first part of an IP address is used as a network address, the last part as a host address. If you take the example 192.168.123.132 and divide it into these two parts, you get 192.168.123. Network .132 Host or 192.168.123.0-network address.

Concept note-3: -The IP address is a 32-bit number that uniquely identifies a network interface on a machine. An IP address is typically written in decimal digits, formatted as four 8-bit fields separated by periods.

Concept note-4: -An IP address is a set of four numbers in the range of 0 to 255 separated by periods. Each of the four numbers is referred to as an octet. The first three numbers are used to identify the network segment, the third number identifies the device.
