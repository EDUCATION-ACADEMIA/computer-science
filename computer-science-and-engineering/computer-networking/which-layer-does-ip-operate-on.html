COMPUTER NETWORKING

[SOURCES]
Which layer does IP operate on

(A) Application

(B) Transport

(C) ** Network

(D) Link

EXPLANATIONS BELOW

Concept note-1: -The most significant protocol at layer 3 (also called the network layer) is the Internet Protocol, or IP. IP is the standard for routing packets across interconnected networks–hence, the name internet. It is an encapsulating protocol similar to the way Ethernet is an encapsulating protocol.

Concept note-2: -The Internet Layer of the TCP/IP model aligns with the Layer 3 (Network) layer of the OSI model. This is where IP addresses and routing live.

Concept note-3: -Layer 4 provides for the transparent transfer of data for users, systems, and applications and reliable data transfer services to the upper levels. Since the vast majority of our network traffic is IP-based nowadays, it’s probably easiest to think about layer 4 as it relates to IP traffic specifically.

Concept note-4: -In terms of the OSI model, IP is a network-layer protocol. It provides a datagram service between applications, supporting both TCP and UDP.

Concept note-5: -The four layers of the TCP/IP Model include Network Access Layer, Internet Layer, Host to Host Layer, and Application Layer.
