Identify two network hardware that can be used to setup an Ethernet LAN.

(A) Wireless Network Interface Card

(B) ** LAN Cable

(C) Wireless Router

(D) ** Switch

EXPLANATIONS BELOW

Concept note-1: -To connect a computer or device to a local area network (LAN), several hardware components are required: a network interface card (NIC) or wireless network interface controller (WNIC) a transmission medium, either wired or wireless.

Concept note-2: -Bridge. In regards to networking (network devices bridge), a bridge is a device that connects two LANs or two segments of the same LAN. Networking bridges are also known as network bridges and bridging. There are two types of bridges: the Transparent bridge and the Source Routing bridge.

Concept note-3: -There are two primary LAN types: wired LANs and wireless LANs (WLANs). A wired LAN uses switches and Ethernet cabling to connect endpoints, servers and internet of things (IoT) devices to the corporate network.

Concept note-4: -Hub. A hub is a physical device used to join multiple devices on the same LAN. For example, a laptop, desktop computer and printer can connect into a hubâ€™s ports with Ethernet cables and be part of the same local network.

Concept note-5: -The bridge is used to connect similar LANs with the same protocols. A bridge operates at the data link layer. It is also used for interconnecting two LANs working on the same protocol. It has a single input and single output port, thus making it a 2 port device.
