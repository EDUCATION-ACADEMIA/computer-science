COMPUTER SCIENCE AND ENGINEERING

COMPUTER NETWORKING

[SOURCES]
A computer network can share printers and software.

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -The statement is true. In a computer network, the printers and software are shared. When more than one computer is connected through wired or wireless then the network is formed. The main advantage of a computer network is that it allows the data and resources can be used by any system in that network.

Concept note-2: -The statement is False. Software sharing refers to the use of a single device by many systems. It lets anybody to view any type of software, equipment, or data over a network, regardless of the actual location of the resource.

Concept note-3: -The statement given is false because we can share management systems by being connected to one network but we cannot share printers or other things by connecting them to a network.

Concept note-4: -A network allows sharing of files, data, and other types of information giving authorized users the ability to access information stored on other computers on the network. Distributed computing uses computing resources across a network to accomplish tasks.

Concept note-5: -In the search box on the taskbar, type control panel and then select Control Panel. Under Hardware and Sound, select View devices and printers. Select and hold (or right-click) the printer you want to share, select Printer properties, and then choose the Sharing tab. On the Sharing tab, select Share this printer.
