If your computer screen freezes

(A) push the Escape button

(B) ** use Control, Alt, and Delete to open the Task Manager

(C) Use Control Z to undo the last thing you did

EXPLANATIONS BELOW

Concept note-1: -As a next troubleshooting step, try pressing Alt + Tab to switch to another program. If it’s still stuck and you don’t see the menu switcher appear, try Ctrl + Shift + Esc to open the Task Manager and kill any unresponsive programs. You should also know how to force-close apps without using Task Manager.

Concept note-2: -In Windows 95 or any later systems, the Ctrl-Alt-Delete keyboard shortcut brings up the Task Manager window or a drop-down menu that allows a user to see the status of all currently running programs and to terminate any of them.

Concept note-3: -Restart your phone If your phone is frozen with the screen on, hold down the power button for about 30 seconds to restart.

Concept note-4: -Use Ctrl + Shift + Esc Key Combination. If Windows freezes ctrl alt delete does not work, you can try using the Ctrl + Shift + Esc combination and close all unresponsive apps.

Concept note-5: -Computers. Also Ctrl-Alt-Delete . a combination of three keys on a PC keyboard, usually labeled Ctrl, Alt, and Delete, held down simultaneously in order to close an application that is not responding, reboot the computer, log in, etc.
