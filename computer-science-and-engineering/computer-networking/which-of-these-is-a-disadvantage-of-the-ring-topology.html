COMPUTER NETWORKING

[SOURCES]
Which of these is a disadvantage of the ring topology?

(A) ** If one cable is disabled the whole network can crash

(B) If the central hub crashes the whole network breaks

(C) Only four computers can be attached to the ring

(D) Everyone has to sit facing each other

EXPLANATIONS BELOW

Concept note-1: -The major disadvantage of a ring topology is that if any individual connection in the ring is broken, the entire network is affected. Ring topologies may be used in either LANs (local area networks) or WANs (wide area networks).

Concept note-2: -Adding or removing the computers disturbs the network activity. If the central hub fails, the whole network fails to operate.

Concept note-3: -Because a unidirectional ring topology provides only one pathway between any two nodes, unidirectional ring networks may be disrupted by the failure of a single link. A node failure or cable break might isolate every node attached to the ring.
