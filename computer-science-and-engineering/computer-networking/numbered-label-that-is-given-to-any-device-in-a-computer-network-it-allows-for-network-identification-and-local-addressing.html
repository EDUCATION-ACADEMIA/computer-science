EXPLANATIONS BELOW

Concept note-1: -An Internet Protocol address (IP address) is a numerical label such as 192.0.2.1 that is connected to a computer network that uses the Internet Protocol for communication. An IP address serves two main functions: network interface identification and location addressing.

Concept note-2: -An IP address, or Internet Protocol address, is a series of numbers that identifies any device on a network. Computers use IP addresses to communicate with each other both over the internet as well as on other networks.

Concept note-3: -IP address stands for internet protocol address; it is an identifying number that is associated with a specific computer or computer network. When connected to the internet, the IP address allows the computers to send and receive information.

Concept note-4: -An IP address is a set of four numbers in the range of 0 to 255 separated by periods. Each of the four numbers is referred to as an octet. The first three numbers are used to identify the network segment, the third number identifies the device.
