What is a disadvantage of a standalone computer?

(A) ** Every standalone computer would need a printer to be able to print.

(B) File servers can be expensive to buy.

EXPLANATIONS BELOW

Concept note-1: -Users are restricted to a single computer. The same software cannot be installed simultaneously. It is much cheaper to connect every computer to one printer than to buy a printer for each standalone computer. Standalones are harder to monitor.

Concept note-2: -hard to keep data up-to-date (care must be taken that if 2 people work with the same set of data, that 2 different copies are not produced) Disadvantage. harder to install/update software (each computer must be done individually) Disadvantage. back-ups must be done by each user.

Concept note-3: -Standalone Disadvantages Standalone computers have drawbacks. First of all, users are restricted to a single computer. On a network, users can access their files from any connected computer. Second, the same software cannot be installed simultaneously.

Concept note-4: -A desktop or laptop computer that is used on its own without requiring a connection to a local area network (LAN) or wide area network (WAN). Although it may be connected to a network, it is still a stand-alone PC as long as the network connection is not mandatory for its general use.
