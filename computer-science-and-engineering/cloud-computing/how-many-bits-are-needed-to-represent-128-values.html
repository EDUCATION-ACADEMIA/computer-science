How many bits are needed to represent 128 values?

(A) 6

(B) ** 7

(C) 8

(D) 9

EXPLANATIONS BELOW

Concept note-1: -The ASCII character set uses 7 bits, allowing for 128 different characters. These were used to represent lower-case and upper-case letters in the English alphabet, numbers, symbols, and some commands.

Concept note-2: -128 in binary is 10000000. Unlike the decimal number system where we use the digits 0 to 9 to represent a number, in a binary system, we use only 2 digits that are 0 and 1 (bits). We have used 8 bits to represent 128 in binary.

Concept note-3: -reason for getting this doubt is-128 representation in binary form is 1000 0000.In this MSB is 1 which is the sign bit and magnitude is 0 which is not-128.

Concept note-4: -How Many Bits Does 7 in Binary Have? We can count the number of zeros and ones to see how many bits are used to represent 7 in binary i.e. 111. Therefore, we have used 3 bits to represent 7 in binary.
