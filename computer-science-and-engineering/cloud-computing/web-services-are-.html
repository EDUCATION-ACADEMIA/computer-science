Web Services are ____

(A) Application IDE

(B) Application Designing Tool

(C) None of these

(D) ** Application Components

EXPLANATIONS BELOW

Concept note-1: -Web services can offer application-components like: currency conversion, weather reports, or even language translation as services. Connect existing software. Web services can help to solve the interoperability problem by giving different applications a way to link their data.

Concept note-2: -A Web Services component is a reusable software unit that (1) communicates via open Internet protocols like HTTP, HTTPS, SMTP, etc.; (2) processes XML messages framed using SOAP; (3) describes its messages using XML Schema (XSD); (4) provides a full interface description using WSDL; and (5) can be discovered via UDDI.

Concept note-3: -Web services are self-contained, modular applications that can be described, published, located, and invoked over a network. They implement a services oriented architecture (SOA), which supports the connecting or sharing of resources and data in a very flexible and standardized manner.

Concept note-4: -The SOAP web services architecture is based on interactions between three components: a service provider, a service requester, and an optional service registry. The collection of software that provides a web service.

Concept note-5: -Web services in cloud computing can be found and invoked over the network. The web service would be able to deliver functionality to the client that invoked the web service. A web service is a set of open protocols and standards that allow data to be exchanged between different applications or systems.
