Which one of the following statements is true about the SSO?

(A) It has multiple authentication servers

(B) It manages single accesses to other systems

(C) ** In SSO, User sends the ticket to intranet server.

(D) All of the above

EXPLANATIONS BELOW

Concept note-1: -What is true about SSO? C. In SSO, User sends the ticket to intranet server. Explanation: SSO has single authentication server, managing multiple accesses to other systems and user sends the ticket to intranet server.

Concept note-2: -Advantages of SSO include the following: Users need to remember and manage fewer passwords and usernames for each application. The process of signing on and using applications is streamlinedâ€“no need to reenter passwords. The chances of phishing are lessened.

Concept note-3: -Which of the following is true of the identity-as-a-service (IDaaS) model? IDaaS is also known as authentication as a service (AaaS). IDaaS uses standards such as Open Authorization (OAuth) and Security Assertion Markup Language (SAML) to facilitate identity management in the cloud.

Concept note-4: -Which of the following is the correct statement about cloud computing? Explanation: Abstraction and Virtualization are the two essential concepts.

Concept note-5: -Virtualization is a technique, which allows to share single physical instance of an application or resource among multiple organizations or tenants customers.
