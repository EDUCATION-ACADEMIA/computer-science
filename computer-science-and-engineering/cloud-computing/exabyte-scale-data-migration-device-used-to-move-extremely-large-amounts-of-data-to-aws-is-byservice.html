Exabyte-scale data migration device used to move extremely large amounts of data to AWS is by ____ Service

(A) AWS Snowcone

(B) AWS Snowball

(C) ** AWS Snowmobile

(D) AWS SnowCloud

EXPLANATIONS BELOW

Concept note-1: -AWS Snowmobile is an exabyte-scale data transfer service that is used to move large volumes of data to Amazon Web Services. Each Snowmobile allows transfer for up to 100PB of data. It is a 45-foot-long ruggedized shipping container that is pulled by a semi-trailer truck.

Concept note-2: -Snowmobile enables customers to quickly migrate exabyte-scale datasets from on-premises to AWS in a more secure, fast, and low-cost manner.

Concept note-3: -AWS DataSync You can use DataSync to transfer data at speeds up to 10 times faster than open-source tools. You can use DataSync to copy data over AWS Direct Connect or internet links to AWS for one-time data migrations, recurring data processing workflows, and automated replication for data protection and recovery.

Concept note-4: -Snowball vs Snowmobile A Snowball is a rugged shipping container with a semi-trailer truck. While a single Snowball can store up to 80 TB of data, a single Snowmobile can store 100 PB of data, which is almost 1, 250 times the capacity of a single Snowball.

Concept note-5: -DGI uses AWS’s Snowballs, which are 50 TB or 80 TB hardware devices that can be loaded up and sent to AWS for upload to the cloud. Transferring DGI’s full collection of data in 80TB increments would take up to four years though, Scott says. That’s why DGI is one of the first customers to use the Snowmobile.
