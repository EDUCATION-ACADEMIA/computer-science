CLOUD COMPUTING

[SOURCES]
A start-up company has decided to provide IAAS and initially they have the following<br />requirement<br />a. Need to run multiple OS on individual Machines.<br />b. How is the above requirement achieved and Who will manage this set-up?<br />c. OS need to migrate from one machine to another, who will manage this?<br />d. With the available limited number of systems, -CPU, Memory, I/O needs to be shared<br />among many users, how is this set-up done.<br />e. Need an open source which performs the “Hardware assisted virtualization”<br />Suggest and Explain the solution for the above requirements .

(A) Software as a solution

(B) Platform as a solution

(C) ** virtualization as a solution

(D) none of the mentioned

EXPLANATIONS BELOW

Concept note-1: -In an IaaS service model, a cloud provider hosts the infrastructure components that are traditionally present in an on-premises data center. This includes servers, storage and networking hardware, as well as the virtualization or hypervisor layer.

Concept note-2: -Infrastructure as a service(IaaS): Operating system. Virtual machine and storage. IP addresses. Provides infrastructure.

Concept note-3: -IaaS refers to a combination of hosting, hardware provisioning and basic services needed to run a cloud. PaaS refers to the provision of a computing platform and the provision and deployment of the associated set of software applications (called a solution stack) to an enterprise by a cloud provider.

Concept note-4: -Infrastructure as a Service (IaaS) is a cloud computing service where enterprises rent or lease servers for compute and storage in the cloud. Users can run any operating system or applications on the rented servers without the maintenance and operating costs of those servers.
