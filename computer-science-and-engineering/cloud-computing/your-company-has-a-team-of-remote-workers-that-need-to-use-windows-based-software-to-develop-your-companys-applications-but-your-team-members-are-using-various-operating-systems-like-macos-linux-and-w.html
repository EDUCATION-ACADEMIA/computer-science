CLOUD COMPUTING

[SOURCES]
Your company has a team of remote workers that need to use Windows-based software to develop your company’s applications, but your team members are using various operating systems like macOS, Linux, and Windows. Which Azure compute service would help resolve this scenario?

(A) Azure App Service

(B) ** Azure Virtual Desktop

(C) Azure Container Instances

EXPLANATIONS BELOW

Concept note-1: -Which Azure compute service would help resolve this scenario? That’s correct. Windows Virtual Desktop enables your team members to run Windows in the cloud, with access to the required applications for your company’s needs.

Concept note-2: -Sign in to the Azure portal. From the left menu, select Storage accounts, then select the name of your storage account. Select Containers, then select the thumbnails container. Select Upload to open the Upload blob pane.

Concept note-3: -Which service could help you manage the VMs that your developers and testers need to ensure that your new app works across various operating systems? Azure DevTest Labs is used to manage VMs for testing, including configuration, provisioning, and automatic de-provisioning.

Concept note-4: -Virtual machine scale sets are an Azure Stack Hub compute resource. You can use scale sets to deploy and manage a set of identical virtual machines (VMs).

Concept note-5: -Answer : Create a network security group rule that prevents access from another VM on the same network.
