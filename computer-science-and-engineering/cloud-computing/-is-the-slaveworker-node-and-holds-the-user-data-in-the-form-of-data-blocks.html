(A) ** DataNode

(B) NameNode

(C) Data block

(D) Replication

EXPLANATIONS BELOW

Concept note-1: -DataNode is the slave/worker node and holds the user data in the form of Data Blocks. There can be any number of DataNodes in a Hadoop Cluster.

Concept note-2: -NameNode and DataNodes HDFS exposes a file system namespace and allows user data to be stored in files. Internally, a file is split into one or more blocks and these blocks are stored in a set of DataNodes.
