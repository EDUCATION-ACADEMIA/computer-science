Which of the following is an edge-storage or content-delivery system that caches data in different physical locations?

(A) Amazon Relational Database Service

(B) Amazon SimpleDB

(C) ** Amazon Cloudfront

(D) Amazon Associates Web Services

EXPLANATIONS BELOW

Concept note-1: -3. Which of the following is an edge-storage or content-delivery system that caches data in different physical locations? System (Amazon S3).

Concept note-2: -Amazon CloudFront uses a global network of edge locations and regional edge caches for content delivery.

Concept note-3: -Amazon ElastiCache is a web service that makes it easy to deploy, operate, and scale an in-memory cache in the cloud.

Concept note-4: -AWS edge locations provide an additional layer of network infrastructure that provides these benefits to any web application that uses Amazon CloudFront, Global Accelerator and Amazon Route 53. With these services, you can comprehensively protect on the edge your applications running from AWS Regions.

Concept note-5: -6) What is SimpleDB? Ans: It is a structured data store that supports indexing and data queries to both EC2 and S3.
