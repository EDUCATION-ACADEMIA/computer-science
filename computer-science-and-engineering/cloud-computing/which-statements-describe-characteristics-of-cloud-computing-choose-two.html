CLOUD COMPUTING

[SOURCES]
Which statements describe characteristics of cloud computing? (Choose Two)

(A) A business can access print services online without the use of an network services.

(B) ** Users can access Internet based resources using either subscription-based or pay-per-use plans.

(C) ** Users can access applications and store files using networked devices from any place connected to the Internet.

(D) Businesses are able access their internal network using fiber optic infrastructure.

(E) Users are able to access local services using storage devices located near fog prone environments.

EXPLANATIONS BELOW

Concept note-1: -On-demand self-service. Broad network access. Resource pooling. Rapid elasticity.

Concept note-2: -Cloud computing is composed of 5 essential characteristics, viz: On-demand Self Service. Broad Network Access. Resource Pooling.

Concept note-3: -On-demand self-services: The Cloud computing services does not require any human administrators, user themselves are able to provision, monitor and manage computing resources as needed. Broad network access: The Computing services are generally provided over standard networks and heterogeneous devices.

Concept note-4: -Resources Pooling Resource pooling is one of the essential characteristics of Cloud Computing. Resource pooling means a cloud service provider can share resources among several clients, providing everyone with a different set of services per their requirements.
