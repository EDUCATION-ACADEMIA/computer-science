CLOUD COMPUTING

[SOURCES]
When we create a Virtual Machine in Cloud, it called as which model?

(A) Platform as a Service

(B) ** Infrastructure as a Service

(C) Software as a Service

(D) Container as a Service

EXPLANATIONS BELOW

Concept note-1: -IaaS in cloud computing is when you rent access to cloud infrastructure resources as individual services from a cloud service provider (CSP), including servers, virtual machines, networking resources, and storage.

Concept note-2: -IaaS (Infrastructure-as-a-Service) In IaaS, the cloud service provides computing infrastructure, such as virtual machines, storage, containers, and serverless computing.

Concept note-3: -Infrastructure-as-a-service (IaaS), also known as cloud infrastructure services, is a form of cloud computing in which IT infrastructure is provided to end users through the internet. IaaS is commonly associated with serverless computing.

Concept note-4: -Virtualization in IaaS IaaS or Infrastructure as a Service is driven by one of the critical technologies known as Virtualization. It enables multiple operating systems with dierent congurations to run on a physical machine at the same time.

Concept note-5: -Cloud computing services offer shared resources such as servers, databases, and networks via the internet. Software as a Service (SaaS), Infrastructure as a Service (IaaS), and Platform as a Service (PaaS) are the three service models of cloud computing, each built to address specific business requirements.
