Which of the architectural layer is used as backend in cloud computing?

(A) client

(B) ** cloud

(C) software

(D) Network

EXPLANATIONS BELOW

Concept note-1: -Answer:-The architectural layer used as a backend in cloud computing is cloud. The Cloud computing is on demand accessibility of computer resources. In simple words, cloud computing refers to the idea of using a network of the remote server on the Internet to accumulate, manage, and route the data.

Concept note-2: -The back end is everything the user does not normally see and the things that process the information. This includes the network connectivity, the cloud hardware including servers, and the cloud-based software applications that run on the cloud servers.

Concept note-3: -8. Which of the architectural layer is used as backend in cloud computing? Explanation: The cloud creates a system where resources can be pooled and partitioned as needed.

Concept note-4: -IaaS, or infrastructure as a service, is on-demand access to cloud-hosted physical and virtual servers, storage and networking-the backend IT infrastructure for running applications and workloads in the cloud.

Concept note-5: -Security is the backend component of cloud computing, which insurance the security of data in the cloud.
