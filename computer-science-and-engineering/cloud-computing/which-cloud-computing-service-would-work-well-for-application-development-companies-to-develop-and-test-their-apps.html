Which Cloud computing service would work well for application development companies to develop and test their apps?

(A) SaaS

(B) IaaS

(C) ** PaaS

(D) ITaaS

EXPLANATIONS BELOW

Concept note-1: -Platform as a service (PaaS) is a cloud computing offering that provides users with a cloud environment in which they can develop, manage, and deliver applications.

Concept note-2: -PaaS, or platform as a service, is on-demand access to a complete, ready-to-use, cloud-hosted platform for developing, running, maintaining and managing applications. SaaS, or software as a service, is on-demand access to ready-to-use, cloud-hosted application software.

Concept note-3: -Platform as a Service (PaaS) is a cloud service model that offers users a flexible and scalable cloud platform to develop, deploy, run, and manage apps. It provides the hardware and software resources needed for cloud application development.

Concept note-4: -Serverless lets developers put all their focus into writing the best front-end application code and business logic they can. All developers need to do is write their application code and deploy it to containers managed by a cloud service provider.
