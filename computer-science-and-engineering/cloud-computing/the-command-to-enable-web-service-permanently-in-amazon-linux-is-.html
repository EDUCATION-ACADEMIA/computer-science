The command to enable web service permanently in Amazon Linux is ____

(A) sudo systemctl start httpd

(B) ** sudo systemctl enable httpd

(C) sudo systemctl restart httpd

(D) sudo systemctl on httpd

EXPLANATIONS BELOW

Concept note-1: -Now open the browser and enter http://ec2-ip or http://ec2-url you will see the default page. Enter the command “systemctl enable httpd” and then “systemctl start httpd” on AmazonLinux2. Now you can access the app on boot rather than running the app again and again. You are complete.

Concept note-2: -Amazon Linux 2 is a Linux operating system from Amazon Web Services (AWS). It provides a security-focused, stable, and high-performance execution environment to develop and run cloud applications. Amazon Linux 2 is provided at no additional charge.

Concept note-3: -Enter “systemctl start httpd. service” to START the Apache server. Enter “systemctl enable httpd. service” to ENABLE the Apache web server on the instance.
