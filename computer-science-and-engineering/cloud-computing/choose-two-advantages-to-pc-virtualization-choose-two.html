Choose two advantages to PC virtualization? (Choose two.)

(A) Host machines will experience faster performance.

(B) It allows malware to be identified more easily.

(C) ** Organizations can save money.

(D) ** More than one OS can run on the same hardware.

(E) Malware will not run in a virtual environment.

EXPLANATIONS BELOW

Concept note-1: -What are two advantages of using PC virtualization? (Choose two.) It prevents viruses and worms from functioning in a virtualized environment. It allows multiple operating systems to run on a single PC simultaneously. It boosts the performance of the host machine through virtualization.

Concept note-2: -OS-level virtualization takes advantage of the low overhead of the OS-level architecture in an effort to efficiently use server resources. This provides low overhead while running many VMs on a single physical server.

Concept note-3: -Using virtualization software, you can run multiple operating systems on one physical machine. The technology is now mainstream enough that it is a built-in feature in Windows 7 Professional (and above) and in Windows Server 2008 R2.

Concept note-4: -Desktop virtualization lets you run multiple desktop operating systems, each in its own VM on the same computer. There are two types of desktop virtualization: Virtual desktop infrastructure (VDI) runs multiple desktops in VMs on a central server and streams them to users who log in on thin client devices.
