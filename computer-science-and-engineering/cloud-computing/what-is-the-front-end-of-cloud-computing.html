COMPUTER SCIENCE AND ENGINEERING

CLOUD COMPUTING

[SOURCES]
What is the front end of cloud computing?

(A) The third party company

(B) ** The personal computer user

(C) The internet

(D) Your personal home server

EXPLANATIONS BELOW

Concept note-1: -Clients or cloud clients are the frontend platforms in cloud computing infrastructures. Users connect directly with these clients, which include servers, fat (or thick) clients, thin (or thin) clients, zero clients, tablets, and mobile devices.

Concept note-2: -Front end software architecture includes the software that allows cloud computing software to be run from the user’s side. In today’s web-powered world, the front-end software architecture generally takes the form of a web browser or client-side application.

Concept note-3: -The front end is the side the computer user, or client, sees. The back end is the “cloud” section of the system. The front end includes the client’s computer (or computer network) and the application required to access the cloud computing system.

Concept note-4: -What is Personal Cloud Computing? PCC stands for personal cloud computing, which is a network-attached storage (NAS) device that allows users to store data, photographs, music, movies, and other items and is designed for media streaming.

Concept note-5: -End User Computing (EUC) encompasses user access to enterprise applications and data anywhere, anytime, using one or more devices to access virtual desktop infrastructure (VDI) located either at the enterprise’s premises or in the public cloud.
