COMPUTER SCIENCE AND ENGINEERING

CLOUD COMPUTING

[SOURCES]
Which of the following statements could be used to describe a private cloud deployment?

(A) ** A cloud environment maintained within an enterprise’s own data center

(B) ** A private environment within a third-party or public cloud provider’s architecture

(C) A cloud service inaccessible to anyone but the cultural elite

EXPLANATIONS BELOW

Concept note-1: -Which of the following statements describe a private cloud? A cloud environment that is maintained by the IT staff and data center of an organization.

Concept note-2: -Explanation: The private cloud allows systems and services to be accessible within an organization. The private cloud is more secured because of its private nature. Option B is correct.

Concept note-3: -The private cloud is defined as computing services offered either over the Internet or a private internal network and only to select users instead of the general public.

Concept note-4: -There are four cloud deployment models: public, private, community, and hybrid. Each deployment model is defined according to where the infrastructure for the environment is located. There are three main cloud service models: Software as a Service, Platform as a Service, and Infrastructure as a Service.
