Which of the following are levels of access you can set for files that you share on Google Drive?

(A) ** Can edit, Can comment, and Can view

(B) Can delete, Can edit, and Can comment

(C) Can share, Can edit, and Can view

(D) Can copy, Can download, and Can comment

EXPLANATIONS BELOW

Concept note-1: -Once you start a document, there are three levels of access you can grant to any file or folder: Can edit, Can comment, or Can view.

Concept note-2: -Google Drive gives you access to a suite of tools that allows you to create and edit a variety of files, including documents, spreadsheets, and presentations.

Concept note-3: -Sharing with a link You can easily share a file with a larger group of people by providing a link to any file in your Google Drive. A link is basically a URL or web address for any file you want to share.
