HDFS works in a ____ fashion.

(A) ** master-worker

(B) master-slave

(C) worker/slave

(D) all of the mentioned

EXPLANATIONS BELOW

Concept note-1: -HDFS works in a Master –Slave fashion.In which it have one master node that is known as Name data(It manage the whole system), no of slave mode is known as Data nodes. HDFS is fault tolerant and useful to design at low cost Hardware.

Concept note-2: -HDFS works in master-slave fashion, NameNode is the master daemon which runs on the master node, DataNode is the slave daemon which runs on the slave node.

Concept note-3: -3. HDFS – Hadoop Distributed File System.

Concept note-4: -HDFS has a master/slave architecture. An HDFS cluster consists of a single NameNode, a master server that manages the file system namespace and regulates access to files by clients.

Concept note-5: -Q 2-What is are true about HDFS? A-HDFS filesystem can be mounted on a local client’s Filesystem using NFS. B-HDFS filesystem can never be mounted on a local client’s Filesystem. C-You can edit a existing record in HDFS file which is already mounted using NFS.
