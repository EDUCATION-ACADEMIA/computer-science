This type of solution allows companies to deploy complete desktop environments, which allows their IT to provide a persistent environment.

(A) Office 365

(B) Google Drive

(C) AutoDesk

(D) ** virtual desktop infrastructure

EXPLANATIONS BELOW

Concept note-1: -Virtual Desktop Infrastructure (VDI) is a technology that refers to the use of virtual machines to provide and manage virtual desktops. VDI hosts desktop environments on a centralized server and deploys them to end-users on request.

Concept note-2: -A: An Amazon WorkSpace is a cloud-based virtual desktop that can act as a replacement for a traditional desktop.

Concept note-3: -The three most popular types of desktop virtualization are Virtual desktop infrastructure (VDI), Remote desktop services (RDS), and Desktop-as-a-Service (DaaS).

Concept note-4: -The core product of the VMware VDI solution is VMware Horizon. It is based on VMware vSphere virtualization, adding functionality to manage and deliver virtualized desktops. In VMware Horizon, user desktops are based on VMs running on ESXi hosts.

Concept note-5: -VMware Horizon Cloud Service on Microsoft Azure is a VMware desktop virtualization service that aims to simplify the delivery of virtual desktops and applications on Azure by extending Azure Virtual Desktop.
