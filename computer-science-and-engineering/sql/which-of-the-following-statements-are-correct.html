EXPLANATIONS BELOW

Concept note-1: -There are four main types of JOINs in SQL: INNER JOIN, OUTER JOIN, CROSS JOIN, and SELF JOIN.

Concept note-2: -Q 30-Which of the following is true about SQL joins? A-The join condition is not separated from other search conditions in a query. B-The ON clause makes code difficult to understand. C-The join condition for natural join is basically an equijoin of all columns with same name.

Concept note-3: -Inner Join clause in SQL Server creates a new table (not physical) by combining rows that have matching values in two or more tables. This join is based on a logical relationship (or a common field) between the tables and is used to retrieve data that appears in both tables.

Concept note-4: -A JOIN clause is used to combine rows from two or more tables, based on a related column between them.
