Which of the following is used to a list of values?

(A) ** IN

(B) IS

(C) LIKE

(D) BETWEEN

EXPLANATIONS BELOW

Concept note-1: -For loop statement allows us to specify a list of values in its statement.

Concept note-2: -The values that make up a list are called its elements, or its items. We will use the term element or item to mean the same thing. Lists are similar to strings, which are ordered collections of characters, except that the elements of a list can be of any type.

Concept note-3: -A list is a data structure in Python that is a mutable, or changeable, ordered sequence of elements. Each element or value that is inside of a list is called an item. Just as strings are defined as characters between quotes, lists are defined by having values between square brackets [ ] .

Concept note-4: -The list() function creates a list object. A list object is a collection which is ordered and changeable.

Concept note-5: -In Python, we can create a list by surrounding all the elements with square brackets [] and each element separated by commas. It can be used to store integer, float, string and more.
