If your audit has a target of the Windows Application Log or the Windows Security Log, you can use a tool that can read Windows Event Logs, such as. Choose the correct answer

(A) Task Manager

(B) ** Event Viewer

(C) Event Manager

(D) Performance monitor

EXPLANATIONS BELOW

Concept note-1: -Microsoft includes the Event Viewer in its Windows Server and client operating system to view Windows event logs. Users access the Event Viewer by clicking the Start button and entering Event Viewer into the search field. Users can then select and inspect the desired log.

Concept note-2: -Right click on the Start button and select Control Panel > System Security and double-click Administrative Tools. Double-click Event Viewer. Select the type of logs that you wish to review (ex: Application, System)

Concept note-3: -Windows security auditing lets you audit access to an object, e.g. file, folder, registry key and other system objects that have system access control list (SACL). It is important task for a system administrator to organize file server auditing, but it may be reasonable to audit not only file servers.
