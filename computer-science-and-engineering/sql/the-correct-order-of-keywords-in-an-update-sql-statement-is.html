The correct order of keywords in an UPDATE SQL statement is?

(A) ** UPDATE, SET, WHERE

(B) UPDATE, WHERE, SET

(C) SET, WHERE, UPDATE

(D) WHERE, UPDATE, SET

EXPLANATIONS BELOW

Concept note-1: -The correct answer is Select, where, group by, having.

Concept note-2: -The ‘UPDATE’ statement in MySQL has its own syntax. The ‘SET’ keyword is followed by an assignment list which indicates the set of columns whose value needs to be updated with a specified value.

Concept note-3: -The update command is a DML command used to update a table’s records. The SET keyword is used in the update command. The WHERE clause is also used in the update command for updating single or multiple columns of the table.

Concept note-4: -The UPDATE statement is used to modify the existing records in a table.
