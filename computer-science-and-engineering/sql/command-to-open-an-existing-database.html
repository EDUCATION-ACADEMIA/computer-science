Command to open an existing database

(A) SHOW DATABASES

(B) ** USE

(C) SHOW TABLES

EXPLANATIONS BELOW

Concept note-1: -Answer: The command to open an existing database is as follows: USE database name; Question 16.

Concept note-2: -To access a specific database, type the following command at the mysql> prompt, replacing dbname with the name of the database that you want to access: Copy use dbname; Make sure you do not forget the semicolon at the end of the statement. After you access a database, you can run SQL queries, list tables, and so on.
