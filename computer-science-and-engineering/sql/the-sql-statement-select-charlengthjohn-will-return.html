The SQL statement “SELECT CHAR ____ LENGTH(’John’);” will return,

(A) 1

(B) 2

(C) 3

(D) ** 4

EXPLANATIONS BELOW

Concept note-1: -Well, you can use the LEN() function to find the length of a String value in SQL Server, for example, LEN(emp name) will give you the length of values stored in the column emp name.

Concept note-2: -The CHAR and VARCHAR types are declared with a length that indicates the maximum number of characters you want to store. For example, CHAR(30) can hold up to 30 characters. The length of a CHAR column is fixed to the length that you declare when you create the table. The length can be any value from 0 to 255.

Concept note-3: -The LENGTH() function returns the length of a string (in bytes).

Concept note-4: -The LEFT() function extracts a number of characters from a string (starting from left).
