Which SQL keyword is used to sort the result-set?

(A) ORDER

(B) SORT BY

(C) ** ORDER BY

(D) SORT

EXPLANATIONS BELOW

Concept note-1: -The ORDER BY keyword is used to sort the result-set in ascending or descending order. The ORDER BY keyword sorts the records in ascending order by default. To sort the records in descending order, use the DESC keyword.

Concept note-2: -The SQL ORDER BY clause is used to sort the data in ascending or descending order, based on one or more columns.

Concept note-3: -To get the sorted rows of a result-set, PostgreSQL offers an ORDER BY clause. The table rows can be sorted ascendingly or descendingly with the help of the ORDER BY clause. The ASC or DESC keyword is used in the ORDER BY clause to sort the table rows in a particular order.

Concept note-4: -To change the data in the rows of a database table, we can make use of the UPDATE statement in SQL. The SQL UPDATE command changes the existing data in one or more records of a table.
