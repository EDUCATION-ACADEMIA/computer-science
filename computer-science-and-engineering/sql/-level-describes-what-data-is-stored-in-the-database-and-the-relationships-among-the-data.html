____ level describes what data is stored in the database and the relationships among the data.

(A) Physical

(B) ** Logical

(C) Conceptual

(D) None of the Above

EXPLANATIONS BELOW

Concept note-1: -The logical level describes the relationships between the data. Physical level describes how the data is stored.

Concept note-2: -Physical Level This is the lowest level in the three level architecture. It is also known as the internal level. The physical level describes how data is actually stored in the database.

Concept note-3: -Data abstraction in DBMS means hiding unnecessary background details from the end user to make the accessing of data easy and secure. In DBMS, there are 3 levels of data abstraction: Physical Level or Internal Level: This is the layer where the raw data is stored in file format on physical hard drives.

Concept note-4: -A schema describes the logical structure of a database.

Concept note-5: -Physical Level: Defines how the data is actually stored employing various data structures. Logical level: Describes the relationship which exists among the stored data. View level: Provides a high level view of a section of data.
