What is def in the following statement? CREATE TRIGGER abc ( ____ ) ( ____ ) ON def FOR EACH ROW ghi;

(A) trigger name

(B) ** table name

(C) trigger statement

EXPLANATIONS BELOW

Concept note-1: -7. What is abc in the following MySQL statement? Explanation: In MySQL, the trigger creation construct is the ‘CREATE TRIGGER’ construct. It specifies the trigger name, the type of statement for which it is activated, and the table name and statement.

Concept note-2: -The MERGE statement can also be the triggering event for an UPDATE, DELETE, or INSERT trigger. The event definition must specify the table or view on which the trigger is defined. (SELECT or UPDATE events for triggers on tables can also specify one or more columns.)

Concept note-3: -The FOR EACH ROW option determines whether the trigger is a row trigger or a statement trigger. If you specify FOR EACH ROW, then the trigger fires once for each row of the table that is affected by the triggering statement.

Concept note-4: -A trigger defines a set of actions that are executed when a database event occurs on a specified table. A database event is a delete, insert, or update operation.
