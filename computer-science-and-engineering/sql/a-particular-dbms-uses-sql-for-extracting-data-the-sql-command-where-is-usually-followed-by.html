A particular DBMS uses SQL for extracting data. The SQL command “WHERE” is usually followed by:

(A) ** a criteria

(B) a table name

(C) a user name

(D) a mathematical formula

EXPLANATIONS BELOW

Concept note-1: -The SELECT statement is used to select data from a database. The data returned is stored in a result table, called the result-set.

Concept note-2: -In SQL, to retrieve data stored in our tables, we use the SELECT statement.

Concept note-3: -The UPDATE statement is used to modify the existing records in a table.

Concept note-4: -The SQL USE statement is used to select any existing database in the SQL schema.
