(B) Aggregate functions can be used in WHERE clause even if GROUP BY is used.SELECT DEPT, SUM(Salary) FROM Employee WHERE SUM(Salary) > 90000 GROUP BY DEPT

(C) Nested aggregate function can be used in SELECT clause without GROUP BY clause.SELECT MAX(AVG(Salary)) FROM Employee;

(D) ** Order By cannot be used on columns on which Grouping is not being done.SELECT Dept, SUM(Salary) FROM Employee GROUP BY Dept ORDER BY Designation;
