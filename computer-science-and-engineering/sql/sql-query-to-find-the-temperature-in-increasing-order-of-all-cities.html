SQL query to find the temperature in increasing order of all cities.

(A) SELECT city FROM weather ORDER BY temperature

(B) SELECT city, temperature FROM weather

(C) SELECT city, temperature FROM weather ORDER BY temperature

(D) ** SELECT city, temperature FROM weather ORDER BY city

EXPLANATIONS BELOW

Concept note-1: -It can be used with the SELECT, INSERT, UPDATE and DELETE statement. The IN is a conditional operator used to reduce the use of multiple OR conditions in the SELECT, INSERT, UPDATE and DELETE statement. Hence the option B is the correct choice.
