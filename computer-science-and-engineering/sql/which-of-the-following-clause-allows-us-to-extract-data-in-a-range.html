Which of the following clause allows us to extract data in a range?

(A) ** Between

(B) Range

(C) Find In

(D) Like

EXPLANATIONS BELOW

Concept note-1: -The WHERE clause is a powerful part of the SQL language that allows you to restrict results to values falling within specified ranges.

Concept note-2: -The BETWEEN operator is used in the WHERE clause to select a value within a range of values. We often use the BETWEEN operator in the WHERE clause of the SELECT, UPDATE and DELETE statements.

Concept note-3: -The BETWEEN operator is used in the WHERE clause. The column we want to apply the range condition on is specified in the “column name” parameter. The starting value of the range of values is specified in the “range start” parameter and the ending value in the “range end” parameter.

Concept note-4: -The SQL Server between operator is used to retrieve values based on a certain range.

Concept note-5: -An SQL SELECT statement retrieves records from a database table according to clauses (for example, FROM and WHERE ) that specify criteria.
