Which operator selects values within a given range?

(A) AND

(B) LIKE

(C) SELECT

(D) ** BETWEEN

EXPLANATIONS BELOW

Concept note-1: -The BETWEEN operator selects values within a given range. The values can be numbers, text, or dates. The BETWEEN operator is inclusive: begin and end values are included.

Concept note-2: -And Operator. Determines whether the value of an expression falls within a specified range of values. You can use this operator within SQL statements.

Concept note-3: -The IN operator checks a value within a set of values separated by commas and retrieve the rows from the table which are matching. The IN returns 1 when the search value present within the range otherwise returns 0. Name of the column of the table.
