You can add a column in existing table using SQL in a database with which of the following?

(A) INSERT

(B) COLUMN

(C) ** ALTER

(D) ** TABLE

(E) ** ADD

EXPLANATIONS BELOW

Concept note-1: -The ALTER TABLE statement is used to add, delete, or modify columns in an existing table.
