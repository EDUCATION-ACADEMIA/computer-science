The AVG SQL function returns the ____

(A) the sum of values in a column

(B) ** average in the values in agroup.

(C) maximum value from a column.

EXPLANATIONS BELOW

Concept note-1: -SQL Server AVG() function is an aggregate function that returns the average value of a group. In this syntax: ALL instructs the AVG() function to take all values for calculation. ALL is used by default.

Concept note-2: -The AVG() function returns the average value of a numeric column.

Concept note-3: -The AVG function returns the average (arithmetic mean) of the input expression values. The AVG function works with numeric values and ignores NULL values.
