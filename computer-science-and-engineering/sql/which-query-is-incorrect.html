Which query is incorrect?

(A) ** SELECT * FROM EMP WHERE CITY IN (’MUMBAI’, ‘DELHI’, ‘CHENNAI’);

(B) ** SELECT * FROM EMP WHERE CITY NOT IN (’MUMBAI’, ‘DELHI’, ‘CHENNAI’);

(C) SELECT * FROM EMP GROUP BY DEPT;

(D) ALL THE ABOVE

EXPLANATIONS BELOW

Concept note-1: -The * operator represents all the columns of a table. So, you don’t need to specify each column name in the SELECT query to get data from all the columns. SELECT * FROM Employee; The above query returns all the rows and columns data from the Employee table, as shown below.
