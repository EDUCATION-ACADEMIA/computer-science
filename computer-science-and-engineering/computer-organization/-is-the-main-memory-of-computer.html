____ is the main memory of computer?

(A) ** RAM

(B) ROM

(C) PROM

(D) EPROM

EXPLANATIONS BELOW

Concept note-1: -RAM (Random Access Memory) is the hardware in a computing device where the operating system (OS), application programs and data in current use are kept so they can be quickly reached by the device’s processor. RAM is the main memory in a computer.

Concept note-2: -the main memory of a computer can also be called as primary memory. it is also known as random access memory that is RAM . when computer is shut down all the data in RAM is erased. primary memory is computer memory that is accessed directly by the CPU .

Concept note-3: -Random Access Memory This differentiates RAM from storage devices such as tapes or hard drives where the data is accessed sequentially. RAM is the main memory of a computer. Its objective is to store data and applications that are currently in use. The operating system controls the usage of this memory.

Concept note-4: -RAM is a temporary memory bank where your computer stores data it needs to retrieve quickly. RAM keeps data easily accessible so your processor can quickly find it without having to go into long-term storage to complete immediate processing tasks.

Concept note-5: -RAM stands for random-access memory, but what does that mean? Your computer RAM is essentially short term memory where data is stored as the processor needs it. This isn’t to be confused with long-term data that’s stored on your hard drive, which stays there even when your computer is turned off.
