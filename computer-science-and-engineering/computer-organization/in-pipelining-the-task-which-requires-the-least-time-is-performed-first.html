In pipelining the task which requires the least time is performed first.

(A) True

(B) ** False

EXPLANATIONS BELOW

Concept note-1: -In pipelining the task which requires the least time is performed first. Explanation: This is done to avoid starvation of the longer task.

Concept note-2: -Explanation: In pipelining, each step operates parallel with other steps. It stores and executes instructions in an orderly manner.

Concept note-3: -The three instructions are placed into the pipeline sequentially. In the first cycle the core fetches the ADD instruction from memory. In the second cycle the core fetches the SUB instruction and decodes the ADD instruction. In the third cycle, both the SUB and ADD instructions are moved along the pipeline.

Concept note-4: -A pipelined processor uses a 4-stage instruction pipeline with the following stages: Instruction fetch (IF), Instruction decode (ID), Execute (EX) and Writeback (WB). The arithmatic operations as well as the load and store operations are carried out in the EX stage.

Concept note-5: -Pipelined execution time. = Time taken to execute first instruction + Time taken to execute remaining instructions.
