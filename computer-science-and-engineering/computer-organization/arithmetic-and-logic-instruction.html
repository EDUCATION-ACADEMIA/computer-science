Arithmetic and logic instruction

(C) Memory access

EXPLANATIONS BELOW

Concept note-1: -These are the fundamental arithmetic and logical operations of the processor and operate on values in the general-purpose registers, or a register and an immediate value. Multiply and divide instructions can be considered special cases of these instructions.

Concept note-2: -The arithmetic instructions define the set of operations performed by the processor Arithmetic Logic Unit (ALU). The arithmetic instructions are further classified into binary, decimal, logical, shift/rotate, and bit/byte manipulation instructions.

Concept note-3: -ARM uses four main modes: register, immediate, base, and PC-relative addressing.

Concept note-4: -An instruction, stored in the memory, is fetched into the control unit by supplying the memory with the address of the instruction. The control unit decodes the instruction in order to find the sequence of operation necessary to execute it.

Concept note-5: -umull does an unsigned multiplication, and smull does a signed multiplication.
