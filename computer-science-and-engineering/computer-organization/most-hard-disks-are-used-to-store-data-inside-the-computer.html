(A) ** True

(B) False

Concept note-2: -Answer: Yes hard disk is a permanent storage device.

Concept note-3: -Explanation : A hard disk is a not kind of primary memory, it is a secondary memory. The secondary memory of the computer is known as auxiliary memory. Data on the primary memory i.e. internal memory is directly accessed by the processing unit whereas the secondary memory card data is not directly accessed by the CPU.
