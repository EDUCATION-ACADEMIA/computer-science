Operating system can be matched with(more than one option is correct)

(A) ** Component of computer system

(B) ** Interface between user and hardware

(C) ** Provide platform for other software to run

(D) Used to keep our system efficient and trouble free

EXPLANATIONS BELOW

Concept note-1: -Operating system runs on a computer hardware and serve as platform for other software’s to run on it. The other software that run on operating system platform are known as application or application programs.

Concept note-2: -A cross-platform application is an application that runs the same way on multiple operating systems. Every computer and mobile device has an operating system. Regardless of the type of the computer or device, however, their operating systems provide many similar functions.

Concept note-3: -A multitasking operating system (also called multiprocessing operating system) supports two or more active processes simultaneously.

Concept note-4: -Multiuser/Multitasking OS It is an operating system that permits several users to utilize the programs that are concurrently running on a single network server. The single network server is termed as “Terminal server". “Terminal client” is a software that supports user sessions. Examples include UNIX, MVS, etc.
