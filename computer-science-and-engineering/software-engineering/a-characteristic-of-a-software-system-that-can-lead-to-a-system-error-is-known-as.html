COMPUTER SCIENCE AND ENGINEERING

SOFTWARE ENGINEERING

[SOURCES]
A characteristic of a software system that can lead to a system error is known as?

(A) Human error or mistake

(B) ** System fault

(C) System error

(D) System failure

EXPLANATIONS BELOW

Concept note-1: -System fault A characteristic of a software system that can lead to a system error. For example, failure to initialise a variable could lead to that variable having the wrong value when it is used. Human error or mistake Human behaviour that results in the introduction of faults into a system.

Concept note-2: -An error in computer data is called Bug. A software bug is an error, flaw, failure or fault in a computer program or system that causes it to produce an incorrect or unexpected result, or to behave in unintended ways.

Concept note-3: -A) Software is developed or engineered; it is not manufactured in the classical sense. B) Software doesn’t “wear out. C) Software can be custom built or custom build.

Concept note-4: -The dependability of a system reflects the user’s degree of trust in that system. It reflects the extent of the user’s confidence that it will operate as users expect and that it will not ‘fail’ in normal use. Dependability covers the related systems attributes of reliability, availability and security.
