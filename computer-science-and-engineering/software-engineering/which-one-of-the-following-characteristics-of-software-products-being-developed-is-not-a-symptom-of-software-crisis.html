COMPUTER SCIENCE AND ENGINEERING

SOFTWARE ENGINEERING

[SOURCES]
Which one of the following characteristics of software products being developed is not a symptom of software crisis?

(A) Fail to meet user requirements.

(B) Expensive.

(C) ** Highly interactive.

(D) Difficult to alter, debug, and enhance.

EXPLANATIONS BELOW

Concept note-1: -Which one of the following characteristics of software products being developed is not a symptom of software crisis? Fail to meet user requirements.

Concept note-2: -The linear sequential model, prototype model and spiral model are examples of a software process model. Each model has a different approach to the flow of processes for developing a software. The COCOMO model, on the other hand, is a cost estimation model.

Concept note-3: -Which of these software engineering activities are not a part of software processes ? Explanation: Software dependence is an attribute and not an engineering activity for process.

Concept note-4: -Abolishing the project planning and testing is not applicable as testing and planning are essential in SDLC.
