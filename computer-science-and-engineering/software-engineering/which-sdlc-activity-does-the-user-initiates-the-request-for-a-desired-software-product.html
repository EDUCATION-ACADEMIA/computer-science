Which SDLC activity does the user initiates the request for a desired software product?

(A) Requirement gathering

(B) Implementation

(C) Disposition

(D) ** Communication

EXPLANATIONS BELOW

Concept note-1: -Communication. This is the first step where the user initiates the request for a desired software product. He contacts the service provider and tries to negotiate the terms.

Concept note-2: -System Analysis phase the developers decide a roadmap for project plan-Process Modelling and SDLC.

Concept note-3: -Big Bang model not suitable for large software projects but good for learning-Process Modelling and SDLC.

Concept note-4: -Evolutionary Process Model In evolutionary development, requirements are implemented by category rather than by priority.

Concept note-5: -Answer-D) Waterfall model is not suitable for accommodating changes.
