Estimation of software development effort for organic software is COCOMO is

(A) ** E = 2.4 (KLOC)1.05 PM

(B) E = 3.4 (KLOC)1.06 PM

(C) E = 2.0 (KLOC)1.05 PM

(D) E = 2.4 (KLOC)1.07 PM

EXPLANATIONS BELOW

Concept note-1: -Intermediate COCOMO Model KLOC = The size of the code for the project in Kilo lines of code. a, b = The constant parameters for the software project.

Concept note-2: -Calculate the effort estimate by multiplying the initial estimate with all the multiplying factors i.e., multiply the values in step1 and step2.

Concept note-3: -COCOMO: It is a procedural cost estimate model for software projects and often used as a process of reliably predicting the various parameters associated with making a project such as size, effort, cost, time, and quality.
