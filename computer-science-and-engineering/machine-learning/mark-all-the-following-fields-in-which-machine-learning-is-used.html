MACHINE LEARNING

[SOURCES]
Mark all the following fields in which Machine Learning is used

(A) ** extract patterns from data

(B) ** allowing computers to identify related data

(C) ** forecast future outcomes

(D) ** make predictions or identify important relationships

EXPLANATIONS BELOW

Concept note-1: -Predictive modelling largely overlaps with the field of machine learning. There are two types of predictive models. They are Classification models, that predict class membership, and Regression models that predict a number. These models are then made up of algorithms.

Concept note-2: -Machine learning model predictions allow businesses to make highly accurate guesses as to the likely outcomes of a question based on historical data, which can be about all kinds of things – customer churn likelihood, possible fraudulent activity, and more.

Concept note-3: -The three machine learning types are supervised, unsupervised, and reinforcement learning.

Concept note-4: -Supervised learning algorithms model the relationship between features (independent variables) and a label (target) given a set of observations. Then the model is used to predict the label of new observations using the features.
