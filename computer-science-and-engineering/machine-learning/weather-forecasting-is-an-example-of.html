Weather forecasting is an example of

(A) Clustering

(B) Classification

(C) ** Regression

EXPLANATIONS BELOW

Concept note-1: -Linear Regression is a machine learning algorithm used for the prediction of parameter which is in continuous nature. In this project, linear regression has been used for forecasting the minimum and maximum temperature and wind speed.

Concept note-2: -Regression algorithms can be used to solve the regression problems such as Weather Prediction, House price prediction, etc. Classification Algorithms can be used to solve classification problems such as Identification of spam emails, Speech Recognition, Identification of cancer cells, etc.

Concept note-3: -A simple model for weather forecasting has been described. The model is simple due to the fact that it uses simple mathematical equation using Multiple Linear Regression (MLR) equations that can be easily understood by a medium educated farmer.

Concept note-4: -Regression is a defense mechanism in which people seem to return to an earlier developmental stage. This tends to occur around periods of stress-for example, an overwhelmed child may revert to bedwetting or thumb-sucking. Regression may arise from a desire to reduce anxiety and feel psychologically safe.

Concept note-5: -For example, if it is sunny and 87 degrees today, the persistence method predicts that it will be sunny and 87 degrees tomorrow. If two inches of rain fell today, the persistence method would predict two inches of rain for tomorrow.
