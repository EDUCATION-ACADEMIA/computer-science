Convolutional Neural Network (CNN) for an image classification task has the following operations.1. Max Pooling2. Convolution Operation3. Flatten4. Fully connected layerIdentify the correct sequence from the option below:

(A) 4, 3, 2, 1

(B) ** 2, 1, 3, 4

(C) 3, 1, 2, 4

(D) 4, 2, 1, 3

EXPLANATIONS BELOW

Concept note-1: -Step 1: Convolution. Step 1b: ReLU Layer. Step 2: Pooling. Step 3: Flattening.

Concept note-2: -Input image (starting point) Convolutional layer (convolution operation) Pooling layer (pooling) Input layer for the artificial neural network (flattening)

Concept note-3: -What is Max Pooling? Pooling is a feature commonly imbibed into Convolutional Neural Network (CNN) architectures. The main idea behind a pooling layer is to “accumulate” features from maps generated by convolving a filter over an image.

Concept note-4: -The Convolutional Neural Network (CNN or ConvNet) is a subtype of Neural Networks that is mainly used for applications in image and speech recognition. Its built-in convolutional layer reduces the high dimensionality of images without losing its information. That is why CNNs are especially suited for this use case.
