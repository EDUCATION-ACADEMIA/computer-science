What is a pivot table?

(A) ** A table that helps you analyze data by aggregating the individual items of a more extensive data collection within one or more discrete categories.

(B) A table that inverts the columns and rows so they switch places

(C) A table that color-codes the information that you are interested about

(D) A table that applies an offset function in order to map values onto a new data set.

EXPLANATIONS BELOW

Concept note-1: -A pivot table is a statistics tool that summarizes and reorganizes selected columns and rows of data in a spreadsheet or database table to obtain a desired report. The tool does not actually change the spreadsheet or database itself, it simply “pivots” or turns the data to view it from different perspectives.

Concept note-2: -A PivotTable is an interactive way to quickly summarize large amounts of data. You can use a PivotTable to analyze numerical data in detail, and answer unanticipated questions about your data.

Concept note-3: -A table which horizontally-organised data points. A table used to compute financial pivot values.

Concept note-4: -It involves summarizing the data, obtaining the needed values and presenting the results. Excel provides PivotTable to enable you summarize thousands of data values easily and quickly so as to obtain the required results.

Concept note-5: -An Excel Pivot Table is a potent tool that we can use to slice and dice the data. You can track and analyze the hundreds and thousands of data points with a small table that can be changed dynamically to enable us to find the different perspectives of the data.
