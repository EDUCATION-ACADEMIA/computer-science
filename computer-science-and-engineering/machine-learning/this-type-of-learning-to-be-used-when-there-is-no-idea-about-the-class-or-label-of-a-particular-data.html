This type of learning to be used when there is no idea about the class or label of a particular data

(A) Supervised learning algorithm

(B) ** Unsupervised learning algorithm

(C) Semi-supervised learning algorithm

(D) Reinforcement learning algorithm

EXPLANATIONS BELOW

Concept note-1: -K-means clustering K-means is called an unsupervised learning method, which means you donâ€™t need to label data. Instead, unsupervised learning aims to group and characterize unlabeled datasets. In doing so, it often identifies patterns and similarities in groups of data.

Concept note-2: -Clustering is an unsupervised learning technique, which groups unlabeled data points based on their similarity and differences. Hence, points are grouped into clusters in such a way that those in a same group have the most similarity with each other, while points in different groups have little to no similarities.

Concept note-3: -Unsupervised Learning:-Unsupervised Learning is the second type of machine learning, in which unlabeled data are used to train the algorithm, which means it used against data that has no historical labels.

Concept note-4: -Unsupervised learning (UL) is a machine learning algorithm that works with datasets without labeled responses. It is most commonly used to find hidden patterns in large unlabeled datasets through cluster analysis.
