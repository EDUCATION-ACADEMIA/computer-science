(B) big data

EXPLANATIONS BELOW

Concept note-1: -Answer: Process data is called information. The data that is processed is known as information.

Concept note-2: -When data is processed, organized, structured or presented in a given context so as to make it useful, it is called information. Example.

Concept note-3: -Information is created when data is presented in a way that has meaning to the recipient.
