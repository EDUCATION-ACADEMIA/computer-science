If we didn’t assign a base estimators to the bagging classifier it will use by default:

(A) Linear regression

(B) ** Decision tree

(C) KNN

(D) Logistic regression

EXPLANATIONS BELOW

Concept note-1: -Because bagging and boosting each rely on collections of classifiers, they’re known as ‘ensemble’ methods.

Concept note-2: -Bagging is a method of merging the same type of predictions. Boosting is a method of merging different types of predictions. Bagging decreases variance, not bias, and solves over-fitting issues in a model. Boosting decreases bias, not variance.

Concept note-3: -(2 points) What is the biggest weakness of decision trees compared to logistic regression classifiers? b. Answer: Decision trees are more likely to overfit the data since they can split on many different combination of features whereas in logistic regression we associate only one parameter with each feature.

Concept note-4: -Bagging meta-estimator. In ensemble algorithms, bagging methods form a class of algorithms which build several instances of a black-box estimator on random subsets of the original training set and then aggregate their individual predictions to form a final prediction.
