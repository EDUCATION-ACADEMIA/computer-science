AND Gate:IF A and B are ON, C would light up?

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -The OR gate gets its name from the fact that it behaves after the fashion of the logical inclusive “or.” The output is “true” if either or both of the inputs are “true.” If both inputs are “false, ” then the output is “false.” In other words, for the output to be 1, at least input one OR two must be 1.

Concept note-2: -Rule for an AND gate: output is “high” only if first input and second input are both “high.” Rule for an OR gate: output is “high” if input A or input B are “high.” Rule for a NAND gate: output is not “high” if both the first input and the second input are “high.”

Concept note-3: -The logic gate truth table is as follows: AND: True if A and B are both true. OR: True if A or B are true. XOR: True if either A or B are true, false if both are true.

Concept note-4: -Sometimes you need to AND more than two inputs together. In fact, 3-and 4-input AND gates are just as common as the dual-input variety.
