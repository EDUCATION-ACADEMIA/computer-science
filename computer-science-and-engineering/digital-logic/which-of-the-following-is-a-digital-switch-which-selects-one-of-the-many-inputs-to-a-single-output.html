Which of the following is a digital switch which selects one of the many inputs to a single output.

(A) ** Multiplexer

(B) Demultiplexer

(C) Encoder

(D) Decoder

EXPLANATIONS BELOW

Concept note-1: -A digital multiplexer can be used for parallel to serial conversion and as many to one switch. Multiplexer is also known as data selector that selects between several analogy or digital input signals and forwards it to a single output line.

Concept note-2: –The multiplexer is a combinational circuit that selects binary information from one of many input lines and directs it to a single output line.

Concept note-3: -Multiplexers select one of the input signals and transmit it to a single output. A decoder can accept “n” binary codes and convert them into an output signal. Multiplexers transmit data and signals.

Concept note-4: -In electronics, a multiplexer (or mux; spelled sometimes as multiplexor), also known as a data selector, is a device that selects between several analog or digital input signals and forwards the selected input to a single output line. The selection is directed by a separate set of digital inputs known as select lines.

Concept note-5: -Demultiplexer: It is a device that reverses the process of multiplexer i.e. it converts data from a single input line to multiple output lines.
