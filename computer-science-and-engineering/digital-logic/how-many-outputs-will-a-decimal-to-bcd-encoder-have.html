(B) 8

Concept note-1: -Decimal-to-BCD encoder is the type of encoder that usually consists of ten input lines and 4 output lines. Each input line corresponds to each decimal digit and 4 outputs correspond to the BCD code which is represented in 4 bits.

Concept note-2: -simple decimal-to-BCD encoder is a digital circuit that has 10 input lines and 4 output lines. The inputs represent the 10 decimal numbers from 0 to 9, where only one input can be active. The outputs indicate the BCD code that represents the active input.
