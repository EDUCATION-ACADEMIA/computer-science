COMPUTER SCIENCE AND ENGINEERING

DIGITAL LOGIC

[SOURCES]
Which of the following two input logic functions can act as a Universal gate?

(A) A ⊕ B

(B) AB

(C) ** (A+B)’

(D) None of the above

EXPLANATIONS BELOW

Concept note-1: -The NAND and NOR gates are universal gates.

Concept note-2: -Among the following, NAND gate is the universal gate.

Concept note-3: -Assertion (A): XOR gate is not universal gate. Reason (R): It is not possible to realize any Boolean function using XOR gates only. Explanation: A universal gate is one which can be used to realize any Boolean function using that type of gates only.

Concept note-4: -NAND is a universal gate. universal gate means other logic gates can be represented as a combination of NAND gate.
