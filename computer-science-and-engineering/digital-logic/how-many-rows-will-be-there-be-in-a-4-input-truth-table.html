How many rows will be there be in a 4 input truth table?

(A) 4

(B) 8

(C) ** 16

(D) 32

EXPLANATIONS BELOW

Concept note-1: -For four inputs, there are sixteen rows.

Concept note-2: -Hence the number of rows in the truth table. For a well formed formula with four variables in it will be 16, So the answer is 16.

Concept note-3: -For n number of inputs to the Digital circuits, 2n entries are required for the truth table. Therefore, 16 are truth table entries necessary for a four-input circuit.

Concept note-4: -This number grows exponentially at 2n, where n is the number of inputs. So, a 4-input AND gate has 16 possible combinations, 5 inputs would be 32 outputs, and so on.

Concept note-5: -The number of rows that a truth-table needs is determined by the number of basic statement letters involved in the set of formulas that will be involved in the computation. The formula for the rows is 2n where n = the number of basic statement letters involved.
