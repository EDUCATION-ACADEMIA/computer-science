Which of the following combinations cannot be combined into K-map groups?

(A) Corners in the same row

(B) Corners in the same column

(C) ** Diagonal corners

(D) Overlapping combinations

EXPLANATIONS BELOW

Concept note-1: -Question 1: Answer :c Diagonal corners can not be combined karnaugh-map groups Rules: No zeros allowed.

Concept note-2: -Explanation: Each “1” entry in a K-map square represents a HIGH for each input truth table condition that produces a HIGH output. Thus, it represents a minterm.

Concept note-3: -An XNOR gate is a basic comparator, because its output is “1” only if its two input bits are equal.

Concept note-4: -1. Which statement below best describes a Karnaugh map? It is simply a rearranged truth table. The Karnaugh map eliminates the need for using NAND and NOR gates.
