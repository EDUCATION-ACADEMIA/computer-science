If 3-input OR gate has 8 input possibilities how many possibilities will result in HIGH output?

(A) 1

(B) ** 7

(C) 2

(D) 8

EXPLANATIONS BELOW

Concept note-1: -Question: If a 3-input NOR gate has eight input possibilities, how many of those possibilities will result in a HIGH output? Answers: 2.

Concept note-2: -A three-input gate has eight possibilities (000, 001, 010, 011, 100, 101, 110, and 111) for input states. The number of possible input states is equal to two to the power of the number of inputs: This increase in the number of possible input states obviously allows for more complex gate behavior.

Concept note-3: -Sometimes you need to AND more than two inputs together. In fact, 3-and 4-input AND gates are just as common as the dual-input variety.

Concept note-4: -What input values will cause an AND logic gate to produce a HIGH output? At least one input is HIGH.

Concept note-5: -The output of NOR gate is high when all inputs are low. Explanation: As we see the truth table for NOR gate : When both the input is low (0), then output is high (1).
