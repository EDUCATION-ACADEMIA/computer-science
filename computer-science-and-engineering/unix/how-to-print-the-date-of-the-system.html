how to print the date of the system

(A) date

(B) date +%d-%m-%y

(C) date +’%D-%M-%Y’

(D) ** date +’%d-%m-%Y’

EXPLANATIONS BELOW

Concept note-1: -To format date in MM-DD-YYYY format, use the command date +%m-%d-%Y . Please observe the upper and lower case letters : %m for month, %d for day and %Y for year.

Concept note-2: -Linux date Command Format Options These are the most common formatting characters for the date command: %D – Display date as mm/dd/yy. %Y – Year (e.g., 2020) %m – Month (01-12)

Concept note-3: -To use the date in MM-DD-YYYY format, we can use the command date +%m-%d-%Y. It is very important to note that format option codes are case sensitive. In this example, we have used %m for a month, %d for a day and %Y for a year.
