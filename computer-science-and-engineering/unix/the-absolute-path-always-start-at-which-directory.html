The absolute path always start at which directory?

(A) current directory

(B) ** root directory

(C) home directory

(D) Default partition

EXPLANATIONS BELOW

Concept note-1: -Traces the path from the /(root) directory. Absolute path names always begin with the slash (/) symbol.

Concept note-2: -An absolute path is defined as the specifying the location of a file or directory from the root directory(/). In other words we can say absolute path is a complete path from start of actual filesystem from / directory.

Concept note-3: -Relative path It starts at your current directory and never starts with a / .

Concept note-4: -An absolute path is the full path to a file or directory. It is relative to the root directory ( / ). Note that it is a best practice to use absolute paths when you use file paths inside of scripts. For example, the absolute path to the ls command is: /usr/bin/ls .

Concept note-5: -The file path in Linux A forward slash (/) designates the root directory, which is where an absolute path always begins. On the other hand, a relative route begins in the current directory. In the file system, the root directory is the one with the highest level.
