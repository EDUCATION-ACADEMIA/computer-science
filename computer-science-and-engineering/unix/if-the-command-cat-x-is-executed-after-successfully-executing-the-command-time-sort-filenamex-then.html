If the command cat x is executed after successfully executing the command time sort filename > x, then

(A) only the time details will be displayed

(B) ** only the sorted contents of the file filename will be displayed

(C) an error message will be displayed

(D) None of the above

EXPLANATIONS BELOW

Concept note-1: -The cat command is a utility command in Linux. One of its most common usages is to print the content of a file onto the standard output stream. Other than that, the cat command also allows us to write some texts into a file.

Concept note-2: -For more information about this command, type man more at the Unix system prompt. cat–Displays the contents of a file on your terminal. Result: Displays the contents of the file “newfile” on your terminal.

Concept note-3: -Explanation: cat command supports-n option which is used for displaying file contents along with line number while-v is used for displaying nonprintable ASCII characters in the file.
