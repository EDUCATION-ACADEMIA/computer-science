[SOURCES]
cd command cannot be used without any argument.

(A) True

(B) ** False

EXPLANATIONS BELOW

Concept note-1: -Explanation: cd command can be used without any arguments. When it used in such a way, then it changes our current directory to home directory.

Concept note-2: -Clarification: cd (change directory) command is used for moving around the file system. cd command is usually invoked with a argument. After invocation, it changes the current directory to the directory specified as argument. Cp command is used for copying files while rm command is used for deleting files.

Concept note-3: -Note: Repeating this command toggles the current working directory between the current and the previous working directory. Calling cd without arguments sets the working directory to the value of the HOME environment variable, if the variable exists.

Concept note-4: -If you use the cd command without any arguments then your working directory is changed to your home directory.

Concept note-5: -The cd command uses the following options:-L : Force following symbolic links. This option is enabled by default.
