EXPLANATIONS BELOW

Concept note-1: -The used value of a CSS property is its value after all calculations have been performed on the computed value. After the user agent has finished its calculations, every CSS property has a used value. The used values of dimensions (e.g., width, line-height ) are in pixels.

Concept note-2: -A CSS property is a characteristic (like color) whose associated value defines one aspect of how the browser should display the element.

Concept note-3: -CSS Rule Sets A CSS rule set contains one or more selectors and one or more declarations. The selector(s), which in this example is h1, points to an HTML element. The declaration(s), which in this example are color: blue and text-align: center style the element with a property and value.

Concept note-4: -Property names that are prefixed with–, like–example-name, represent custom properties that contain a value that can be used in other declarations using the var() function.
