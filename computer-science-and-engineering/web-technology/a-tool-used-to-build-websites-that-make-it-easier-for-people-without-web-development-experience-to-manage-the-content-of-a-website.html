WEB TECHNOLOGY

[SOURCES]
A tool used to build websites that make it easier for people without web development experience to manage the content of a website.

(A) ** Content Management System (CMS)

(B) Custom Module

(C) Framework

(D) Staging

(E) Hamburger Navigation

EXPLANATIONS BELOW

Concept note-1: -A CMS has two components: a content management application (CMA) and a content delivery application (CDA). The CMA is a graphical user interface that enables users to design, create, modify and remove content from a website without HTML knowledge.

Concept note-2: -A content management system (CMS) is an application that is used to manage content, allowing multiple contributors to create, edit and publish. Content in a CMS is typically stored in a database and displayed in a presentation layer based on a set of templates like a website.

Concept note-3: -A content management system, often abbreviated as CMS, is software that helps users create, manage, and modify content on a website without the need for specialized technical knowledge.

Concept note-4: -Examples of content management systems: WordPress, Joomla, Drupal, Magento, Squarespace, Wix, Ghost.

Concept note-5: -A content management system helps you create, manage, and publish content on the web. It also helps keep content organized and accessible so it can be used and repurposed effectively. There are various kinds of content management systems available-from cloud-based to a headless CMS-to meet every audience need.
