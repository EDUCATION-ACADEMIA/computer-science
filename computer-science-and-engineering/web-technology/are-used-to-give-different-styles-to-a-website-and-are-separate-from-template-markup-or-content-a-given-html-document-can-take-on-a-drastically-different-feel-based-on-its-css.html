Concept note-1: -Cascading Order What style will be used when there is more than one style specified for an HTML element? All the styles in a page will “cascade” into a new “virtual” style sheet by the following rules, where number one has the highest priority: Inline style (inside an HTML element)

Concept note-2: -CSS (Cascading Style Sheets) is used to style and layout web pages-for example, to alter the font, color, size, and spacing of your content, split it into multiple columns, or add animations and other decorative features.

Concept note-3: -An internal CSS is used to define a style for a single HTML page. An internal CSS is defined in the <head> section of an HTML page, within a <style> element.

Concept note-4: -To link your CSS to your HTML, you have to use the link tag with some relevant attributes. The link tag is a self-closing tag you should put at the head section of your HTML.
