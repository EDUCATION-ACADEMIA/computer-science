Concept note-1: -While there is no hard and fast rule for what breakpoints should be used, values of common CSS breakpoints are 320px or 480px (for mobile phones), 768px (for tablets), and 1920px (for desktop computers).

Concept note-2: -CSS breakpoints are points where the website content responds according to the device width, allowing you to show the best possible layout to the user.
