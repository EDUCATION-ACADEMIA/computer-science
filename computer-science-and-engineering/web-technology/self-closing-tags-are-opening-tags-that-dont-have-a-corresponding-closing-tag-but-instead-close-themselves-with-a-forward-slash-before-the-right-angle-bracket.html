Self-closing tags are opening tags that don’t have a corresponding closing tag but instead close themselves with a forward slash before the right angle bracket.

(A) ** Self-Closing Tag

(B) Declarations

(C) Wireframe

(D) Sitemaps

(E) SSL

EXPLANATIONS BELOW

Concept note-1: -A self-closing tag in HTML is a kind of HTML tag that does not need to be closed manually by its closing tag, which means it does not have a separate closing tag as </tag>. Some few self-closing tags are <input/>, <hr/>, <br/>, <img/>, etc.

Concept note-2: -↑ The full list of valid self-closing tags in HTML5 is: area, base, br, col, embed, hr, img, input, keygen, link, meta, param, source, track, and wbr.

Concept note-3: -Tags with opening and closing tags are called container tags, and tags with only one opening tag and no closing tag are called empty tags.

Concept note-4: -<em> is not a self closing tag, em tag is used for emphasis content.
