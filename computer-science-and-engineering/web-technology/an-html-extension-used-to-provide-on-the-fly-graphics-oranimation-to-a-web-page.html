An HTML extension used to provide on-the-fly graphics oranimation to a Web page.

(A) ** Canvas

(B) RTF

(C) Canvo

(D) POP

EXPLANATIONS BELOW

Concept note-1: -The HTML <canvas> element is used to draw graphics, on the fly, via scripting (usually JavaScript). The <canvas> element is only a container for graphics. You must use a script to actually draw the graphics. Canvas has several methods for drawing paths, boxes, circles, text, and adding images.

Concept note-2: -The canvas tag in HTML5 helps users draw graphics or images on the fly using JavaScript. We can use it for drawing paths, boxes, circles, adding images, etc. The canvas tag has two attributes: width and height for setting the width and height of the canvas.

Concept note-3: -HTML is a graphical design tool used to develop web pages that are displayed with a browser. bracket. XML lets developers define their own markup language. It is best practice to use all uppercase characters for element and attribute names when writing HTML 5.

Concept note-4: -An HTML file is nothing more than plain ASCII text, but all HTML files must have a special file extension for web browsers to recognize them. This extension is either . htm OR . html.
