Concept note-1: -The HTML <p> align Attribute is used to specify the alignment of paragraph text content.

Concept note-2: -For left to right text, the default is align=left, while for right to left text, the default is align=right .

Concept note-3: -To set text alignment in HTML, use the style attribute. The style attribute specifies an inline style for an element. The attribute is used with the HTML <p> tag, with the CSS property text-align for the center, left and right alignment.
