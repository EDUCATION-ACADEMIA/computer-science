When an image is saved as a JPG, all layers contained in the image are merged. In other words, the image is:

(A) ancored

(B) imported

(C) exported

(D) ** flattened

EXPLANATIONS BELOW

Concept note-1: -When an image is saved, all layers contained in the image are merged. In other words, the image is: flattened.

Concept note-2: -To merge all the layers in your Photoshop document, select all the layers and right-click (Control+click for Mac) on the layers panel and select Merge Layers or, without needing to select them, use the shortcut Shift+Ctrl+E (Shift+Cmd+E for Mac). Photoshop also has the option to merge visible layers.

Concept note-3: -Merging all visible layers into the background layer to reduce file size.

Concept note-4: -One of the quickest is to locate the menu button on the layers window tab. Click the menu button to bring up a list of options and simply select Flatten Image.
