If a 1024 x 768 image is to be resized to one that is 640 pixels wide with the same aspect ratio, what would be the height of resized image in pixels.

(A) 256

(B) ** 480

(C) 512

(D) 856

EXPLANATIONS BELOW

Concept note-1: -Here we have given a image with size 1024 × 768 pixels. Also, width changes to 640 pixels while aspect ratio remain same. Height of resized image will be 480 pixels.

Concept note-2: -If you divide 1024 by 768, the result is 1.33, so the 1024x768 computer resolution can be described as having an aspect ratio of 1.33:1.

Concept note-3: -PIXEL DIMENSIONS are the horizontal and vertical measurements of an image expressed in pixels. The pixel dimensions may be determined by multiplying both the width and the height by the dpi.

Concept note-4: -When you resize an image and do not resample it, you change the image’s size without changing the amount of data in that image. Resizing without resampling changes the image’s physical size without changing the pixel dimensions in the image.
