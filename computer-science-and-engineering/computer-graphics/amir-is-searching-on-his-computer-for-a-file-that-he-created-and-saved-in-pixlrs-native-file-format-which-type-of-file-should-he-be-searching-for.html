Amir is searching on his computer for a file that he created and saved in Pixlr’s native file format. Which type of file should he be searching for?

(A) PSD

(B) ** PXD

(C) PDF

(D) EPS

EXPLANATIONS BELOW

Concept note-1: -Therefore, he searches for all files with a . jpg file extension.

Concept note-2: -TIFF-Tagged Image File A TIF is a large raster file that doesn’t lose quality. This file type is known for using “lossless compression, ” meaning the original image data is maintained regardless of how often you might copy, re-save, or compress the original file.

Concept note-3: -Vector images are typically used for logos, icons, typesetting and digital illustrations.

Concept note-4: -EPS (short for Encapsulated PostScript) is a vector format designed for printing to PostScript printers and imagesetters. It is considered the best choice of graphics format for high resolution printing of illustrations. EPS files are created and edited in illustration programs such as Adobe Illustrator or CorelDRAW.

Concept note-5: -TIFF files are best used for: Print publishing only. Which image file format creates a large raster file best used for print but not recommended for use on the Web?
