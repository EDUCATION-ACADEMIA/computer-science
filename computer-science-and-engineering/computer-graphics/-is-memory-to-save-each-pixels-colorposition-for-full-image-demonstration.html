EXPLANATIONS BELOW

Concept note-1: -A framebuffer (frame buffer, or sometimes framestore) is a portion of random-access memory (RAM) containing a bitmap that drives a video display. It is a memory buffer containing data representing all the pixels in a complete video frame. Modern video cards contain framebuffer circuitry in their cores.

Concept note-2: -Each bit plane requires the full complement of memory for a given raster resolution; e.g., a 3-bit plane frame buffer for a 1024 X1024 raster requires 3, 145, 728 (3 X 1024 X1024) memory bits.

Concept note-3: -The number of pixels stored in the frame buffer of a graphics system is known as Resolution.

Concept note-4: -The primary roles of the frame buffer are the storage, conditioning, and output of the video signals that drive the display device. The industry standard for color applications allocates 8 bits of intensity control for each display primary or approximately 16.8 million discretely addressable colors.
