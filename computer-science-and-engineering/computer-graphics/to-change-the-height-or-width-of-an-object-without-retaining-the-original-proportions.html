To change the height or width of an object without retaining the original proportions.

(A) Skew

(B) Crop

(C) Align

(D) ** Distort

EXPLANATIONS BELOW

Concept note-1: -Manually setting the object to a specific proportion: In the dialog box, click the Size tab. Under Scale, enter the percentage of the original height or width you want the object resized to. Note: To change the proportions but keep the original aspect ratio, select the Lock aspect ratio check box.

Concept note-2: -To maintain an object’s proportions while resizing it, press and hold the Shift key while you drag a corner sizing handle.

Concept note-3: -To have Photoshop scale the image without altering the aspect ratio and making it appear stretched, make sure the option to “constrain proportions” is enabled.

Concept note-4: -Currently, if you want to resize an object (by clicking and dragging a corner) without distorting it, you need to hold down the shift key.
