Which of the following is the extension of a vector image file obtained from Illustrator?

(A) .JPEG

(B) ** .AI

(C) .PSD

(D) .DOC

EXPLANATIONS BELOW

Concept note-1: -The most common types of vector file are AI (Adobe Illustrator), EPS (Encapsulated PostScript), PDF (Portable Document Format) and SVG (Scalable Vector Graphics).

Concept note-2: -AI, which stands for Adobe Illustrator, is the proprietary Adobe file type for vector images. Other common vector image file types include PDF, EPS, and SVG.

Concept note-3: -An AIT file is a template file used by Adobe Illustrator. You can make one in that program (File > Save as Template), or download premade templates (File > New). Conversions to PDF, AI, and other formats are supported.

Concept note-4: -Both pioneers of the vector file format, EPS and AI are standard image file types used in Adobe Illustrator.
