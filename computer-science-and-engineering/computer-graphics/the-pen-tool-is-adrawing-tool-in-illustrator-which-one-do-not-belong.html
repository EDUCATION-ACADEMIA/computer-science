The Pen tool is a ____ drawing tool in Illustrator. Which one do not belong?

(A) Free hand

(B) Anchor point

(C) Bezier curves

(D) ** Raster

EXPLANATIONS BELOW

Concept note-1: -As mentioned earlier, the Pen Tool is all about the Anchor Points you connect to create lines – be they straight lines or curvy lines. To start using the Illustrator CC Pen Tool, go to the Toolbar and select the Pen Tool (P) to begin drawing. Alternatively, you can use the keyboard shortcut P to select the Pen Tool.

Concept note-2: -The standard pen tool is used to create vector shapes by creating straight segments and curves. The Freeform Pen tool is used to draw paths, just like the pencil tool. It does not allow us to precisely create curves and segments. It is just a hand drawing tool.

Concept note-3: -The revamped Pencil tool in Illustrator CC offers a new way to draw vector paths. This tutorial shows you how to use it.

Concept note-4: -The Pencil and shape tools produce central vector lines. This means that a pencil line’s control points (used to deform its shape) are located along the length of the central spine, while the Brush line’s control points are located along the contour.
