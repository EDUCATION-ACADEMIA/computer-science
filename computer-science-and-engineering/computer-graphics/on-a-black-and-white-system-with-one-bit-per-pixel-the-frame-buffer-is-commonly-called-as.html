On a black and white system with one bit per pixel, the frame buffer is commonly called as

(A) Pix map

(B) Multi map

(C) ** Bitmap

(D) All of the mentioned

EXPLANATIONS BELOW

Concept note-1: -Answer-B, Explanation-On a black and white system with one bit per pixel, the frame buffer is commonly known as a bitmap.

Concept note-2: -A framebuffer (frame buffer, or sometimes framestore) is a portion of random-access memory (RAM) containing a bitmap that drives a video display. It is a memory buffer containing data representing all the pixels in a complete video frame. Modern video cards contain framebuffer circuitry in their cores.

Concept note-3: -If the refresh buffer employs one bit per pixel, it is referred to as a bitmap. If it employs multiple bits per pixel, it is referred to as pixmap.

Concept note-4: -Explanation: Bit map frame buffer is always 1 bit per pixel and it defines the mapping of points from domain to bits. It is also referred to as a bit array.

Concept note-5: -right option is option A, Explanation-The amount of memory in frame buffer is called Bit plane.At a minimum there is one memory bit for each pixel in the rater; this amount of memory is called a bit plane.
