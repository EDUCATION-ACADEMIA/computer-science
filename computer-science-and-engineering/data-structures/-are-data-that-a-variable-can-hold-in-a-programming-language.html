(A) Variable

(B) ** Data Type

EXPLANATIONS BELOW

Concept note-1: -Depending on the type system of a programming language, variables may only be able to store a specified data type (e.g. integer or string). Alternatively, a datatype may be associated only with the current value, allowing a single variable to store anything supported by the programming language.

Concept note-2: -Variables are the nouns of a programming language: they are the entities (values, data) that act or are acted upon.
