Which statement best describes what happens to the temporary working memory (RAM) of a computer when it is shut down?

(A) ** It is deleted

(B) It is saved in the users document files

(C) It is saved in the computer’s data storage files

(D) It is synced automatically with the cloud

EXPLANATIONS BELOW

Concept note-1: -Which statement best describes what happens to the temporary working memory (RAM) of a computer when it is shut down? The data is not saved.

Concept note-2: -Random Access Memory is volatile. That means data is retained in RAM as long as the computer is on, but it is lost when the computer is turned off. When the computer is rebooted, the OS and other files are reloaded into RAM, usually from an HDD or SSD.

Concept note-3: -When the computer is shut down, the information in RAM is erased. The data saved in ROM is saved even when your computer is powered off.

Concept note-4: -RAM (random access memory) is a computer’s short-term memory, where the data that the processor is currently using is stored. Your computer can access RAM memory much faster than data on a hard disk, SSD, or other long-term storage device, which is why RAM capacity is critical for system performance.

Concept note-5: -RAM is volatile memory, which means that the information temporarily stored in the module is erased when you restart or shut down your computer.
