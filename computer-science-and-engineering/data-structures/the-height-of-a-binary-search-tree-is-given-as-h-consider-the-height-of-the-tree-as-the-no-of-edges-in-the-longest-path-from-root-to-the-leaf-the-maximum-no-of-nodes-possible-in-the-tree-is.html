The height of a Binary Search Tree is given as h. Consider the height of the tree as the no. of edges in the longest path from root to the leaf. The maximum no. of nodes possible in the tree is?

(A) ** 2 ( h + 1 ) − 1

(B) 2 ( h − 1 ) − 1

(C) 2 ( h − 1 ) + 1

(D) 2 ( h ) + 1
