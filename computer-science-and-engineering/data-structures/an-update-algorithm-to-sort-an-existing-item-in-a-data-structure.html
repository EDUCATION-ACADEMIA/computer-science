Concept note-1: -Linear Search It is the most simple search algorithm in data structure and checks each item in the set of elements until it matches the search element until the end of data collection. When data is unsorted, a linear search algorithm is preferred.

Concept note-2: -Explanation: Out of the given options quick sort is the only algorithm which is not stable.
