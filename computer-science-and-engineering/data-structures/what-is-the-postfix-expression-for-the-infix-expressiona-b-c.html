What is the postfix expression for the infix expression:a-b-c?

(A) a-b-c

(B) ** ab-c-

(C) ____ abc

(D) -ab-c

EXPLANATIONS BELOW

Concept note-1: -Explanation: Using the infix to postfix expression conversion algorithm, the corresponding postfix expression is found to be abc*+de*+.

Concept note-2: -The postfix expression abc+de/*-is equivalent to which of the following infix expression? Hence, correct choice is a-(b+c)*(d/e). So, the equivalent infix expression is (1 + 2) * 3 – (4 * 5) and it’s value is-11.

Concept note-3: -The multiplication operator comes immediately before the operands B and C, denoting that * has precedence over +. The addition operator then appears before the A and the result of the multiplication. In postfix, the expression would be A B C * +.
