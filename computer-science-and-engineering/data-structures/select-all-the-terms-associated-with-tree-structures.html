EXPLANATIONS BELOW

Concept note-1: -The topmost node in the tree is called the root. Every node (excluding a root) in a tree is connected by a directed edge from exactly one other node. This node is called a parent.

Concept note-2: -Tree data structure is a non-linear data structure. Concept note-3: -Trees have three main parts-crowns (canopies), trunks, and roots. Each part has a special job to do in keeping the tree healthy and growing.
