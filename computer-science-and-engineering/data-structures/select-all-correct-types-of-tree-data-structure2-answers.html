(A) ** Binary Tree

(B) ** Balanced Binary Tree

(C) Linked Tree

(D) Connected Tree

(E) Directed Linked Tree

EXPLANATIONS BELOW

Concept note-1: -Types of Tree Data Structure Binary Tree. Binary Search Tree (BST) AVL Tree. B-Tree.

Concept note-2: -Adelson-Velskii and Landis (AVL) trees are binary trees which are balanced.

Concept note-3: -Explanation: AVL tree is more balanced than a Red-black tree because AVL tree has less height than Red-black tree given that both trees have the same number of elements.

Concept note-4: -A binary tree is balanced if the height of the tree is O(Log n) where n is the number of nodes. For Example, the AVL tree maintains O(Log n) height by making sure that the difference between the heights of the left and right subtrees is at most 1.
