Which of the following is an example of a FIFO data structure?

(A) Stack

(B) Array

(C) ** Queue

(D) Linked List

EXPLANATIONS BELOW

Concept note-1: -A linear queue is a linear data structure based on FIFO (First in First Out) principle, in which data enters from the rear end and is deleted from the front end.

Concept note-2: -FIFO (First-In-First-Out) queues have all the capabilities of the standard queues, but are designed to enhance messaging between applications when the order of operations and events is critical, or where duplicates canâ€™t be tolerated.

Concept note-3: -The data structure that implements FIFO is Queue.

Concept note-4: -The primary difference between Stack and Queue Data Structures is that Stack follows LIFO while Queue follows FIFO data structure type.

Concept note-5: -Which of the following data structures can be used to implement queues? Stack, Arrays, and LinkedList can be used to implement Queues so all the options are correct.
