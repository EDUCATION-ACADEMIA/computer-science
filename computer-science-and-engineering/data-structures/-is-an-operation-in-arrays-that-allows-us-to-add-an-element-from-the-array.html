____ is an operation in arrays that allows us to add an element from the array.

(A) Delete

(B) Edit

(C) ** Insert

(D) Search

EXPLANATIONS BELOW

Concept note-1: -The push() method is used for adding an element to the end of an array.

Concept note-2: -Insertion Operation Insert operation is to insert one or more data elements into an array. Based on the requirement, a new element can be added at the beginning, end, or any given index of array.

Concept note-3: -In this scenario, we are given the exact location (index) of an array where a new data element (value) needs to be inserted. First we shall check if the array is full, if it is not, then we shall move all data elements from that location one step downward.
