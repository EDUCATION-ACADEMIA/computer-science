What is an advantage of a circular queue?

(A) ** Free space can be reused

(B) Easy to program

(C) Gives preference to important items

(D) It is static

EXPLANATIONS BELOW

Concept note-1: -Advantages. Circular Queues offer a quick and clean way to store FIFO data with a maximum size. Conserves memory as we only store up to our capacity (opposed to a queue which could continue to grow if input outpaces output.)

Concept note-2: -Efficient utilization of memory: In the circular queue, there is no wastage of memory as it uses the unoccupied space, and memory is used properly in a valuable and effective manner as compared to a linear queue.

Concept note-3: -I would say the biggest disadvantage to a circular queue is you can only store queue. length elements. If you are using it as a buffer, you are limiting your history depth. Concept note-4: -Advantages of Deque: You are able to add and remove items from the both front and back of the queue. Deques are faster in adding and removing the elements to the end or beginning. The clockwise and anti-clockwise rotation operations are faster in a deque.
