COMPUTER SCIENCE AND ENGINEERING

DATA STRUCTURES

[SOURCES]
A single elementary unit of information representing an attribute of an entity

(A) Attribute and Entity

(B) Entity Set

(C) ** Field

(D) Record

EXPLANATIONS BELOW

Concept note-1: -Field-A field is a single elementary unit of information that represents an entity’s attribute.

Concept note-2: -A data item that does not have subordinate data items is called an elementary item . A data item that is composed of one or more subordinate data items is called a group item . A record can be either an elementary item or a group item.

Concept note-3: -File − File is a collection of records of the entities in a given entity set.

Concept note-4: -A data item refers to a single unit of values. “Data items that divided into a subitems are called group items” and “such data items which are not divided into sub items are called elementary data items".

Concept note-5: -DATA STRUCTURES UNIT-1: Data Structure: It is a particular way of organizing and storing data in a computer. So that it can be accessed and modified efficiently. A data structure will have a collection of data and functions or operations that can be applied on the data.
