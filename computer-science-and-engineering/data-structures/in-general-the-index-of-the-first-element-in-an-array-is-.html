In general, the index of the first element in an array is ____

(A) ** 0

(B) -1

(C) 1

(D) 2

EXPLANATIONS BELOW

Concept note-1: -Explanation: In general, Array Indexing starts from 0. Thus, the index of the first element in an array is 0.

Concept note-2: -The first element in an array corresponds to index 0.

Concept note-3: -The first element in every array has index zero and is sometimes called the zeroth element. Thus, the elements of array c are c[0], c[1], c[2] and so on. The highest index in array c is 11, which is 1 less than 12-the number of elements in the array. Array names follow the same conventions as other variable names.

Concept note-4: -Short Answer An array is a pointer, and the index is used as the offset. The first element of the array is precisely at the pointerâ€™s memory location; therefore, the offset is zero.

Concept note-5: -So the first element is 0, second is 1, so on. So if the there are n elements in a list, the last element is n-1.
