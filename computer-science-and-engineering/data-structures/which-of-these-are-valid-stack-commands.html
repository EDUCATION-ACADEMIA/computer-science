EXPLANATIONS BELOW

Concept note-2: -Syntax Of Using Stack In C++ stack <type> stackName; stack is the name of the stack template keyword we use to construct a stack object. type is a valid C++ data type passed as an argument to the stack template. It indicates the data type of the elements stored in the stack.

Concept note-3: -Types of Stack. There are two types of stacks they are register stack and the memory stack.

Concept note-4: -A Stack can be used for evaluating expressions consisting of operands and operators. Stacks can be used for Backtracking, i.e., to check parenthesis matching in an expression.
