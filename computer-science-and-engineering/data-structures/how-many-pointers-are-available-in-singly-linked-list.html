how many pointers are available in singly linked list

(A) 4

(B) 3

(C) ** 1

(D) 2

EXPLANATIONS BELOW

Concept note-1: -In a singly-linked list, each node contains one item of data and one pointer.

Concept note-2: -In a doubly linked circular linked list, each node has pointers that point to both the next node and the previous node.

Concept note-3: -The doubly linked list node accomplishes this in the obvious way by storing two pointers: one to the node following it (as in the singly linked list), and a second pointer to the node preceding it.

Concept note-4: -Singly Linked Lists: Each node contains only one pointer to the next node. This is what we have been talking about so far. Doubly Linked Lists: Each node contains two pointers, a pointer to the next node and a pointer to the previous node.

Concept note-5: -In C language, a linked list can be implemented using structure and pointers . struct LinkedList int data; struct LinkedList *next; ; The above definition is used to create every node in the list. The data field stores the element and the next is a pointer to store the address of the next node.
