linked list is supported in c++

(A) ** yes

(B) no

EXPLANATIONS BELOW

Concept note-1: -In C language, a linked list can be implemented using structure and pointers . struct LinkedList int data; struct LinkedList *next; ; The above definition is used to create every node in the list. The data field stores the element and the next is a pointer to store the address of the next node.

Concept note-2: -Essentially, linked lists function as an array that can grow and shrink as needed, from any point in the array. Linked lists have a few advantages over arrays: Items can be added or removed from the middle of the list. There is no need to define an initial size.

Concept note-3: -A linked list in C/C++ is basically a linear data structure based on the concept of dynamic memory allocation. It is implemented with the help of pointers. The linked list in C and C++ tutorial is specially designed for the beginners, who are not aware of the importance of linked lists.

Concept note-4: -In C++ the linked list can be represented with a class and a Node class separately, which has two members, namely data and a next pointer which points to the next node.
