What will be the output of the following C code?#include <stdio.h>int main(){int a = 1, b = 1, c;c = a++ + b;printf("%d, %d", a, b);}

(A) a = 1, b = 1

(B) a = 1, b = 2

(C) a = 2, b = 2

(D) ** a = 2, b = 1

EXPLANATIONS BELOW

Concept note-1: -When we say Output, it means to display some data on screen, printer, or in any file. C programming provides a set of built-in functions to output the data on the computer screen as well as to save it in text or binary files.

Concept note-2: -In C++ the default return type of main is void, i.e. main() will not return anything. But, in C default return type of main is int, i.e. main() will return an integer value by default.

Concept note-3: -What will be the output of the following C code on a 64 bit machine? Explanation: Since the size of a union is the size of its maximum data type, here int is the largest data type. Hence the size of the union is 4.

Concept note-4: -After the source code has been assembled, it will produce an Object files (e.g. .o, . obj) and then linked, producing an executable files. An object and executable come in several formats such as ELF (Executable and Linking Format) and COFF (Common Object-File Format).
