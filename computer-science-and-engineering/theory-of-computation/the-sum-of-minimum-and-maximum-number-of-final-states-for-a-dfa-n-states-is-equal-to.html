The sum of minimum and maximum number of final states for a DFA n states is equal to:

(A) ** n+1

(B) n

(C) n-1

(D) n+2

EXPLANATIONS BELOW

Concept note-1: -Answers. The maximum number of final states for a DFA can be total number of states itself and minimum would always be 1, as no DFA exits without a final state. Therefore, the solution is n+1.

Concept note-2: -âˆ´ Minimum number of states required for DFA = 3.

Concept note-3: -For an arbitrary NFA with N states, the maximum number of states in an equivalent minimized DFA is 2N.

Concept note-4: -A state in a DFA will be a subset of the set of states of the equivalent NFA. So, the maximum number of states in the equivalent DFA of an NFA, will be 2n, where n is the number of states in NFA, as a set with n items has maximum 2n subsets. GATE CS Answer Key has been released!

Concept note-5: -Take for example : An NFA has 7 states out of which 3 are final states. Its equivalent DFA can have atmost 128 states.
