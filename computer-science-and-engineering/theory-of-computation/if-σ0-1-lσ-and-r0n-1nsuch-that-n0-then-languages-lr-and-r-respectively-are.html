THEORY OF COMPUTATION

If = (0, 1), L = * and R = (0n 1nsuch that n > 0 )then languages L ∪ R and R respectively are

(A) Regular, Regular

(B) ** Regular, Not regular

(C) Not regular, Not regular

(D) None of these

EXPLANATIONS BELOW

Concept note-1: -Explanation : CFG is a higher than regular language. So we can draw a regular equivalent to CFG. And some non regular like context sensitive can’t be generated by cfg.

Concept note-2: -All regular languages are context-free languages, but not all context-free languages are regular. Most arithmetic expressions are generated by context-free grammars, and are therefore, context-free languages.

Concept note-3: -Explanation: Every regular language can be produced by context free grammar and context free language can be produced by context sensitive grammar and so on.
