Concept note-1: -In the notation 0, 1*, set notation 0, 1 is used to denote “0 or 1". Alternatively, + is also used to denote alternation (JFLAP, for example). So (0+1) can also mean “0 or 1". | is also used to denote alternation (0|1) in theoretical computer science context.

Concept note-4: -For example for alphabet ∑ = 0, 1 w = 010101 is a string. Length of a string is denoted as |w| and is defined as the number of positions for the symbol in the string. For the above example length is 6.
