Concept note-3: -Explanation: According to the question, minimum of 3 states are required to recognize an octal number divisible by 3.

Concept note-4: -For the acceptance and rejection of any string we can simply check for the movement on each input alphabet between states. A string is accepted if we stop at any final state of the DFA. For string u=abbaba the string ends at t (final state) hence it is accepted by the DFA.
