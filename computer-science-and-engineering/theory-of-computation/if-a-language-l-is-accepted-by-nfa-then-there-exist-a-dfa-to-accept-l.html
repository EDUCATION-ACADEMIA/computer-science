If a language L is accepted by NFA, then there exist a DFA to accept L

(A) ** Yes

(B) No

EXPLANATIONS BELOW

Concept note-1: -The set of all strings accepted by an NFA is the language the NFA accepts. This language is a regular language. For every NFA a deterministic finite automaton (DFA) can be found that accepts the same language.

Concept note-2: -A language L is recognized by a DFA if and only if there is an NFA N such that L(N) = L. In other words: For any DFA D, there is an NFA N such that L(N) = L(D), and • For any NFA N, there is a DFA D such that L(D) = L(N).
