E-close of state is a combination of self state and ____

(A) reachable state

(B) initial state

(C) final state

(D) ** -reachable state

EXPLANATIONS BELOW

Concept note-1: -The epsilon closure E(q) of a state q in Q is the union of the set q with the set of all states that can be reached from q via one or more transitions. If R is a set of states from Q, the epsilon closure E(R) is defined as the union of the epsilon closures of all the states in R.

Concept note-2: -Unreachable states are the states that are not reachable from the initial state of the DFA, for any input string. These states can be removed. Dead states are the states from which no final state is reachable. These states can be removed unless the automaton is required to be complete.

Concept note-3: --closure: -closure for a given state A means a set of states which can be reached from the state A with only (null) move including the state A itself.

Concept note-4: -Explanation: -closure is defined as the set of states being reached through -transitions from a starting state.
