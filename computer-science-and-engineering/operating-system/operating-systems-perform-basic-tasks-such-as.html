OPERATING SYSTEM

[SOURCES]
Operating Systems perform basic tasks such as

(A) creating BIOS data for users.

(B) recognizing input from the system clock.

(C) ** recognizing input from the keyboard.

(D) creating programs for unauthorized users.

EXPLANATIONS BELOW

Concept note-1: -An operating system performs basic tasks such as recognizing input from a keyboard, sending output to the display screen, keeping track of files and directories on disk and controlling peripheral devices; for example, disk drives and printers.

Concept note-2: -The operating system (OS) manages all of the software and hardware on the computer. It performs basic tasks such as file, memory and process management, handling input and output, and controlling peripheral devices such as disk drives and printers.

Concept note-3: -Real-time: Real time operating system (RTOS) responds to input instantly. General-purpose operating systems, such as DOS and UNIX, are not real-time.

Concept note-4: -The kernel is responsible for performing the following tasks: Input-Output management.

Concept note-5: -The fundamental role of the operating system in computer Input / Output is to manage and organize I/O operations and all I/O devices. The various devices that are connected to the computer need to be controlled and it is a key concern of operating-system designers.
