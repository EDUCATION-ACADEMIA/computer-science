COMPUTER SCIENCE AND ENGINEERING

OPERATING SYSTEM

[SOURCES]
Smart phones don’t need operating systems, only computers do.

(A) True

(B) ** False

EXPLANATIONS BELOW

Concept note-1: -The answer is true because all cell phones have an operating system. an operating system (OS) is just the internal software system that displays the interface of your cell phone.

Concept note-2: -The most important software in any smartphone is its operating system (OS). An operating system manages the hardware and software resources of smartphones.

Concept note-3: -It manages the computer’s memory and processes, as well as all of its software and hardware. It also allows you to communicate with the computer without knowing how to speak the computer’s language. Without an operating system, a computer is useless.

Concept note-4: -As every general-purpose computer must have an operating system to run other programs and applications.

Concept note-5: -A mobile operating system allows the user to run other different application software on the mobile, tablets, etc. Moreover, we can say that it is a type of operating system which is specially designed for mobiles, tablets, smartwatches, etc.
