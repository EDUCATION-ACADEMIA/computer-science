COMPUTER SCIENCE AND ENGINEERING

OPERATING SYSTEM

[SOURCES]
During the boot process, the ____ looks for the system files.

(A) CPU

(B) ** BIOS

(C) CD Room

EXPLANATIONS BELOW

Concept note-1: -BIOS (basic input/output system) are the program a computer’s microprocessor uses to start the computer system after it is powered on.

Concept note-2: -The BIOS. When an x86 computer is booted, the processor looks at the end of system memory for the Basic Input/Output System or BIOS program and runs it. The BIOS controls not only the first step of the boot process, but also provides the lowest level interface to peripheral devices.

Concept note-3: -BIOS identifies, configures, tests and connects computer hardware to the OS immediately after a computer is turned on. The combination of these steps is called the boot process. These tasks are each carried out by BIOS’ four main functions: Power-on self-test (POST).

Concept note-4: -The boot process is controlled by the computer’s BIOS (basic input output system)which usually starts in Read Only Memory ROM. The bootstrap loader from ROM then loads the operating system.

Concept note-5: -Booting is basically the process of starting the computer. When the CPU is first switched on it has nothing inside the Memory. In order to start the Computer, load the Operating System into the Main Memory and then Computer is ready to take commands from the User.
