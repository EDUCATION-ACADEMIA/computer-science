Which of the following is a segment of memory or storage, such as the disk in the accompanying figure, in which items are placed while waiting to be transferred from an input device or to an output device?

(A) Process

(B) Index

(C) ** Buffer

(D) Throttle

EXPLANATIONS BELOW

Concept note-1: -A buffer is a segment of memory or storage in which items are placed while waiting to be transferred from an input device or to an output device. An operating system commonly uses buffers with printed documents.

Concept note-2: -Plug and Play means the operating system automatically configures new devices as you install or connect them.

Concept note-3: -The operating system (sometimes called the “OS”) provides a common set of controls for managing computer hardware, making it easier for users to interact with computers and for programmers to write application software.

Concept note-4: -Hibernate: Your computer saves its current state to your hard drive. When you boot up your computer, your computer’s state will be saved, including all opened programs and data, so you can quickly access any work you’ve stepped away from.
