EXPLANATIONS BELOW

Concept note-1: -A system using virtual memory uses a section of the hard drive to emulate RAM. With virtual memory, a system can load larger or multiple programs running at the same time, enabling each one to operate as if it has more space, without having to purchase more RAM.

Concept note-2: -Microsoft allows you to use an external hard drive as RAM on Windows 10. You should open Control Panel, access Advanced system settings, and choose the external hard drive to configure manually. Please follow the 12 steps below to set up virtual RAM on Windows 10 now.

Concept note-4: -Virtually all desktop and laptop PCs have virtual memory, a technology that uses part of the hard drive as though it were RAM. Virtual memory conveys a number of advantages, including the ability to run very large programs on any compatible computer, including those with modest amounts of physical RAM.
