Concept note-3: -A multi-user operating system is an operating system that permits several users to access a single system running to a single operating system. These systems are frequently quite complex, and they must manage the tasks that the various users connected to them require.

Concept note-4: -A GUI or user interface (UI) is the part of an OS that permits an operator to get the information. A user interface based on text displays the text as well as its commands which are typed over a command line with the help of a keyboard.
