A system tool used to clean up your disk

(A) ** Disk clean up

(B) Cookie

(C) defragmenter

(D) GUI

EXPLANATIONS BELOW

Concept note-1: -In the search box on the taskbar, type disk cleanup, and select Disk Cleanup from the list of results. Select the drive you want to clean up, and then select OK. Under Files to delete, select the file types to get rid of. To get a description of the file type, select it.

Concept note-2: -Clean up tools remove the clutter of data your PC collects over a period of time. Clean up tools can be used to solve the clutter problem. Disk Clean up scans files and lists all of the files you might like to delete. For instance Clean up tools can remove cookies.

Concept note-3: -Disk Cleanup is a Microsoft software utility first introduced with Windows 98 and included in all subsequent releases of Windows. It allows users to remove files that are no longer needed or that can be safely deleted.

Concept note-4: -disk cleanup is not a system tool. The most well-known mobile OSs are Android, iOS, Windows phone OS, and Symbian.

Concept note-5: -Click Start, and then click Run. In the Open box, type the following command, and then press Enter: c:32.exe /dDrive Note In this command, the placeholder Drive represents the drive letter of the hard disk to be cleaned.
