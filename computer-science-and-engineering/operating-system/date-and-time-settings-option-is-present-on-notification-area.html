Date and time settings option is present on notification area.

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -The notification area is a part of the taskbar that provides a temporary source for notifications and status. It can also be used to display icons for system and program features that are not on the desktop. The notification area was known historically as the system tray or status area.

Concept note-2: -Detailed Solution. The correct answer is Taskbar. Date and Time’ located in taskbar in Windows 10 operating system.

Concept note-3: -The time and date displays on the system tray. A system tray is a part of the toolbar in Microsoft Windows. This system is mailnly used to show the status of many functions and events.

Concept note-4: -The International Date Line (IDL) is located at about 180° east (or west). It is halfway around the world from the prime meridian (0° longitude), the reference point of time zones, which runs through Greenwich, UK.
