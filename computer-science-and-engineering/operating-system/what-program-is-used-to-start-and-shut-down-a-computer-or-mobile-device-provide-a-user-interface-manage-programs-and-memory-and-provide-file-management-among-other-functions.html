OPERATING SYSTEM

[SOURCES]
What program is used to start and shut down a computer or mobile device, provide a user interface, manage programs and memory, and provide file management, among other functions?

(A) ** An operating System

(B) A File Utility

(C) A System Utility

(D) Application Software

EXPLANATIONS BELOW

Concept note-1: -An operating system is the most important software that runs on a computer. It manages the computer’s memory and processes, as well as all of its software and hardware.

Concept note-2: -System Software. System software – it is designed to run a computer’s hardware and application software, and make the computer system available for use. It serves as the interface between hardware, application software, and the user.

Concept note-3: -An operating system (OS) is system software that manages computer hardware, software resources, and provides common services for computer programs.

Concept note-4: -The process of starting or restarting a computer is called booting.

Concept note-5: -The correct answer is Operating system. An operating system or OS is software installed on a computer’s hard drive that enables the computer hardware to communicate and operate with the computer software.
