File system File system is?

(A) The shell keeps the end users access kernel .

(B) ** uses to name, store, and organize files on a drive

(C) produce basic information about data

(D) All of above

EXPLANATIONS BELOW

Concept note-1: -A file system stores and organizes data and can be thought of as a type of index for all the data contained in a storage device. These devices can include hard drives, optical drives and flash drives.

Concept note-2: -With the help of WIN32 or NTFS file systems, Windows hard drives are formatted. NTFS: NTFS is the file system, which stands for NT file system and stores and retrieves files on Windows NT operating system and other versions of Windows like Windows 2000, Windows XP, Windows 7, and Windows 10.

Concept note-3: -In file storage, data is stored in files, the files are organized in folders, and the folders are organized under a hierarchy of directories and subdirectories. To locate a file, all you or your computer system need is the path-from directory to subdirectory to folder to file.

Concept note-4: -What is a file system? File systems are how data is arranged, sent, and received to a hard drive partition. They also specify an index and what types of information can be included on a file (i.e., read and write permissions, file type, file name, etc.)
