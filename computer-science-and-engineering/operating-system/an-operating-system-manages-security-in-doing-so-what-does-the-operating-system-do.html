COMPUTER SCIENCE AND ENGINEERING

OPERATING SYSTEM

[SOURCES]
An operating system manages security. In doing so what does the operating system do?

(A) Ensures that different processes can utilise the CPU and do not interfere with each other or crash

(B) ** Allows creation and deletion of user accounts

(C) Communicates with and receives data input to a keyboard/mouse/other valid input device

(D) Ensures that programs/data do not corrupt each other

EXPLANATIONS BELOW

Concept note-1: -It has user authentication methods that ensure legitimacy of user access. OS provides antivirus protection against malicious attacks and has inbuilt firewall which acts as a filter to check the type of traffic entering into the system.

Concept note-2: -The operating system is responsible for implementing a security system that ensures the authenticity of a user who is executing a specific program. In general, operating systems identify and authenticate users in three ways.

Concept note-3: -A user interface (UI) refers to the part of an operating system, program, or device that allows a user to enter and receive information. A text-based user interface (see the image to the left) displays text, and its commands are usually typed on a command line using a keyboard.

Concept note-4: -The operating system is the physical environment where your application runs. Any vulnerability in the operating system could compromise the security of the application. By securing the operating system, you make the environment stable, control access to resources, and control external access to the environment.
