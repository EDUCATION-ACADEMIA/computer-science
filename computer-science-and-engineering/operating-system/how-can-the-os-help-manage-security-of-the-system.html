How can the OS help manage security of the system?

(A) The OS can hire backup

(B) ** User accounts may be password protected

(C) Files can self destruct after a set time

EXPLANATIONS BELOW

Concept note-1: -OS provides antivirus protection against malicious attacks and has inbuilt firewall which acts as a filter to check the type of traffic entering into the system.

Concept note-2: -The safety of their system resources such as saved data, disks, memory, etc. is secured by the security systems against harmful modifications, unauthorized access, and inconsistency. It provides a mechanism (encryption and authentication) to analyze the user before allowing access to the system.

Concept note-3: -The OS shares resources between users, depending on what type of resources the users need. The OS must ensure that the system stays well-balanced in resources to meet each userâ€™s needs and not affect other users who are connected. Some examples of a multi-user OS are Unix, Ubuntu, MacOS, Windows and all Linux based OS.
