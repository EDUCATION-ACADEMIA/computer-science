Computers can only understand the 0s and 1s.

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -The only language that the computer can process or execute is called machine language. It consists of only 0s and 1s in binary, that a computer can understand. In short, the computer only understands binary code, i.e 0s and 1s.

Concept note-2: -Computers use binary to store data. Not only because it’s a reliable way of storing the data, but computers only understand 1s and 0s-binary. A computer’s main memory consists of transistors that switch between high and low voltage levels-sometimes 5V, sometimes 0.

Concept note-3: -0 is generally represented by no electricity, and 1 is generally represented by electricity present. like an on off switch. These switches feed into a processor, that does basic things such as adding. if you feed 101 and 010 into the addition chip it outputs the three result bits 111.
