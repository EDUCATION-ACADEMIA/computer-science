How many passes will it take to sort this data in ascending order, 1, 2, 3, 5, 4.

(A) 2

(B) 5

(C) 3

(D) ** 1

EXPLANATIONS BELOW

Concept note-1: -Explanation: Even though the first two elements are already sorted, bubble sort needs 4 iterations to sort the given array.

Concept note-2: -Expert-Verified Answer. 5 interchanges are required to sort 5, 1, 6, 2 4 in ascending order using bubble sort.

Concept note-3: -Time complexity of Merge Sort is O(n*Log n) in all the 3 cases (worst, average and best) as merge sort always divides the array in two halves and takes linear time to merge two halves. It requires equal amount of additional space as the unsorted array.

Concept note-4: -The least number of comparisons needed to sort (order) five elements is 8.
