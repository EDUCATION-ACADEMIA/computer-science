A datatype that can hold more than one value is known as ____

(A) ** Composite

(B) Concrete

(C) Abstract

(D) Atomic

EXPLANATIONS BELOW

Concept note-1: -A variable holds more than one value if you declare it to be of a composite data type. Composite Data Types include structures, arrays, and classes. A variable of a composite data type can hold a combination of elementary data types and other composite types. Structures and classes can hold code as well as data.

Concept note-2: -Composite data types are data types that have one or more fields dynamically linked to fields in another data type.

Concept note-3: -Conclusion. A composite number is a positive integer divisible by smaller positive integers other than 1 and itself. Composite numbers can be identified by using the divisibility method.

Concept note-4: -Why is a class known as composite data type? A composite data type is one which is composed with various primitive data types. A class can contain various primitive data types as its data members so it is known as a composite data type.

Concept note-5: -A complex data type is usually a composite of other existing data types. For example, you might create a complex data type whose components include built-in types, opaque types, distinct types, or other complex types.
