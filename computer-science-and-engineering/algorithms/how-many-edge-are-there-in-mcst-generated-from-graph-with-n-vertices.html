How many edge are there in MCST generated from graph with ‘n’ vertices

(A) ** n-1

(B) n/2-1

(C) n-2

(D) n/2

EXPLANATIONS BELOW

Concept note-1: -Property 1: A free tree with N >= 1 vertices has exactly N-1 edges.

Concept note-2: -Explanation: A graph can have many spanning trees. And a complete graph with n vertices has n(n-2) spanning trees.

Concept note-3: -And an optimal solution is feasible solution for which rule (A) is maximized. Of these Six feasible solutions, solution 4 yields the maximum profit. Therefore solution 4 is optimal for the given problem instance.

Concept note-4: -An algorithm to construct a Minimum Spanning Tree for a connected weighted graph. It is a Greedy Algorithm. The Greedy Choice is to put the smallest weight edge that does not because a cycle in the MST constructed so far. If the graph is not linked, then it finds a Minimum Spanning Tree.
