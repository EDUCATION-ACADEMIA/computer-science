What is the average case complexity of Quick sort algorithm?

(A) ** O(n log n)

(B) O(log n)

(C) O(n)

(D) O(n2)

EXPLANATIONS BELOW

Concept note-1: -Quicksort’s average case time complexity is O(n*logn) . The following is the average-case recurrence. 3). Worst Case Complexity: The worst-case situation is when the partitioning algorithm picks the largest or smallest element as the pivot element every time.

Concept note-4: -Quicksort applied to a list of n elements, again assumed to be all different and initially in random order. This popular sorting algorithm has an average-case performance of O(n log(n)), which contributes to making it a very fast algorithm in practice.
