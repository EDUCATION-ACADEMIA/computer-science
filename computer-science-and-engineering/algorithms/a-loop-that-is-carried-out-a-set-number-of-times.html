A loop that is carried out a set number of times

(A) ** Count controlled

(B) Condition controlled

(C) Infinite

(D) Array

EXPLANATIONS BELOW

Concept note-1: -A count-controlled loop is used when the number of iterations to occur is already known. Steps that are part of the loop are indented . Indentation is used to show which steps are to be iterated. In this example, the variable ‘count’ is used to keep track of how many times the algorithm has iterated.
