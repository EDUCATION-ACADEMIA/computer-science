when can algorithms be used

(A) ** Any time to design solutions to problems

(B) Only with computers

(C) Only when programming

(D) Only with flowcharts

EXPLANATIONS BELOW

Concept note-1: -The correct answer is (a) any time to design solutions to problems. The word Algorithm signifies “an interaction or set of rules to be continued in computations or other critical thinking tasks".

Concept note-2: -Algorithms are used for calculation, data processing, and automated reasoning.” Whether you are aware of it or not, algorithms are becoming a ubiquitous part of our lives.

Concept note-3: -An algorithm is a defined set of step-by-step procedures that provides the correct answer to a particular problem. By following the instructions correctly, you are guaranteed to arrive at the right answer. An algorithm is often expressed in the form of a graph, where a square represents each step.

Concept note-4: -Algorithms can be designed for any type of problem, i.e. mathematical, logical, or any complex problems. Example: Depth-first-search, traveling salesman, sorting algorithms, etc. But, after some steps, the algorithm would result in a finite solution before ending.
