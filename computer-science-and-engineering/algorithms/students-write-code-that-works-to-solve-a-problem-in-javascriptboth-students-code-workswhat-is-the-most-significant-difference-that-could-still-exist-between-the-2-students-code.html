Concept note-1: -DOM-related Errors DOM is also known as Document Object Model. It’s one of the crucial elements of a JavaScript-based website interface. Because it handles most of the website’s content, structure, and style, DOM-related errors are responsible for 68% of the top 10 JavaScript errors.

Concept note-2: -Programmers must first understand how a human solves a problem, then understand how to translate this “algorithm” into something a computer can do, and finally how to “write” the specific syntax (required by a computer) to get the job done.
