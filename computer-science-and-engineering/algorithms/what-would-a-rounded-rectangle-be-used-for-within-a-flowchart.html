What would a rounded rectangle be used for within a flowchart?

(A) Input / Output

(B) Output

(C) ** Start / Stop

(D) Decision

EXPLANATIONS BELOW

Concept note-1: -Process: The process topic a rounded rectangle represents an action that needs to be performed for the process to move to the next step.

Concept note-2: -The Oval The oval, or terminator, is used to represent the start and end of a process. Use the Gliffy flowchart tool to drag and drop one of these bad boys and you’ve got yourself the beginning of a flowchart. Remember to use the same symbol again to show that your flowchart is complete.

Concept note-3: -In most flowcharts, the rectangle is the most common shape. It is used to show a process, task, action, or operation. It shows something that has to be done or an action that has to be taken. The text in the rectangle almost always includes a verb.

Concept note-4: -The rounded rectangle symbol on a flowchart is known as a “terminator.” It is used to show the start, stop, beginning or end of a process.

Concept note-5: -Document – The Document object is a rectangle with a wave-like base. This shape is used to represent a Document or Report in a process flow. Rounded box – This is used to represent an event which occurs automatically.
