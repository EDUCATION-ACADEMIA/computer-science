Data type that can only be true or false

(A) int

(B) string

(C) ** bool

(D) float

EXPLANATIONS BELOW

Concept note-1: -In computer science, the Boolean (sometimes shortened to Bool) is a data type that has one of two possible values (usually denoted true and false) which is intended to represent the two truth values of logic and Boolean algebra.

Concept note-2: -A Boolean variable has only two possible values: true or false. It is common to use Booleans with control statements to determine the flow of a program.

Concept note-3: -The BOOLEAN data type stores TRUE or FALSE data values as a single byte. The following table shows internal and literal representations of the BOOLEAN data type. You can compare two BOOLEAN values to test for equality or inequality.

Concept note-4: -A Boolean is a type of data that has only two possible values: true or false. You can think of a boolean like the answer to a yes or no question.

Concept note-5: -The Python Boolean type has only two possible values: True. False.
