Which of the following are input devices?

(A) Speakers

(B) ** mouse

(C) ** camera

(D) printer

EXPLANATIONS BELOW

Concept note-1: -Keyboards, scanners, mouses, joysticks and digital cameras are examples of input devices.

Concept note-2: -A digital camera is an input device that captures images (and sometimes video) digitally. Digital cameras use an image sensor chip to capture the image. However, it can also be used as an output device. The digital camera can send pictures to our computer.

Concept note-3: -A camera is an optical instrument that captures images. Most cameras can capture 2D images, while some more advanced models can capture 3D images.

Concept note-4: -Anything supplying data to the computer is an input device. This includes mice, keyboards, hard disks, memory sticks, phones, scanners and digital cameras for a start. Any device that the computer supplies data to is an output device.

Concept note-5: -Is Camera an Input Device or Output Device? The camera is an input device. The camera captures images and videos from outside the world; later, this data and information are presented to a computer system for further processing, such as editing and modifying using the latest photo and video editing software.
