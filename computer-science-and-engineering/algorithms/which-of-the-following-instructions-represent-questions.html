Which of the following instructions represent questions?

(A) ** ELSE and ELSE IF

(B) ELSE and THEN

(C) ELSE IF and THEN

Concept note-1: -The ELSE IF instruction allows there to be more than two paths through an algorithm . Any number of ELSE IF instructions can be added to an algorithm. It is used along with other instructions: IF represents a question.

Concept note-2: -In programming, selection is usually represented by the statements IF and ELSE: IF represents the question. ELSE points to what to do if the answer to the question is false.
