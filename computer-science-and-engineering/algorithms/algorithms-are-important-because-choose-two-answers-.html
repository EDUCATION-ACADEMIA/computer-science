Algorithms are important because (choose two answers )

(A) ** The automation of tasks makes new things impossible

(B) ** The automation of redundant, tedious, or dangerous tasks frees people from having to perform these boring, time-consuming, or potentially lethal tasks.

(C) Algorithm can not be generalized

EXPLANATIONS BELOW

Concept note-1: -Algorithms are a very important topic in Computer Science because they help software developers create efficient and error free programs. The most important thing to remember about algorithms is that there can be many different algorithms for the same problem, but some are much better than others!

Concept note-2: -Algorithms are used to find the best possible way to solve a problem, based on data storage, sorting and processing, and machine learning. In doing so, they improve the efficiency of a program. Algorithms are used in all areas of computing. Because it is a fantastic way of automating computer decisions.

Concept note-3: -Algorithms are instructions for solving a problem or completing a task. Recipes are algorithms, as are math equations. Computer code is algorithmic. The internet runs on algorithms and all online searching is accomplished through them.

Concept note-4: -To solve a given problem, you create an algorithm. A problem can be solved in a variety of ways. As a result, many solution algorithms for a given problem can be derived.
