An outer join, either Left or Right, retreives records that matchas well as records that do not match.

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -The FULL OUTER JOIN keyword returns all records when there is a match in left (table1) or right (table2) table records.

Concept note-2: -Full Join or the Full Outer Join returns all those records which either have a match in the left(Table1) or the right(Table2) table.

Concept note-3: -There is no difference between RIGHT JOIN and RIGHT OUTER JOIN . Both are the same. That means that LEFT JOIN and LEFT OUTER JOIN are the same.

Concept note-4: -Left Outer Join returns all of the rows in the current data and all the data from the matching rows in the joined data, adding rows when there is more than one match. This can result in an expanded row count.
