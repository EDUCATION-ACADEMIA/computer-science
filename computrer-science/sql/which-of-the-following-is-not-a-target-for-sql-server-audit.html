Which of the following is not a target for SQL Server Audit?

(A) File

(B) Windows Application Log

(C) ** Ring Buffer

EXPLANATIONS BELOW

Concept note-1: -The results of an audit are sent to a target, which can be a file, the Windows Security event log, or the Windows Application event log.

Concept note-2: -The ring buffers are created during SQL Server startup, and record alerts within the SQL Server system for internal diagnostics.

Concept note-3: -SQL Server Audit consists of several object components, including SQL Server Audit, SQL Server Audit Specification, Database Audit Specification, and a target.

Concept note-4: -Explanation: ON State enables the audit.
