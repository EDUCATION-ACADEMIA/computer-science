You can view a complete list of server-scoped permissions using the system function sys.fn ____ builtin ____ permissions:

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -SQL Server includes a very useful system function sys. fn my permissions to list all the permissions of a particular principal (user or login) and this system function will help you list all permissions of a principal on a specific database object (securable).

Concept note-2: -To check the user permissions in a SQL Server instance, you can query the sys. database permissions and the sys. database principals system catalog views.
