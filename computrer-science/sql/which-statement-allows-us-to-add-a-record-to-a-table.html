Which statement allows us to add a record to a table?

(A) Add To

(B) Update To

(C) Add Into

(D) ** Insert Into

EXPLANATIONS BELOW

Concept note-1: -The SQL INSERT statement is used to insert a one or more records into a table.

Concept note-2: -The INSERT INTO statement of SQL is used to insert a new row in a table.

Concept note-3: -To insert records into a table, enter the key words insert into followed by the table name, followed by an open parenthesis, followed by a list of column names separated by commas, followed by a closing parenthesis, followed by the keyword values, followed by the list of values enclosed in parenthesis.

Concept note-4: -The ALTER TABLE statement is used to add, delete, or modify columns in an existing table. The ALTER TABLE statement is also used to add and drop various constraints on an existing table.
