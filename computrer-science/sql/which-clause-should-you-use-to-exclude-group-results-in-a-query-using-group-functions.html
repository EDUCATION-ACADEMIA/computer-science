Which clause should you use to exclude group results in a query using group functions?

(A) WHERE

(B) ** HAVING

(C) GROUP BY

(D) ORDER BY

EXPLANATIONS BELOW

Concept note-1: -To exclude some group results, use Having-clause. GROUP BY dept; This is because the group function cannot be used in WHERE clause.

Concept note-2: -Use the WHERE clause to exclude rows you do not want grouped, and use the HAVING clause to filter records after they have been grouped.

Concept note-3: -Clause Use to Exclude Group Results.

Concept note-4: -The GROUP BY clause is used with the SELECT statement to make a group of rows based on the values of a specific column or expression. The SQL AGGREGATE function can be used to get summary information for every group and these are applied to an individual group.
