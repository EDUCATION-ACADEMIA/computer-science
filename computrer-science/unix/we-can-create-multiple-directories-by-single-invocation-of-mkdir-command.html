We can create multiple directories by single invocation of mkdir command.

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -You can create directories one by one with mkdir, but this can be time-consuming. To avoid that, you can run a single mkdir command to create multiple directories at once.

Concept note-2: -We can create multiple directories by single invocation of mkdir command. Explanation: Multiple directories can be created by one mkdir command.

Concept note-3: -For example, to remove a directory named dir 001 in the current directory type the following command on the terminal. 9. Multiple directories can be removed using single rmdir command. Clarification: Like mkdir command, we can delete multiple directories using one shot of rmdir command.

Concept note-4: -Use the mkdir command to create one or more directories specified by the Directory parameter. Each new directory contains the standard entries dot (.) and dot dot (..). You can specify the permissions for the new directories with the-m Mode flag.

Concept note-5: -The mkdir command in Linux/Unix allows users to create or make new directories (also referred to as folders in some operating systems).
