[SOURCES]
Please choose all true statements regarding Unix kernel:

(A) ** Interfaces with hardware components

(B) ** Generates inodes for new files and process id’s for new processes

(C) ** Performs memory management

(D) Users can directly access the kernel to run applications

EXPLANATIONS BELOW

Concept note-1: -Memory management is the functionality of an operating system which handles or manages primary memory and moves processes back and forth between main memory and disk during execution. Memory management keeps track of each and every memory location, regardless of either it is allocated to some process or it is free.

Concept note-2: -The Windows kernel-mode memory manager component manages physical memory for the operating system. This memory is primarily in the form of random access memory (RAM). The memory manager manages memory by performing the following major tasks: Managing the allocation and deallocation of memory virtually and dynamically.

Concept note-3: -Memory management is the process of controlling and coordinating a computer’s main memory. It ensures that blocks of memory space are properly managed and allocated so the operating system (OS), applications and other running processes have the memory they need to carry out their operations.
