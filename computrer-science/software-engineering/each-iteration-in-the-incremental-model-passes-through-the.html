Concept note-3: -Incremental Model is a process of software development where requirements divided into multiple standalone modules of the software development cycle. In this model, each module goes through the requirements, design, implementation and testing phases.

Concept note-4: -As the name indicates, Iterative and incremental development (IID) is a model that is an incremental model that is developed in multiple cycles of iterations. Project is started with a comparatively small task or component and increments are made in each cycle of the iterations until desired product is reached.
