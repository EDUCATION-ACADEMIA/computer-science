(A) It begins with SRS

(B) It ends with Product Design Problem

(C) Analysis is done and end product is project mission statement

(D) ** None of the mentioned

EXPLANATIONS BELOW

Concept note-1: -What is true about generic software product design process? Explanation: It begins with product design problem ends with SRS and starting point is project mission statement, hence all choices are false.

Concept note-3: -Software design is the process of mapping user requirements to a functional tool. It involves multiple iterations of gathering user requirements combined with programming language selection, data object creation, prototyping of the tool, and testing the prototype.

Concept note-4: -The software design phase comes after the feasibility and resources analysis phase. The quality of the software depends upon the design phase a lot.
