What is the main objective of Software Project Planning?

(A) Estimation of resources

(B) Estimation of cost

(C) Estimation of schedule

(D) ** All of the above

EXPLANATIONS BELOW

Concept note-1: -The objective of software project planning is to provide a framework that enables the manager to make reasonable estimates of resources, cost, and schedule. These estimates are made within a limited time at the beginning of a software project and should be updated regularly as the project progresses.

Concept note-2: -Q. 1 What is the main objective of software project planning? A) Estimation of resources.

Concept note-3: -Explanation: The first step in project planning is to establish the objectives and scope.

Concept note-4: -What is a project objective? Project objectives in project management are the specific, tangible outcomes that will be produced and delivered by the project. They identify and describe the concrete actions or deliverables that will work together to achieve the broader, higher-level goals of the project as a whole.

Concept note-5: -Software project planning is task, which is performed before the production of software actually starts. It is there for the software production but involves no concrete activity that has any direction connection with software production; rather it is a set of multiple processes, which facilitates software production.
