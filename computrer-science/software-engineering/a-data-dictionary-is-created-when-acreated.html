A data dictionary is created when a ____ created.

(A) Instance

(B) Segment

(C) ** Database

(D) Dictionary

EXPLANATIONS BELOW

Concept note-1: -When you use the Database Configuration Assistant to create a database, Oracle automatically creates the data dictionary. Thereafter, whenever the database is in operation, Oracle updates the data dictionary in response to every DDL statement.

Concept note-2: -Data dictionaries are used to provide detailed information about the contents of a dataset or database, such as the names of measured variables, their data types or formats, and text descriptions. A data dictionary provides a concise guide to understanding and using the data.

Concept note-3: -Explanation: Data dictionary is also called as system catalog.
