SOFTWARE ENGINEERING

[SOURCES]
Software is the set of instructions or programs that make a computer function. It cannot be touched or seen.

(A) ** Yes.

(B) No.

(C) I don’t know.

EXPLANATIONS BELOW

Concept note-1: -The term ‘software’ refers to the set of electronic program instructions or data a computer processor reads in order to perform a task or operation. In contrast, the term ‘hardware’ refers to the physical components that you can see and touch, such as the computer hard drive, mouse, and keyboard.

Concept note-2: -Software is a part of computer that cannot be touched and seen.

Concept note-3: -Software is a set of instructions, data or programs used to operate computers and execute specific tasks. It is the opposite of hardware, which describes the physical aspects of a computer. Software is a generic term used to refer to applications, scripts and programs that run on a device.

Concept note-4: -The components that can be seen or touched are called hardware of the computer.

Concept note-5: -The software is the part of the computer which we cannot see and touch but it is presented on the computer. It is a general term which is used to start the various types of programs which are used in the computer. It works inside the computer so that’s why we cannot see and touch the software.
