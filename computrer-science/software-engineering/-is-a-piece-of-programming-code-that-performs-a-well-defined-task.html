SOFTWARE ENGINEERING

[SOURCES]
____ is a piece of programming code that performs a well-defined task.

(A) . Computer Program

(B) . Computer Software

(C) ** . Both A & B

(D) . None of the above

EXPLANATIONS BELOW

Concept note-1: -A computer program is piece of programming code which performs a well defined task where as software includes programming code, its documentation and user guide.

Concept note-2: -In computer programming, computer code refers to the set of instructions, or a system of rules, written in a particular programming language (i.e., the source code).

Concept note-3: -Computer Program is piece of programming code which performs well defined task.

Concept note-4: -Today, software takes on a dual role. It is a product and, at the same time, the vehicle for delivering a product. As a product, it delivers the computing potential embodied by computer hardware or, more broadly, a network of computers that are accessible by local hardware.

Concept note-5: -Software paradigms refer to the methods and steps, which are taken while designing the software. There are many methods proposed and are in work today, but we need to see where in the software engineering these paradigms stand.
