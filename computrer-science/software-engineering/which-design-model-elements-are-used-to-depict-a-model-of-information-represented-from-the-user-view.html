Which design model elements are used to depict a model of information represented from the user view?

(A) Architectural design elements

(B) Component-level design elements

(C) ** Data design elements

(D) . Interface design elements

EXPLANATIONS BELOW

Concept note-1: -Data design elements The data design element produced a model of data that represent a high level of abstraction. This model is then more refined into more implementation specific representation which is processed by the computer based system.

Concept note-2: -Designing a model is an important phase and is a multi-process that represent the data structure, program structure, interface characteristic, and procedural details. It is mainly classified into four categories – Data design, architectural design, interface design, and component-level design.

Concept note-3: -Like other software engineering activities, data design (sometimes referred to as data architecting) creates a model of data and/or information that is represented at a high level of abstraction (the customer/user’s view of data).

Concept note-4: -4 Component-Level Design Elements: The component-level design for software is the equivalent to a set of detailed drawings (and specifications) for each room in a house.
