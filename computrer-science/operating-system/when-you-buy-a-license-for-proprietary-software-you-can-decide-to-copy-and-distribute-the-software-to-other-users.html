When you buy a license for proprietary software, you can decide to copy and distribute the software to other users.

(A) True

(B) ** False

EXPLANATIONS BELOW

Concept note-1: -When you buy a license for proprietary software, you can decide to copy and distribute the software to other users. Selected: TrueThis answer is incorrect. You can create multiple virtual desktops at any time, but you must sign in as a different user to use a different virtual desktop.

Concept note-2: -A proprietary license grants the licensee the right to use one or more copies of the software, but ownership of those copies remains with the software publisher.

Concept note-3: -A proprietary software license gives the copyright owner several exclusive rights, including the right to: Sell, distribute, or lease the software program. Control how and where the software program is used.

Concept note-4: -Software piracy is the unauthorized use, copying or distribution of copyrighted software. It may take many forms, including: Unauthorized copying of software programs purchased legitimately, sometimes known as “end-user” piracy. Gaining illegal access to protected software, also known as “cracking"

Concept note-5: -A code supplied with a software license that you must enter when you install the software and that helps a software publisher verify that the software was legally purchased.
