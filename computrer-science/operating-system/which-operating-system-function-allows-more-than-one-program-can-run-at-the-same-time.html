Which Operating System function allows more than one program can run at the same time?

(A) User Interface

(B) ** Memory Management

(C) User Management

(D) File Management

EXPLANATIONS BELOW

Concept note-1: -An operating system that allows multiple programmes to run simultaneously on a single processor machine is known as a multiprogramming operating system. The other programmes are prepared to use the CPU while one programme waits for an input/output transfer.

Concept note-2: -A multitasking operating system (also called multiprocessing operating system) supports two or more active processes simultaneously.

Concept note-3: -multiprocessing, in computing, a mode of operation in which two or more processors in a computer simultaneously process two or more different portions of the same program (set of instructions).
