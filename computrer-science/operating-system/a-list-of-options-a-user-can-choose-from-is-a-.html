A list of options a user can choose from is a ____

(A) list

(B) tab

(C) ** menu

(D) form

EXPLANATIONS BELOW

Concept note-1: -A select list is a form element that allows the user to select one or more options from a range of options. Select lists are created using a combination of the HTML <select> and <option> tags. Select lists can be nested inside a <form> element or they can stand alone.

Concept note-2: -F12 button is used to select option.

Concept note-3: -Answer. Answer: Drop down list controls is used to display a list of options from which a user can choose one or more options.

Concept note-4: -List box The list box control displays a list of values or choices.
