Which of the following is a unique combination of characters, such as letters of the alphabet and/or numbers, that identifies one specific user?

(A) Client

(B) ** Username

(C) Password

(D) Cycle

EXPLANATIONS BELOW

Concept note-1: -A user name, or user ID (identification), is a unique combination of characters, such as letters of the alphabet or numbers that identifies one specific user. A password is a private combination of characters associated with the user name that allows access to certain computer resources.

Concept note-2: -A username is a a word, phrase, number or combination of characters that uniquely identifies a user on software, a website, a computer or any computing device or related service that requires user authentication.
