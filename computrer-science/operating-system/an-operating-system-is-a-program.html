An operating system is a program

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -Answer and Explanation: The statement is True. An operating system is a set of programs that manages and controls the computer hardware, program files, data, and other system resources and provides means for the user to control the computer using a graphical user interface (GUI).

Concept note-2: -An operating system is a program that acts as an interface between the computer user and computer hardware, and controls the execution of programs.

Concept note-3: -True, The operating system is system software that performs all basic tasks of a computer system. like: hardware and software management, file management, memory management, etc. operating system is known as resource manager because it manages all resources connected to the computer.

Concept note-4: -Hence false is the correct answer to the above question.
