Windows are

(A) Symbols of application

(B) ** Rectangular area on screen

(C) Graphical User Interface

(D) All of the above

EXPLANATIONS BELOW

Concept note-1: -Window refers to the rectangular area for displaying information and running programs.

Concept note-2: -Window is a typical rectangular area in an application or a document. It is an area on the screen that displays information for a specific program.

Concept note-3: -When a new program is started, an area on the screen is used to display the feedback or output from the program. This area is known as a window.

Concept note-4: -Window is the name given to the rectangular area in an application or a document.

Concept note-5: -In computing, a window is a graphical control element. It consists of a visual area containing some of the graphical user interface of the program it belongs to and is framed by a window decoration. It usually has a rectangular shape that can overlap with the area of other windows.
