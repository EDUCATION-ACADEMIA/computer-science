BIOS is a part of which function?

(A) Memory Management

(B) ** Bootstrap Process

(C) Process Management

(D) System Restore

EXPLANATIONS BELOW

Concept note-1: -BIOS identifies, configures, tests and connects computer hardware to the OS immediately after a computer is turned on. The combination of these steps is called the boot process. These tasks are each carried out by BIOS’ four main functions: Power-on self-test (POST).

Concept note-2: -All modern computer motherboards contain BIOS software. BIOS access and configuration are independent of any operating system because the BIOS is part of the motherboard hardware.

Concept note-3: -So, to recap, BIOS is firmware (in other words, software embedded in a piece of hardware) stored on a ROM chip that lets you access and set up your system at its most basic level.

Concept note-4: -The BIOS includes instructions on how to load basic computer hardware. The BIOS also includes a test referred to as a POST (Power On Self Test) which will ensure that the computer meets requirements to boot up properly.

Concept note-5: -In computing, BIOS (/ˈbaɪɒs, -oʊs/, BY-oss, -⁠ohss; Basic Input/Output System, also known as the System BIOS, ROM BIOS, BIOS ROM or PC BIOS) is firmware used to provide runtime services for operating systems and programs and to perform hardware initialization during the booting process (power-on startup).
