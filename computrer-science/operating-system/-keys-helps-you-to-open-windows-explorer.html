____ keys helps you to open Windows Explorer.

(A) ** Win + E

(B) Ctrl + E

(C) E + Win

(D) Shift + E

EXPLANATIONS BELOW

Concept note-1: -File Explorer in Windows 11 helps you get the files you need quickly and easily. To check it out in Windows 11, select it on the taskbar or the Start menu, or press the Windows logo key + E on your keyboard.

Concept note-2: -Windows logo key + E. Open File Explorer. Windows logo key + F. Open Feedback Hub and take a screenshot.

Concept note-3: -To open File Explorer in Windows 10, select its icon on the taskbar, press the Windows logo key + E on your keyboard, or select Start > Documents (directly beneath your user icon).
