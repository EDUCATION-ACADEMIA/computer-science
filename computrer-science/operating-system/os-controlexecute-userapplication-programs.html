OPERATING SYSTEM

[SOURCES]
O.S Control/execute user/application programs

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -When you double-click the program, the operating system “launches” the program, doing the housekeeping steps of allocating an area of memory within RAM for the program, loading the first section of the program’s machine code into that memory, and finally directing the CPU to start running that code.

Concept note-2: -An operating system may perform the functions of input/output control, resource scheduling, and data management. It provides application programs with the fundamental commands for controlling the computer.

Concept note-3: -The operating system’s job The operating system (OS) manages all of the software and hardware on the computer. It performs basic tasks such as file, memory and process management, handling input and output, and controlling peripheral devices such as disk drives and printers.

Concept note-4: -The CPU reads the instruction from the data bus into an instruction register. The CPU then decodes and executes the instruction (3).
