Which operating system is the Android OS based off of?

(A) Mac OSX

(B) ** Linux

(C) Unix

(D) Windows NT Kernel

EXPLANATIONS BELOW

Concept note-1: -Android OS is a Linux-based mobile operating system that primarily runs on smartphones and tablets. The Android platform includes an operating system based upon the Linux kernel, a GUI, a web browser and end-user applications that can be downloaded.

Concept note-2: -Android is based on Linux and is an open source mobile operating system developed by the Open Handset Alliance led by Google.

Concept note-3: -Android is an open source, Linux-based software stack created for a wide array of devices and form factors. The following diagram shows the major components of the Android platform.

Concept note-4: -The Android kernel is based on an upstream Linux Long Term Supported (LTS) kernel. At Google, LTS kernels are combined with Android-specific patches to form what are known as Android Common Kernels (ACKs). Figure 1.

Concept note-5: -Android, which is based on the Linux kernel, has become the dominant operating system for smartphones. In July 2022, 71.9% of smartphones accessing the internet worldwide used Android.
