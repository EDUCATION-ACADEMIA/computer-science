The ability for a computer to execute more than one task simultaneously

(A) multitasking

(B) multiprogramming

(C) ** multiprocessing

(D) timesharing

EXPLANATIONS BELOW

Concept note-1: -In computing, multitasking is the concurrent execution of multiple tasks (also known as processes) over a certain period of time. New tasks can interrupt already started ones before they finish, instead of waiting for them to end.

Concept note-2: -A system is said to be concurrent if it can support two or more actions in progress at the same time. A system is said to be parallel if it can support two or more actions executing simultaneously.

Concept note-3: -Multitasking, in an operating system, is allowing a user to perform more than one computer task (such as the operation of an application program) at a time.

Concept note-4: -Multiprocessing is a general term that can mean the dynamic assignment of a program to one of two or more computers working in tandem or can involve multiple computers working on the same program at the same time (in parallel).
