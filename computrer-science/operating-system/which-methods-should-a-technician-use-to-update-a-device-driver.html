Which methods should a technician use to update a device driver?

(A) Insert the operating system installation media and load the driver from there

(B) Buy another operating system license

(C) ** Download the driver from the website of the manufacturer

(D) Perform a firmware update

EXPLANATIONS BELOW

Concept note-1: -Updating and removing device drivers Start Device Manager by typing ‘Devmgmt. msc’ in the Run dialog box. Right-click the desired device and select Properties. Click the ‘Driver’ tab and then click the Update Driver button.

Concept note-2: -Driver updates fix security and compatibility problems, errors, broken code, and sometimes even add features to the hardware.
