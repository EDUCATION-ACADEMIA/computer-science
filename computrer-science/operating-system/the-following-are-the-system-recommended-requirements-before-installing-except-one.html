[SOURCES]
The following are the system recommended requirements before installing except one.

(A) 1 GHz 32-bit or 64-bit processor

(B) 1 GB of system memory

(C) ** High Definition Audio Device

(D) 16 GB of available disk space

EXPLANATIONS BELOW

Concept note-1: -Recommended System Requirements shall be the minimum technical specifications of the combination of computer hardware and equipment, peripherals, commercial off-the-shelf software, telecommunications and/or network connections, security firewalls, etc.

Concept note-2: -What Does System Requirements Mean? System requirements are the configuration that a system must have in order for a hardware or software application to run smoothly and efficiently. Failure to meet these requirements can result in installation problems or performance problems.

Concept note-3: -Windows Server 2022 hardware requirements and limitations Both require a 1.4 GHz, 64-bit CPU and 512 MB of RAM; for the Desktop Experience GUI-based version, the system needs 2 GB of RAM. Additionally, 32 GB of disk space are required.
