What acts as the master controller for all activities that take place within a computer system?

(A) Hardware

(B) CPU

(C) Application software

(D) ** Operating System

EXPLANATIONS BELOW

Concept note-1: -The master controller of all the activities that take place in a computer is the operating system. Operating system acts as interface between the user and the computer.

Concept note-2: -4 1 Operating System Activities An operating system is a type of system software that acts as the master controller for all activities that take place.

Concept note-3: -Control Master uses a Master–Slave principle of operation. In all systems there is one “Master Controller” board which is connected to a computer via a standard USB lead and up to 30 slave boards. The job of the Master is to allow the PC to send and receive data to any of the slave boards.

Concept note-4: -System software controls a computer’s internal functioning, chiefly through an operating system, and also controls such peripherals as monitors, printers, and storage devices.

Concept note-5: -The operating system is the master controller of the computer, the glue that holds together all the components of the system, including the administrators, programmers, and users.
