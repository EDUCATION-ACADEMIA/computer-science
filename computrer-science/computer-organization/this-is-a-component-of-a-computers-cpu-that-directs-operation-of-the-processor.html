This is a component of a computer’s CPU that directs operation of the processor.

(A) ** Control Unit (CU)

(B) Arithmetic and Logic Unit (ALU)

(C) Bus

(D) Memory

EXPLANATIONS BELOW

Concept note-1: -The CPU control unit is a component of the computer’s CPU (central processing unit) that directs the processor’s operation. John von Neumann included it in his Von Neumann Architecture.

Concept note-2: -The control unit (CU) is a component of the CPU that directs the operation of the processor. It tells the computer’s memory, arithmetic and logic unit and input and output devices how to respond to the instructions that have been sent to the processor.

Concept note-3: -The ALU performs arithmetic operations, logic operations, and related operations, according to the program instructions. The control unit controls all CPU operations, including ALU operations, the movement of data within the CPU, and the exchange of data and control signals across external interfaces (system bus).

Concept note-4: -The control unit (CU) is a component of a computer’s central processing unit (CPU) that directs the operation of the processor.

Concept note-5: -Control Unit of a digital computer is often called the Nerve centre. It directs the operation of the other units by providing timing and control signals. Most computer resources are managed by the CU. It directs the flow of data between the CPU and the other devices.
