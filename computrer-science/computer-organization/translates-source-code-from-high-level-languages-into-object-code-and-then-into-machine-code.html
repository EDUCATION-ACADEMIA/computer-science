[SOURCES]
Translates source code from high level languages into object code and then into machine code

(A) Assembler

(B) ** Compiler

(C) Interpreter

EXPLANATIONS BELOW

Concept note-1: -The correct answer is Compiler. A compiler takes the program code (source code) and converts the source code to a machine language module (called an object file).

Concept note-2: -Compilers are a type of translator that support digital devices, primarily computers. The name compiler is primarily used for programs that translate source code from a high-level programming language to a lower level language (e.g., assembly language, object code, or machine code) to create an executable program.

Concept note-3: -A compiler takes the source code as a whole and translates it into object code all in one go. Once converted, the object code can be run at any time. This process is called compilation . All of the object files used in a program must be combined before the program can be run.

Concept note-4: -To get from source code to machine language, the programs must be transformed by a compiler. The compiler produces an intermediary form called object code. Object code is often the same as or similar to a computerâ€™s machine language.

Concept note-5: -A compiler translates a program written in a high-level language into Machine language or that transforms computer code written in one programming language into another programming language.
