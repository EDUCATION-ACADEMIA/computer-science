The control unit controls other units by generating ____

(A) CONTROL SIGNALS

(B) ** TIMING SIGNALS

(C) TRANSFER SIGNALS

(D) COMMAND SIGNALS

EXPLANATIONS BELOW

Concept note-1: -The control unit controls other units by generating a)Control signalsb)Timing signalsc)Transfer signalsd)Command SignalsCorrect answer is option ‘B’. Can you explain this answer? The control unit (CU) is a component of a computer’s central processing unit (CPU) that directs the operation of the processor.

Concept note-2: -It fetches internal instructions of the programs from the main memory to the processor instruction register, and based on this register contents, the control unit generates a control signal that supervises the execution of these instructions.

Concept note-3: -Explanation: The main task of a control unit is to generate control signals.

Concept note-4: -The control unit controls and monitors communications between the hardware attached to the computer. It controls the input and output of data, checks that signals have been delivered successfully, and makes sure that data goes to the correct place at the correct time.

Concept note-5: -We use Timing and Controlling unit in 8085 for the generation of timing signals and the signals to control. All the operations and functions both interior and exterior of a microprocessor are controlled by this unit.
