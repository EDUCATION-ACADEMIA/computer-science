C, C++, Java languages are

(A) ** High Level Language

(B) Assembly Language

(C) Machine Language

EXPLANATIONS BELOW

Concept note-1: -They are very widely used and popular in today’s times. Java, C, C++, Python, etc., are a few examples of high-level languages.

Concept note-2: -C language is one of the most popular and oldest computer programming languages because it is a structured, machine-independent language and a high-level language.

Concept note-3: -Java is a high-level, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible.

Concept note-4: -C is a procedural, low level, and compiled language. Java is an object-oriented, high level, and interpreted language. Java uses objects, while C uses functions. Java is easier to learn and use because it’s high level, while C can do more and perform faster because it’s closer to machine code.

Concept note-5: -C language is a type of Procedural Programming language, whereas Java is a type of Object-Oriented language. C language is a middle-level language, but Java is a high-level language.
