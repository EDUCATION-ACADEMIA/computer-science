(A) ALU

(B) Control Unit

(C) ** Cache

EXPLANATIONS BELOW

Concept note-1: -Cache memory is the fastest system memory, required to keep up with the CPU as it fetches and executes instructions. The data most frequently used by the CPU is stored in cache memory.

Concept note-2: -Cache memory operates between 10 to 100 times faster than RAM, requiring only a few nanoseconds to respond to a CPU request. The name of the actual hardware that is used for cache memory is high-speed static random access memory (SRAM).

Concept note-3: -cache memory, also called cache, supplementary memory system that temporarily stores frequently used instructions and data for quicker processing by the central processing unit (CPU) of a computer. The cache augments, and is an extension of, a computerâ€™s main memory.

Concept note-4: -Cache memory is a type of high-speed random access memory (RAM) which is built into the processor . Data can be transferred to and from cache memory more quickly than from RAM. As a result, cache memory is used to temporarily hold data and instructions that the processor is likely to reuse.
