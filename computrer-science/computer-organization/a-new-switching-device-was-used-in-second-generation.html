A new switching device was used in second generation

(A) Vacuum tubes

(B) ** Transistors

(C) Diodes

EXPLANATIONS BELOW

Concept note-1: -A transistor computer, now often called a second-generation computer, is a computer which uses discrete transistors instead of vacuum tubes.

Concept note-2: -A transistor computer, now often called a second generation computer, is a computer which uses discrete transistors instead of vacuum tubes.

Concept note-3: -Vacuum tubes replaced transistors in the second generation | KnowledgeBoat.

Concept note-4: -In this generation, transistors were used that were cheaper, consumed less power, more compact in size, more reliable and faster than the first generation machines made of vacuum tubes. In this generation, magnetic cores were used as the primary memory and magnetic tape and magnetic disks as secondary storage devices.
