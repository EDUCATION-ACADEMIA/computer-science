Can a decoder be used as a De-multiplexer?

(A) ** Yes

(B) No

EXPLANATIONS BELOW

Concept note-1: -A decoder with an enable input can function as a Demultiplexer. A demultiplexer is a circuit that receives information on a single line and transmits this information on one of 2n possible output lines.

Concept note-2: -A combinational logic circuit which has 2n input lines and only 1 output lines is called multiplexer or MUX. A combinational logic circuit which has ‘n’ input lines and maximum 2n output lines is known as decoder.

Concept note-3: -In particular, it includes internal logic that buffers one of the inputs as a TTL-level output on the output pin, and as a result, it cannot be used as a demultiplexer.

Concept note-4: -An encoder with enable pins is called multiplexer while a decoder with enable pins is called demultiplexer.
