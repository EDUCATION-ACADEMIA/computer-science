The format of a file can be known from its file ____

(A) ** extension

(B) file format

(C) file extension

EXPLANATIONS BELOW

Concept note-1: -A particular file format is often indicated as part of a fileâ€™s name by a file name extension (suffix). Conventionally, the extension is separated by a period from the name and contains three or four letters that identify the format.

Concept note-2: -The extension is a three-or four-letter abbreviation that signifies the file type. For example, in letter. docx the filename is letter and the extension is docx. Extensions are important because they tell your computer what icon to use for the file, and what application can open the file.

Concept note-3: -Right-click the file. Select the Properties option. In the Properties window, similar to what is shown below, see the Type of file entry, which is the file type and extension.

Concept note-4: -Detailed Solution. A filename extension is an identifier specified as a suffix to the name of a computer file. The extension indicates a characteristic of the file contents or its intended use.
