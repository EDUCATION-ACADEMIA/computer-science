What is the purpose of Utility Software? Click all that apply

(A) ** To keep the computer organised

(B) ** To ensure it has up to date software

(C) ** To ensure the system runs efficiently

(D) To translate machine code into high level code

EXPLANATIONS BELOW

Concept note-1: -Utility software is software designed to help analyze, configure, optimize or maintain a computer. It is used to support the computer infrastructure-in contrast to application software, which is aimed at directly performing tasks that benefit ordinary users.

Concept note-2: -Utility software helps users configure, analyse, optimise and maintain a computer. This software normally consists of small programs which are thought of as part of the operating system (OS) as they often come built in with the OS.

Concept note-3: -operating system (OS), program that manages a computerâ€™s resources, especially the allocation of those resources among other programs. Typical resources include the central processing unit (CPU), computer memory, file storage, input/output (I/O) devices, and network connections.

Concept note-4: -Antivirus, backup software, file manager, disk compression tool all are utility software.
