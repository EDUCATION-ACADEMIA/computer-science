Typically create vector graphics, which use mathematical formulas to represent image content instead of pixels

(A) Painting Program

(B) ** Drawing Program

(C) Image Editing Program

(D) Audio Capture and Editing Program

EXPLANATIONS BELOW

Concept note-1: -Vector images are based on geometrical formulas (mathematical equations) based on paths, instead of pixels, to represent images. Because of that, you can edit it, resizing and changing colors without any loss of resolution. This makes them ideal for illustrations, icons, and logos.

Concept note-2: -Raster images are compiled using pixels, or tiny dots, containing unique color and tonal information that come together to create the image.

Concept note-3: -Unlike bitmap images, vector images are not made up of pixels. Instead, they are made up of points, curves, and paths defined by mathematical objects called vectors. Vector images can be scaled up or down with no loss in image quality (they are resolution independent).

Concept note-4: -System software includes all the programs that allow you to perform specific tasks on a computer.
