A vector image is created using

(A) coloured pixels.

(B) dots.

(C) ** mathematics formulas.

(D) images.

EXPLANATIONS BELOW

Concept note-1: -The logical data model of vector graphics is based on the mathematics of coordinate geometry, in which shapes are defined as a set of points in a two-or three-dimensional cartesian coordinate system, as p = (x, y) or p = (x, y, z).

Concept note-2: -Vector graphics use mathematical equations to draw out your designs. These mathematical equations are translated into points that are connected by either lines or curves, also known as vector paths, and they make up all the different shapes you see in a vector graphic.

Concept note-3: -Vector (also known as “object-oriented") graphics are constructed using mathematical formulas describing shapes, colors, and placement. Rather than a grid of pixels, a vector graphic consists of shapes, curves, lines, and text which together make a picture.

Concept note-4: -MAGNITUDE AND DIRECTION OF A VECTOR Given a position vector →v=⟨a, b⟩, the magnitude is found by |v|=√a2+b2. The direction is equal to the angle formed with the x-axis, or with the y-axis, depending on the application.

Concept note-5: -Vector artwork is a term that describes any art made with vector illustration software like Adobe Illustrator. Vector artwork is built from vector graphics, which are images created with mathematical formulas. In comparison, raster art (also referred to as bitmaps or raster images) is created with colorised pixels.
