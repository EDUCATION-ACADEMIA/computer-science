Which of the following statements is true regarding cropping or resizing an image in graphic editing software?

(A) Cropping an image keeps the unused portion of the image.

(B) Resizing an image deletes the unused portion of the image.

(C) ** Cropping an image reduces its file size.

(D) Resizing an image does not reduce its file size.

EXPLANATIONS BELOW

Concept note-1: -Which of the following statements is true regarding cropping or resizing an image? Resizing an image retains the entire image and reduces the display size.

Concept note-2: -Raster Images These are the types of images that are produced when scanning or photographing an object. Raster images are compiled using pixels, or tiny dots, containing unique color and tonal information that come together to create the image. Since raster images are pixel based, they are resolution dependent.

Concept note-3: -Graphics Software are used to create, edit, compose, and render images and illustrations. Graphics Software can support free drawing, image and photo editing, and combining and compositing multiple images. They can also be used to convert image file types.

Concept note-4: -EPS (short for Encapsulated PostScript) is a vector format designed for printing to PostScript printers and imagesetters. It is considered the best choice of graphics format for high resolution printing of illustrations. EPS files are created and edited in illustration programs such as Adobe Illustrator or CorelDRAW.
