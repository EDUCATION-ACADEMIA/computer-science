Which of this is compulsory for 2D reflection.

(A) Origin

(B) Reflection axis

(C) Co-ordinate axis.

(D) ** Reflection plane.

EXPLANATIONS BELOW

Concept note-1: -1 Answer. Best explanation: Reflection axis is the axis with respect to which reflection is done.

Concept note-2: -Abhishek Mishra. Answer: b Explanation: A two-dimensional plane is used for 2D transformations. Transformations are useful for modifying an object’s position, size, orientation, and shape, among other things.

Concept note-3: -Reflection is a kind of rotation where the angle of rotation is 180 degree. The reflected object is always formed on the other side of mirror. The size of reflected object is same as the size of original object.

Concept note-4: -Explanation: Reflection axis is the axis with respect to which reflection is done. In 3D, it is reflection plane rather than reflection axis.

Concept note-5: -Reflection is the mirror image of original object. In other words, we can say that it is a rotation operation with 180°.
