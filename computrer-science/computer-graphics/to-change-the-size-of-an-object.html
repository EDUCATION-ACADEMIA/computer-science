To change the size of an object.

(A) Skew

(B) ** Scale

(C) Distort

(D) Crop

EXPLANATIONS BELOW

Concept note-1: -Resizing objects. You can resize an object by dragging its handles, scaling it by a certain percentage, or typing exact dimensions.

Concept note-2: -Scaling is a kind of transformation which is used to change the size of the object.

Concept note-3: -Scaling an object means multiplying every linear dimension of it by the same factor. Thus you change the size of the object, but not its shape.

Concept note-4: -The process of resizing or transforming an object is called dilation. It is a transformation that makes the objects smaller or larger with the help of the given scale factor.
