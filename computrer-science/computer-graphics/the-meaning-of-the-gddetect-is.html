Concept note-1: -The two int variables gd, gm are graphic driver and graphic mode respectively. The gm handles value that tells us which resolution and monitor we are using. The gd specifies the graphic driver to be used. In our program we have gd=DETECT means we have passed the highest possible value available for the detected driver.

Concept note-2: -initgraph initializes the graphics system by loading a graphics driver from disk (or validating a registered driver), and putting the system into graphics mode. To start the graphics system, first call the initgraph function. initgraph loads the graphics driver and puts the system into graphics mode.

Concept note-3: -graphicsDriver : It is a pointer to an integer specifying the graphics driver to be used. It tells the compiler that what graphics driver to use or to automatically detect the drive. In all our programs we will use DETECT macro of graphics. h library that instruct compiler for auto detection of graphics driver.
