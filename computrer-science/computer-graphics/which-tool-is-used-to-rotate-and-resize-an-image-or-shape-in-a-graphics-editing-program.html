Which tool is used to rotate and resize an image or shape in a graphics editing program?

(A) Move

(B) Free Distort

(C) Crop

(D) ** Free Transform

EXPLANATIONS BELOW

Concept note-1: -The Free Transform tool allows you to manipulate objects and images. For example, you can distort, rotate, reflect, shear, or resize artwork using the Free Transform tool.

Concept note-2: -The Free Transform tool allows you to select an object on the Stage and then interactively change its size or rotate it freely.

Concept note-3: -The Free Transform tool is one of the primary tools in Fresco that lets you easily modify your artwork by applying transformations like scale, skew, perspective, and more.

Concept note-4: -To use the Free Transform feature, select a layer or create a new selections. Got to Edit in the menu, select Free Transform. A bounding box will appear around your selection. You can now transform the selection using the bounding box handles.

Concept note-5: -Use the Transform and Free Transform tools. When you want to rotate an image and make other kinds of transformations at the same time, look no further than the Free Transform tool. Or you can go to Edit > Transform to choose the exact transformation you want to make, like Scale, Skew, Distort, Warp, and more.
