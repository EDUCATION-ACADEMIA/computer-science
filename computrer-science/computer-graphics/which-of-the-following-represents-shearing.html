Which of the following represents shearing?

(A) (x, y) → (x cos()+y sin(), -x sin()+y cos())

(B) (x, y) → (ax, by)

(C) ** (x, y) → (x+ay, y+bx)

(D) (x, y) → (x+a, y+b)

EXPLANATIONS BELOW

Concept note-1: -6. Which of the following represents shearing? Explanation: The first one represent translation, the second one represents scaling, third one rotation and the last one is representing shearing.

Concept note-2: -A transformation that slants the shape of an object is called the shear transformation. There are two shear transformations X-Shear and Y-Shear. One shifts X coordinates values and other shifts Y coordinate values. However; in both the cases only one coordinate changes its coordinates and other preserves its values.

Concept note-3: -Shearing in X axis can be done using this equation, Xnew = Xold + Shx x Y. Ynew = Y.

Concept note-4: -It is transformation which changes the shape of object. The sliding of layers of object occur. The shear can be in one direction or in two directions. Shearing in the X-direction: In this horizontal shearing sliding of layers occur.
