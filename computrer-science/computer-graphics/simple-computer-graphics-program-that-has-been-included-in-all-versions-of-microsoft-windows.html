[SOURCES]
simple computer graphics program that has been included in all versions of Microsoft Windows.

(A) ** Paint

(B) Kolour paint

(C) pinta

EXPLANATIONS BELOW

Concept note-1: -Microsoft Paint is a simple raster graphics editor that has been included with all versions of Microsoft Windows. The program opens, modifies and saves image files in Windows bitmap (BMP), JPEG, GIF, PNG, and single-page TIFF formats.

Concept note-2: -A paint program is a software graphics program that allows the user to draw, color, or paint bitmapped images on a computer. Though most are discontinued, examples include: KolourPaint, XPaint, Deluxe Paint, MacPaint, and Microsoft Paint, which is shown in the picture.

Concept note-3: -Paint has been included with every version of Microsoft Windows since its inception. Paint provides basic functionality for drawing and painting in color or black and white, and shaped stencils and cured line tools to create geometrical designs.

Concept note-4: -Ans. There are two types of selection tools in MS Paint. The rectangular selection and Free Form Selection tool.
