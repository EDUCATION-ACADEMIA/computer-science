What are the three different type of data you should use for testing?

(A) ** Normal, boundary, extreme

(B) Novelle, bound, exponential

(C) Neat, biased, exempt

(D) New, broad, extraordinary

EXPLANATIONS BELOW

Concept note-1: -System modules should be tested using a testing plan with normal, extreme and abnormal test data.

Concept note-2: -Some of the types of test data included in this method are valid, invalid, null, standard production data, and data set for performance.

Concept note-3: -Extreme data is test data at the upper or lower limits of expectations that should be accepted by the system.
