How many dimensions is the maximum in an array

(A) ** 32

(B) 16

(C) 8

(D) 4

EXPLANATIONS BELOW

Concept note-1: -Although an array can have as many as 32 dimensions, it is rare to have more than three.

Concept note-2: -A 3D array is a multi-dimensional array(array of arrays). A 3D array is a collection of 2D arrays . It is specified by using three subscripts:Block size, row size and column size. More dimensions in an array means more data can be stored in that array.

Concept note-3: -There is no fixed limit to the size of an array in C.

Concept note-4: -A two-dimensional array is a data structure that contains a collection of cells laid out in a two-dimensional grid, similar to a table with rows and columns although the values are still stored linearly in memory.

Concept note-5: -Prerequisite :Array in C/C++, More on array A four-dimensional (4D) array is an array of array of arrays of arrays or in other words 4D array is a array of 3D array. More dimensions in an array means more data be held, but also means greater difficulty in managing and understanding arrays.
