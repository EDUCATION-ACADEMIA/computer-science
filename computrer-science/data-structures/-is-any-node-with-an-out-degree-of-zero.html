____ is any node with an out-degree of zero.

(A) root

(B) internal node

(C) parent

(D) ** leaf

EXPLANATIONS BELOW

Concept note-1: -Leaf node: A node with outdegree zero is known as leaf node. A leaf node is also known as terminal node.

Concept note-2: -no. of children a node in binary tree is 2. Hence, leaf node is of degree 0.

Concept note-3: -Leaf Nodes In a directed tree with nodes that has an out-degree of 0 is called a Leaf node or Terminal node.

Concept note-4: -The out-degree of a node is equal to the number of edges with that node as the source.

Concept note-5: -A tree is a nonlinear data structure, compared to arrays, linked lists, stacks and queues which are linear data structures. A tree can be empty with no nodes or a tree is a structure consisting of one node called the root and zero or one or more subtrees.
