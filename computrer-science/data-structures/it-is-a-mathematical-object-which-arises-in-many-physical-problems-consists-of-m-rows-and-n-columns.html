It is a mathematical object which arises in many physical problems, consists of m rows and n columns

(A) Array

(B) ** Matrix

(C) Table

(D) Excel

EXPLANATIONS BELOW

Concept note-1: -A matrix with m rows and n columns is called an m × n matrix, or m-by-n matrix, while m and n are called its dimensions. For example, the matrix A above is a 3 × 2 matrix. Matrices with a single row are called row vectors, and those with a single column are called column vectors.

Concept note-2: -An m × n matrix: the m rows are horizontal and the n columns are vertical. Each element of a matrix is often denoted by a variable with two subscripts. For example, a2, 1 represents the element at the second row and first column of the matrix.

Concept note-3: -The matrices are a two-dimensional set of numbers or symbols distributed in a rectangular shape in vertical and horizontal lines so that their elements are arranged in rows and columns. They are useful for describing systems of linear or differential equations, as well as representing a linear application.

Concept note-4: -matrix, a set of numbers arranged in rows and columns so as to form a rectangular array. The numbers are called the elements, or entries, of the matrix. Matrices have wide applications in engineering, physics, economics, and statistics as well as in various branches of mathematics.
