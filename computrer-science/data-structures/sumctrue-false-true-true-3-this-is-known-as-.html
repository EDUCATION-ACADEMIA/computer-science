sum(c(TRUE, FALSE, TRUE, TRUE))> 3 This is known as ____

(A) Vectorisation

(B) Recycling

(C) ** Implicit coercion

(D) Attributes

EXPLANATIONS BELOW

Concept note-1: -The c() function in R is used to combine or concatenate its argument.

Concept note-2: -Applied to a logical vector v, sum() tells us how many elements in v are TRUE .

Concept note-3: -In R, true values are designated with TRUE, and false values with FALSE. When you index a vector with a logical vector, R will return values of the vector for which the indexing vector is TRUE.
