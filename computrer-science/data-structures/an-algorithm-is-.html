An algorithm is ____

(A) ** A set of steps to follow in order

(B) When you sort data into the correct order

(C) When you search data for a specific entity

EXPLANATIONS BELOW

Concept note-1: -The order of growth of an algorithm is an approximation of the time required to run a computer program as the input size increases. The order of growth ignores the constant factor needed for fixed operations and focuses instead on the operations that increase proportional to input size.

Concept note-2: -An algorithm is a procedure used for solving a problem or performing a computation. Algorithms act as an exact list of instructions that conduct specified actions step by step in either hardware-or software-based routines.

Concept note-3: -An algorithm is a defined set of step-by-step procedures that provides the correct answer to a particular problem. By following the instructions correctly, you are guaranteed to arrive at the right answer. An algorithm is often expressed in the form of a graph, where a square represents each step.

Concept note-4: -An algorithm is made up of three basic building blocks: sequencing, selection, and iteration.
