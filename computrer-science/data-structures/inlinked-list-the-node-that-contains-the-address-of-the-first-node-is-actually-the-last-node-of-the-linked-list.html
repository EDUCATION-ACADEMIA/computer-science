In ____ Linked list the node that contains the address of the first node is actually the last node of the linked list.

(A) singly

(B) doubly

(C) ** circular

(D) none of above

EXPLANATIONS BELOW

Concept note-1: -A circular linked list is a linked list in which last node always point to the first node i.e. last node contains the address of the first node. As in singly linked list, each node in circular linked list consists of two parts.

Concept note-2: -In a circular Singly linked list, the last node of the list contains a pointer to the first node of the list.

Concept note-3: -Circular singly linked list: In a circular Singly linked list, the last node of the list contains a pointer to the first node of the list.

Concept note-4: -Circular linked list is simply a singly or doubly linked list in which the last node or tail is pointing to the head or first node.

Concept note-5: -A variation of linked list is circular linked list, in which the last node in the list points to first node of the list.
