A node is a ____ if it has successor nodes-that is, if it has an out-degree greater than zero.

(A) child

(B) ** parent

(C) siblings

(D) none of above

EXPLANATIONS BELOW

Concept note-1: -Any node which has one or more children is a parent node. A group of nodes with the same parent is called siblings.

Concept note-2: -The ancestors of a node are all the nodes along the path from the root to that node.

Concept note-3: -If the outdegree of every node is less than or equal to 2, in a directed tree than the tree is called a binary tree.

Concept note-4: -1) The degree of root node is always zero. ii) Nodes that are not root and not leaf are called as internal nodes.
