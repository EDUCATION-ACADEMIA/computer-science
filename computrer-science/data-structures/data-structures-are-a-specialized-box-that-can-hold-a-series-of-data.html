DATA STRUCTURES

[SOURCES]
Data structures are a specialized box that can hold a series of data.

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -An array is a linear data structure that collects elements of the same data type and stores them in contiguous and adjacent memory locations.

Concept note-2: -Special data structures include: Data area data structures. File information data structures (INFDS) Program-status data structures.

Concept note-3: -You can create an array with elements of different data types when declare the array as Object. Since System. Object is the base class of all other types, an item in an array of Objects can have a reference to any other type of object.

Concept note-4: -Data Structures are a specialized means of organizing and storing data in computers in such a way that we can perform operations on the stored data more efficiently. Data structures have a wide and diverse scope of usage across the fields of Computer Science and Software Engineering.
