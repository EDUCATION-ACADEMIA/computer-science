It contains multiple tabs, each with several groups of commands.

(A) Menu Bar

(B) ** Ribbon

(C) Navigation Pane

(D) Quick access

EXPLANATIONS BELOW

Concept note-1: -The Ribbon contains all of the commands you will need in order to do common tasks. It contains multiple tabs, each with several groups of commands called “panels".

Concept note-2: -The correct answer is option 1 i.e., Ribbon. Ribbon contains multiple tabs, each with several groups of command. It consists of a number of commands which are grouped under the tabs provided below the title bar.

Concept note-3: -The Ribbon Excel uses a tabbed Ribbon system instead of traditional menus. The Ribbon contains multiple tabs, each with several groups of commands. You will use these tabs to perform the most common tasks in Excel.

Concept note-4: -The Ribbon is located close to the top of the Word window. The Ribbon is organized into a set of task-oriented tabs, and each tab on the Ribbon contains groups of commands.

Concept note-5: -The Ribbon contains all of the commands you will need to perform common tasks in PowerPoint. It has multiple tabs, each with several groups of commands.
