Data Manipulation Language enables users to

(A) Retrieval of information stored in database

(B) Insertion of new information into the database

(C) Deletion of information from the database

(D) ** All of the above

EXPLANATIONS BELOW

Concept note-1: -A data-manipulation language (DML) is a language that enables users to access or manipulate data as organized by the appropriate model.

Concept note-2: -Definition. A data manipulation language allows users to access and retrieve database contents in both interactive and embedded (in a host programming language) modes. DML includes constructs that allow retrieval and manipulation of database content.

Concept note-3: -A DML (data manipulation language) refers to a computer programming language that allows you to add (insert), delete (delete), and alter (update) data in a database. A DML is typically a sublanguage of a larger database language like SQL, with the DML containing some of the languageâ€™s operators.

Concept note-4: -A data manipulation language (DML) is a language that enables users to access and manipulate data in a Table.

Concept note-5: -Expert-Verified Answer. A Data manipulation language [DML] is a language that enables users to access and manipulate data in a database.
