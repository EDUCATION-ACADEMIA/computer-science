(A) Algorithm.

(B) Flowchart.

(C) ** Pseudocode.

(D) Website.

EXPLANATIONS BELOW

Concept note-1: -Pseudocode is a compact and informal high-level description of a computer programming algorithm that uses the structural conventions of a programming language, but is intended for human reading rather than machine reading.

Concept note-2: -Pseudocode. “Pseudocode is a compact and informal high-level. description of a computer programming algorithm. that uses the structural conventions of some. programming language, but is intended for human.

Concept note-3: -Pseudocode is an artificial and informal language that helps programmers develop algorithms.

Concept note-4: -Definition: Pseudocode is an informal way of programming description that does not require any strict programming language syntax or underlying technology considerations. It is used for creating an outline or a rough draft of a program.

Concept note-5: -Pseudocode (pronounced SOO-doh-kohd) is a detailed yet readable description of what a computer program or algorithm must do, expressed in a formally-styled natural language rather than in a programming language.
