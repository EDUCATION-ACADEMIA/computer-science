ALGORITHMS

[SOURCES]
Computers can solve any problem that can be computationally solved.

(A) ** TRUE

(B) FALSE

EXPLANATIONS BELOW

Concept note-1: -The answer is NO. For an algorithm to correctly solve the given problem, it has to produce the correct result for all valid inputs to the problem. If we can find one counterexample, one input where the algorithm does not produce the correct output, then the algorithm is incorrect.

Concept note-2: -Not only is that scientific achievement attributed to him, but also a lot more fantastic scientific achievements. One prominent example of a fantastic scientific achievement that was completed by Alan Turing was proving that computers are not capable of solving every problem that mankind had to offer it.

Concept note-3: -A computational problem is a task solved by a computer. A computation problem is solvable by mechanical application of mathematical steps, such as an algorithm. A problem is regarded as inherently difficult if its solution requires significant resources, whatever the algorithm used.
