(A) ** the steps are clear

EXPLANATIONS BELOW

Concept note-1: -Algorithm: definition An algorithm is an ordered set of unambiguous, executable steps that defines a terminating process.

Concept note-2: -Explanation: An algorithm becomes a program when it is written in the form of a programming language. Thus, any program is an algorithm.
