Concept note-1: -With if statements, our programs can execute a set of instructions only if the condition is true. If we want our programs to execute a different set of instructions when the condition is false, then we can use an else statement.

Concept note-2: -One go-to programming language commonly used by developers all over the world is C++. In fact, most video games, embedded systems, IoT devices, or resource-heavy AI applications run on C++.

Concept note-3: -An algorithm is a set of steps for completing a task or solving a problem.

Concept note-4: -abstraction – focusing on the important information only, ignoring irrelevant detail. algorithms-developing a step-by-step solution to the problem, or the rules to follow to solve the problem.
