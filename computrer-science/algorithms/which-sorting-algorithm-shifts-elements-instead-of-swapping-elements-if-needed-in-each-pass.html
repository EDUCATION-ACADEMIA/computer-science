Which sorting algorithm shifts elements instead of swapping elements if needed in each pass.

(A) Bubble Sort

(B) Selection Sort

(C) ** Insertion Sort

EXPLANATIONS BELOW

Concept note-1: -Answer 4: Insertion sort Explanation: It inserts every ar…

Concept note-2: -Insertion Sort does not perform swapping. It performs insertions by shifting elements in a sequential list to make room for the element that is being inserted. That is why it is an O(N^2) algorithm: for each element out of N, there can be O(N) shifts. Save this answer.

Concept note-3: -This algorithm is called selection sort because it repeatedly selects the next-smallest element and swaps it into place.

Concept note-4: -Which sorting algorithms may make multiple swaps per pass? Bubble sort may make multiple swaps in a single pass.

Concept note-5: -On average, the bubble sort performs poorly compared to the insertion sort. Due to the high number of swaps, it’s expected to generate twice as many write operations and twice as many cache misses. Therefore, we don’t prefer this algorithm for an ordinary sorting job.
