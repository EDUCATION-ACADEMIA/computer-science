what is a pros of linear search?

(A) good with long lists

(B) ** very easy to implement

(C) quick

EXPLANATIONS BELOW

Concept note-1: -With todayâ€™s powerful computers, small to medium arrays can be searched relatively quickly. The list does not need to sorted. Unlike a binary search, linear searching does not require an ordered list. Not affected by insertions and deletions.

Concept note-2: -Linear Search is basically a sequential search algorithm. In this algorithm, the key element is searched in the given input array in sequential order. If the key element is found in the input array, it returns the element.

Concept note-3: -Linear search is easy to implement and understand than other searching techniques.

Concept note-4: -Linear search can be used on both single and multidimensional array, whereas the binary search can be implemented only on the one-dimensional array. Linear search is less efficient when we consider the large data sets. Binary search is more efficient than the linear search in the case of large data sets.
