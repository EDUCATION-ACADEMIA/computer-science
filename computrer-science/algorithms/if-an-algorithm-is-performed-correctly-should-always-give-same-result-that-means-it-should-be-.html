If an Algorithm is performed correctly should always give same result, that means it should be ____

(A) Doable

(B) Understandable

(C) Finite

(D) ** Precise

EXPLANATIONS BELOW

Concept note-1: -An algorithm is correct only if it produces correct result for all input instances. – If the algorithm gives an incorrect answer for one or more input instances, it is an incorrect algorithm.

Concept note-2: -As it turns out, it’s difficult to prove that an algorithm is correct. Programmers often use empirical analysis to find faults in an algorithm, but only formal reasoning can prove total correctness.

Concept note-3: -An algorithm is a set of commands that must be followed for a computer to perform calculations or other problem-solving operations. According to its formal definition, an algorithm is a finite set of instructions carried out in a specific order to perform a particular task.

Concept note-4: -Answer: b) Algorithms are step-by-step rules for reaching a particular solution. Explanation: Algorithm is defined as step-vise procedure of solving and gaining a definite answer for a particular problem.
