The memory manager must have ____

(A) space efficiency

(B) program efficiency

(C) low overhead

(D) ** all the above

EXPLANATIONS BELOW

Concept note-1: -Memory management is the process of controlling and coordinating a computerâ€™s main memory. It ensures that blocks of memory space are properly managed and allocated so the operating system (OS), applications and other running processes have the memory they need to carry out their operations.

Concept note-2: -Memory management keeps track of the status of each memory location, whether it is allocated or free. It allocates the memory dynamically to the programs at their request and frees it for reuse when it is no longer needed.

Concept note-3: -The four major resource managers are: the Process Manager, the Memory Manager, the File Manager, and the Device Manager (which is actually a collection of many different device managers). The Process Manager manages processes in the system and is responsible for the sharing of the CPU.
