Select a Machine Independent phase of the compiler

(A) Syntax Analysis

(B) Intermediate Code generation

(C) Lexical Analysis

(D) ** All of the mentioned

EXPLANATIONS BELOW

Concept note-1: -Explanation: Machine independent phases are Lexical analysis, Syntax analysis, Semantic analysis, Intermediate code generation and sometime code optimization.

Concept note-2: -The correct answer is option 3. Concept: In compiler design, every phase takes inputs from its previous stage and feeds its output to the next phase of the compiler.

Concept note-3: -We basically have two phases of compilers, namely the Analysis phase and Synthesis phase.

Concept note-4: -Explore the Phases of Compiler Semantic Analysis. Intermediate Code Generation. Code Optimization.
