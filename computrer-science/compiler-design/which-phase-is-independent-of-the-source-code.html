Which phase is independent of the source code?

(A) Analysis phase

(B) ** Synthesis phase

(C) None of the mentioned

EXPLANATIONS BELOW

Concept note-1: -Every compiler phase receives input from the stage before it and feeds its output to the stage after it. The compilation is divided into two phases: Analysis (Machine Independent/Language Dependent) Synthesis (Machine Dependent/Language-Independent)

Concept note-2: -Explanation: Machine independent phases are Lexical analysis, Syntax analysis, Semantic analysis, Intermediate code generation and sometime code optimization.

Concept note-3: -The synthesis phase, also known as the code generation or code optimization phase, is the final step of a compiler. It takes the intermediate code generated by the front end of the compiler and converts it into machine code or assembly code, which can be executed by a computer.

Concept note-4: -There are 6 phases in the compiler, namely, lexical analysis, syntax analysis, semantics analysis, intermediate code generation, code optimization, and code generation. Symbol table is a data structure that the compiler generates and maintains to keep track of the semantics of variables.

Concept note-5: -Explore the Phases of Compiler Syntactic Analysis or Parsing. Semantic Analysis. Intermediate Code Generation. Code Optimization.
