Which of the following system software resides in the main memory always

(A) Text Editor

(B) Assembler

(C) Linker

(D) ** Loader

EXPLANATIONS BELOW

Concept note-1: -In a resident-OS computer, which of the following system software must reside in the main memory under all situations? Explanation: Loader is the part of an operating system that is responsible for loading programs and libraries. it places the programs into memory and also prepares them for execution.

Concept note-2: -The small program that starts this sequence is known as a bootstrap loader, bootstrap or boot loader.

Concept note-3: -A part of the system software which under all circumstances must reside in the main memory is: text editor. assembler. linker.

Concept note-4: -The operating system is the best-known example of system software. The OS manages all the other programs in a computer.
