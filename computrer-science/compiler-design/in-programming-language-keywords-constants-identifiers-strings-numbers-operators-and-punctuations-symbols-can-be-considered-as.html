In programming language, keywords, constants, identifiers, strings, numbers, operators and punctuations symbols can be considered as?

(A) Lexemes

(B) ** Tokens

(C) Symbols

(D) Pattern

EXPLANATIONS BELOW

Concept note-1: -In programming language, keywords, constants, identifiers, strings, numbers, operators and punctuations symbols can be considered as tokens. int value = 100; contains the tokens: int (keyword), value (identifier), = (operator), 100 (constant) and; (symbol).

Concept note-2: -A programming token is the basic component of source code. Characters are categorized as one of five classes of tokens that describe their functions (constants, identifiers, operators, reserved words, and separators) in accordance with the rules of the programming language.

Concept note-3: -In programming, a token is a single element of a programming language. There are five categories of tokens: 1) constants, 2) identifiers, 3) operators, 4) separators, and 5) reserved words. For example, the reserved words “new” and “function” are tokens of the JavaScript language.

Concept note-4: -C Tokens are of 6 types, and they are classified as: Identifiers, Keywords, Constants, Operators, Special Characters and Strings.
