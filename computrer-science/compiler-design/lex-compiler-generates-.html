LEX compiler generates ____

(A) yy.lex.c

(B) ** lex.yy.c

(C) executable code

(D) object code

EXPLANATIONS BELOW

Concept note-1: -The lex command stores the yylex function in a file named lex. yy. c. You can use the yylex function alone to recognize simple one-word input, or you can use it with other C language programs to perform more difficult input analysis functions.

Concept note-2: -It is a tool or software which automatically generates a lexical analyzer (finite Automata). It takes as its input a LEX source program and produces lexical Analyzer as its output.

Concept note-3: -yylex() returns a value indicating the type of token that has been obtained. If the token has an actual value, this value (or some representation of the value, for example, a pointer to a string containing the value) is returned in an external variable named yylval.
