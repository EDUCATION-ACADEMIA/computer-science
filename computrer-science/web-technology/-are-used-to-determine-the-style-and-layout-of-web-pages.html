____ are used to determine the style and layout of web pages.

(A) CSS

(B) DOM

(C) IRM

(D) ** All of the Above

EXPLANATIONS BELOW

Concept note-1: -CSS is the language for describing the presentation of Web pages, including colors, layout, and fonts.

Concept note-2: -Cascading Style Sheets (CSS) is used to format the layout of a webpage.

Concept note-3: -Using CSS, you can layout all your elements on your webpage visually. For example, you can position an element at the very top of your page, or 50px below the element before it. To control just how an element will appear in the layout, you need to use the CSS position property.

Concept note-4: -CSS syntax CSS is a rule-based language-you define the rules by specifying groups of styles that should be applied to particular elements or groups of elements on your web page.
