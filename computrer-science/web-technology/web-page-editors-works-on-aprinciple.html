Web page editors works on a ____ principle.

(A) WWW

(B) HYML

(C) ** WYSIWYG

(D) WYGWYSI

EXPLANATIONS BELOW

Concept note-1: -In computing, WYSIWYG (/ˈwɪziwɪɡ/ WIZ-ee-wig), an acronym for What You See Is What You Get, is a system in which editing software allows content to be edited in a form that resembles its appearance when printed or displayed as a finished product, such as a printed document, web page, or slide presentation.

Concept note-2: -Web Page Editor is an application that can help you build a website in an easy and convenient manner. The WYSIWYG principle, in which what you see is what you get, is the basis for most website software. This eliminates the need for coding and allows users to craft their websites with a visual approach.

Concept note-3: -Web pages can be created and modified by using professional HTML editors. However, for learning HTML we recommend a simple text editor like Notepad (PC) or TextEdit (Mac).

Concept note-4: -A browser is a software program used to view web pages.

Concept note-5: -A web page is a simple document displayable by a browser. Such documents are written in the HTML language (which we look into in more detail in other articles). A web page can embed a variety of different types of resources such as: style information-controlling a page’s look-and-feel.
