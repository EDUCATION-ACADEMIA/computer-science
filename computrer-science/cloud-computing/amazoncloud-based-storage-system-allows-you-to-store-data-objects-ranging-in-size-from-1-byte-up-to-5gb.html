Amazon ____ cloud-based storage system allows you to store data objects ranging in size from 1 byte up to 5GB.

(A) EBS

(B) S2

(C) ** S3

(D) Glacier

EXPLANATIONS BELOW

Concept note-1: -Which Amazon cloud-based storage system allows you to store data objects ranging in size from 1 byte up to 5GB? Answer: Amazon S3 cloud-based storage system allows you to store data objects ranging in size from 1 byte up to 5GB. It is because, in S3 containers, storage containers are often referred to as buckets.

Concept note-2: -Individual Amazon S3 objects can range in size from a minimum of 0 bytes to a maximum of 5 TB. The largest object that can be uploaded in a single PUT is 5 GB. For objects larger than 100 MB, customers should consider using the multipart upload capability.

Concept note-3: -Amazon Simple Storage Service (Amazon S3) is an object storage service that offers industry-leading scalability, data availability, security, and performance. You can use Amazon S3 to store and retrieve any amount of data at any time, from anywhere.

Concept note-4: -What are the types of cloud storage? There are three main cloud storage types: object storage, file storage, and block storage. Each offers its own advantages and has its own use cases.
