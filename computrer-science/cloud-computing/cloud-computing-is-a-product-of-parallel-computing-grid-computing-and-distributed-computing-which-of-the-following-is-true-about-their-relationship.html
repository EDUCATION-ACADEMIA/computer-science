CLOUD COMPUTING

[SOURCES]
Cloud computing is a product of parallel computing, grid computing, and distributed computing. Which of the following is true about their relationship?

(A) Distributed computing completely replaces grid computing.

(B) ** Cloud computing is a type of distributed computing.

(C) Parallel computing, grid computing, and distributed computing all belong to cloud computing

(D) Grid computing completely replaces parallel computing.

EXPLANATIONS BELOW

Concept note-1: -Cloud Computing follows a client-server computing architecture. Grid Computing follows a distributed computing architecture. In Cloud Computing, resources are centrally managed. In Grid Computing, resources are managed on a collaboration pattern.

Concept note-2: -Cloud computing is a distributed computing model-TRUE. Solutions which are computed and are in distributed form refers to cloud computing. Various services based on internet are provided by cloud computing network that links or interconnects data which are accessed from other geographical locations.

Concept note-3: -Correct Answer : Option (B) : Cloud Computing makes our business applications mobile and collaborative. Explanation : Cloud computing offers platform independency, as the software is not required to be installed locally on the PC.

Concept note-4: -Cloud computing refers to providing on demand IT resources/services like server, storage, database, networking, analytics, software etc. over internet. Distributed computing refers to solve a problem over distributed autonomous computers and they communicate between them over a network.
