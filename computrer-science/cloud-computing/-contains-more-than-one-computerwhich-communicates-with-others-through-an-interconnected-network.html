____ contains more than one computerwhich communicates with others through an interconnected network.

(A) Client server computing

(B) Peer to Peer Computing

(C) ** Distributed Computing

(D) None of these

EXPLANATIONS BELOW

Concept note-1: -A computer network is a system in which multiple computers are connected to each other to share information and resources.

Concept note-2: -The TCP/IP protocol allows each computer to “see” the other computers on the network, and share files and printers. When computers connect on the same network, it is called a local area network, or LAN.

Concept note-3: -The Internet is the worldwide, publicly accessible network of interconnected computer networks that transmit data by packet switching using the standard Internet Protocol (IP).

Concept note-4: -A computer network is any group of interconnected computing devices capable of sending or receiving data.

Concept note-5: -Hub. Hubs connect multiple computer networking devices together.
