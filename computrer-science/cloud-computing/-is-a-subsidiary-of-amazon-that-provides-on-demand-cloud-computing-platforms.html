____ is a subsidiary of Amazon that provides on-demand cloud computing platforms

(A) Amazon Prime

(B) Amazon Video

(C) ** Amazon Web Services

(D) Amazon Art

EXPLANATIONS BELOW

Concept note-1: -(AWS) is a subsidiary of Amazon that provides on-demand cloud computing platforms and APIs to individuals, companies, and governments, on a metered, pay-as-you-go basis.

Concept note-2: -In 2006, Amazon Web Services (AWS) began offering IT infrastructure services to businesses in the form of web services–now commonly known as cloud computing.

Concept note-3: -Amazon Web Services is a subsidiary of Amazon, which provides an on-demand Cloud Computing platform to individuals, companies, and governments on a paid subscription basis.

Concept note-4: -Amazon EC2. Amazon Elastic Compute Cloud (Amazon EC2) is a web service that provides secure, resizable compute capacity in the cloud. It is designed to make web-scale computing easier for developers.

Concept note-5: -A cloud services platform such as Amazon Web Services owns and maintains the network-connected hardware required for these application services, while you provision and use what you need via a web application.
