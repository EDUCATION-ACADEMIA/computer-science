(B) Public cloud

(C) ** Hybrid cloud

(D) community cloud

EXPLANATIONS BELOW

Concept note-1: -AWS Outposts allows you to run AWS infrastructure and services on premises for a consistent hybrid experience.

Concept note-2: -In this type of cloud, an organization rents cloud services from cloud providers on-demand basis. Explanation: This happens in public cloud.

Concept note-3: -Hybrid cloud integrates public cloud services, private cloud services and on-premises infrastructure and provides orchestration, management and application portability across all three.

Concept note-4: -Cloud elasticity is a popular feature associated with scale-out solutions (horizontal scaling), and it allows for resources to be dynamically added or removed when needed.
