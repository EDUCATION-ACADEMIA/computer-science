CLOUD COMPUTING

[SOURCES]
____ is the on-demand availability of computer system resources, especially data storage and computing power, without direct active management by the user

(A) ** Cloud computing

(B) Grid computing

(C) Utility Computing

(D) Distributed Computing

EXPLANATIONS BELOW

Concept note-1: -Cloud computing is the on-demand availability of computer system resources, especially data storage (cloud storage) and computing power, without direct active management by the user. Large clouds often have functions distributed over multiple locations, each of which is a data center.

Concept note-2: -On-demand computing (ODC) is a delivery model in which computing resources are made available to the user as needed. The resources may be maintained within the userâ€™s enterprise or made available by a cloud service provider.

Concept note-3: -What is cloud computing? Cloud computing is on-demand access, via the internet, to computing resources-applications, servers (physical servers and virtual servers), data storage, development tools, networking capabilities, and more-hosted at a remote data center managed by a cloud services provider (or CSP).

Concept note-4: -Cloud computing is the on-demand delivery of compute power, database, storage, applications, and other IT resources through a cloud services platform via the internet with pay-as-you-go pricing.

Concept note-5: -Cloud computing is the delivery of different services through the Internet. These resources include tools and applications like data storage, servers, databases, networking, and software.
