CLOUD COMPUTING

[SOURCES]
The most utilized. The software is hosted on the providers’ servers and accessed by the customer through the web browser. this concept belongs to:

(A) The PaaS-Platform as a Service model

(B) ** The SaaS-Software as a Service cloud computing model

(C) The IaaS-Infrastructure as a Service model

EXPLANATIONS BELOW

Concept note-1: -SaaS applications are sometimes called Web-based software, on-demand software, or hosted software. Whatever the name, SaaS applications run on a SaaS provider’s servers. The provider manages access to the application, including security, availability, and performance.

Concept note-2: -SaaS works through the cloud delivery model. A software provider will either host the application and related data using its own servers, databases, networking and computing resources, or it may be an ISV that contracts a cloud provider to host the application in the provider’s data center.

Concept note-3: -SaaS, or software as a service, is on-demand access to ready-to-use, cloud-hosted application software.

Concept note-4: -SaaS is a method of software delivery that allows data to be accessed from any device with an internet connection and a web browser. In this web-based model, software vendors host and maintain the servers, databases, and the code that makes up an application.
