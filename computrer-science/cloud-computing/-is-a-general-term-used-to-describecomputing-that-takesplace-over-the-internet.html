[SOURCES]
____ is a general term used to describecomputing that takesplace over the Internet

(A) ** Cloud Computing

(B) Network Computing

(C) Storage Computing

(D) None of the choices

EXPLANATIONS BELOW

Concept note-1: -Cloud Computing is a general term used to describe a new class of network based computing that takes place over the Internet, basically a step on from Utility Computing. In other words, this is a collection/group of integrated and networked hardware, software and Internet infrastructure (called a platform).

Concept note-2: -Cloud computing is a general term for anything that involves delivering hosted services over the internet. These services are divided into three main categories or types of cloud computing: infrastructure as a service (IaaS), platform as a service (PaaS) and software as a service (SaaS).

Concept note-3: -SaaS (Software-as-a-Service) SaaS-also known as cloud-based software or cloud applications-is application software thatâ€™s hosted in the cloud, and that users access via a web browser, a dedicated desktop client, or an API that integrates with a desktop or mobile operating system.

Concept note-4: -SaaS (sometimes called cloud application services) is cloud-hosted, ready-to-use application software. Users pay a monthly or annual fee to use a complete application from within a web browser, desktop client or mobile app.
