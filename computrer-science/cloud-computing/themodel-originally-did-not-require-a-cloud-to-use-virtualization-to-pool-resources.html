The ____ model originally did not require a cloud to use virtualization to pool resources.

(A) NEFT

(B) ** NIST

(C) NIT

(D) All of the mentioned

EXPLANATIONS BELOW

Concept note-1: -The model originally did not require a cloud to use virtualization to pool resources. Explanation: NIST stands for National Institute of Standards and Technology.

Concept note-2: -The National Institute of Standards Technology (NIST) lists five essential characteristics of cloud computing: on-demand self-service, broad network access, resource pooling, rapid elasticity, and measured service.

Concept note-3: -Expert-Verified Answer The correct option is C) Compound Cloud. The Compound Cloud is not a deployment model in cloud computing.

Concept note-4: -On-demand self-service. Broad network access. Resource pooling. Rapid elasticity.
