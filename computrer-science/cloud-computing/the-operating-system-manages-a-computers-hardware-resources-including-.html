The operating system manages a computer’s hardware resources, including ____

(A) Input devices such as a keyboard and mouse

(B) Output devices such as display monitors, printers and scanners

(C) Network devices such as modems, routers and network connections

(D) ** All of the above

EXPLANATIONS BELOW

Concept note-1: -The operating system (OS) manages all of the software and hardware on the computer. It performs basic tasks such as file, memory and process management, handling input and output, and controlling peripheral devices such as disk drives and printers.

Concept note-2: -The operating system must manage the use of hardware resources so that software can complete its tasks. Each program running on a computer is known as a process. The OS manages how software processes share the system’s resources. For example, most software needs to save data to a storage device, such as a hard disk.

Concept note-3: -The operating system manages peripheral input and output devices such as keyboards, monitors, printers, scanners and cameras. It controls the sending and receiving of data . To achieve this, the operating system uses a program called a device driver . Device drivers contain instructions on how to control a device.

Concept note-4: -An operating system (OS) is system software that manages computer hardware and software resources, and provides common services for computer programs.
