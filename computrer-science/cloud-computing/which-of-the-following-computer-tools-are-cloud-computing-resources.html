Which of the following Computer Tools are “Cloud Computing” resources:

(A) ** Google Drive

(B) Ubuntu One

(C) deviant

(D) microsoft office

EXPLANATIONS BELOW

Concept note-1: -Google Drive is a free cloud-based storage service that enables users to store and access files online. The service syncs stored documents, photos and more across all of the user’s devices, including mobile devices, tablets and PCs.

Concept note-2: -Google Drive: Store, manage, and share your personal files. Cloud Storage for Firebase: Manage data for your mobile applications.

Concept note-3: -Google Cloud provides resource containers such as organizations, folders, and projects that allow you to group and hierarchically organize other Google Cloud resources.
