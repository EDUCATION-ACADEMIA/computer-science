CLOUD COMPUTING

[SOURCES]
A regular computer, using a web browser to connect to the cloud.

(A) ** Thick

(B) thin

(C) mobile

(D) data center

EXPLANATIONS BELOW

Concept note-1: -Essentially, any device that can function completely independently of a remote server is a thick client. Everyday examples of thick clients include desktop PCs or laptops running Windows or MacOS.

Concept note-2: -Also referred to as desktop, fat, or heavy client, thick clients are systems that connect to servers even without a network. Put simply, a thick client does not rely on server applications since it can process, store and manage data, as well as perform different tasks independently.

Concept note-3: -Thick clients work with information on the basis of their own hardware and software, and thin clients use software of the main server only for data processing, providing a system with the required graphical interface to give a user possibility to work.

Concept note-4: -The classic example of a thin client is a web browser. They arenâ€™t very interesting on their own but offer a wide range of functionality by connecting to a variety of web servers. Without a network connection, they offer limited functionality.
