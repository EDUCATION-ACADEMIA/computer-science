Web services can be discovered using ____

(A) UDDII

(B) UDII

(C) ** UDDI

(D) UDDDI

EXPLANATIONS BELOW

Concept note-1: -UDDI was originally proposed as a core Web service standard. It is designed to be interrogated by SOAP messages and to provide access to Web Services Description Language (WSDL) documents describing the protocol bindings and message formats required to interact with the web services listed in its directory.

Concept note-2: -Web Services are discovered in the Oracle UDDI Registry by browsing the registry using tools or using the Inquiry API.

Concept note-3: -UDDI manages the discovery of Web services by relying on a distributed registry of businesses and their service descriptions implemented in a common XML format. Before you can publish your business entity and Web service to a public registry, you must first register your business entity with a UDDI registry.

Concept note-4: -Q 3-Which of the following is correct about UDDI? A-UDDI is an XML-based standard for describing, publishing, and finding web services.

Concept note-5: -The basic platform of a web service is XML over HTTP. For platform support services such as discovery, transactions, security, and authentication, other communication protocols and languages apply.
