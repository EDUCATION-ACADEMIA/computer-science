Which of the following is an example of an IaaS Cloud service?

(A) ** EC2

(B) S3

(C) DynamoDB

(D) IAM

EXPLANATIONS BELOW

Concept note-1: -A good example of IaaS is AWS EC2. EC2 delivers scalable infrastructure for companies that want to host cloud-based applications. EC2 users do not own the physical servers-AWS provides virtual servers.

Concept note-2: -Examples of IaaS include Rackspace, Amazon Web Services (AWS) Elastic Compute Cloud (EC2), Microsoft Azure, Google Compute Engine (GCE) and Joyent.

Concept note-3: -Which of the following is an example of an IaaS Cloud service? Explanation: IaaS stands for Infrastructure-as-a-Service. In IaaS, the users receive computing resources like software or hardware tools virtually over the internet. DigitalOcean, Linode, Rackspace are the example of an IaaS service.

Concept note-4: -Amazon Elastic Compute Cloud (Amazon EC2) provides scalable computing capacity in the Amazon Web Services (AWS) Cloud. Using Amazon EC2 eliminates your need to invest in hardware up front, so you can develop and deploy applications faster.

Concept note-5: -AWS (Amazon Web Services) is a comprehensive, evolving cloud computing platform provided by Amazon that includes a mixture of infrastructure-as-a-service (IaaS), platform-as-a-service (PaaS) and packaged-software-as-a-service (SaaS) offerings.
