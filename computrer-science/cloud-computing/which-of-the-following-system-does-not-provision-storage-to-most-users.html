Which of the following system does not provision storage to most users?

(A) PaaS

(B) ** IaaS

(C) CaaS

(D) SaaS

EXPLANATIONS BELOW

Concept note-1: -Which of the following system does not provision storage to most users? Explanation: Cloud storage is storage accessed by a Web service API.

Concept note-2: -Storage. The three primary types of cloud storage are block storage, file storage, and object storage.

Concept note-3: -Answer. (D)Intel. Intel is not a cloud computing service provider.

Concept note-4: -Infrastructure as a service (IaaS) is a form of cloud computing that provides virtualized computing resources over the internet. IaaS is one of the three main categories of cloud computing services, alongside software as a service (SaaS) and platform as a service (PaaS).
