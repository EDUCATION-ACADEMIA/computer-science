Which among the following AWS service create block level storage devices

(A) Amazone S3

(B) ** Amazone EBS

(C) Glacier

(D) DynamoDB

EXPLANATIONS BELOW

Concept note-1: -Amazon EBS provides durable, block-level storage volumes that you can attach to a running instance. You can use Amazon EBS as a primary storage device for data that requires frequent and granular updates.

Concept note-2: -EBS services are a raw block-level storage solution made specifically for use with Amazon EC2 instances. Amazon EBS storage usage is no different than the implementation of any other raw block device when mounted to Amazon EC2 instances.

Concept note-3: -EFS provides a file system interface, file system access semantics (such as strong consistency and file locking), and concurrently accessible storage for up to thousands of EC2 instances. Amazon EBS is a block-level storage service for use with EC2.

Concept note-4: -The Amazon Elastic Block Store (EBS) provides raw block devices that can be attached to Amazon EC2 instances.

Concept note-5: -Amazon S3. Amazon EBS is the AWS block storage offering and Amazon S3 is the AWS object storage offering.
