Testing & monitoring tools are ____ SaaS category

(A) Packaged software

(B) Collaborative software

(C) ** Enabling and management tools

(D) All the above

EXPLANATIONS BELOW

Concept note-1: -What are SaaS Management Platforms? SaaS management platforms (SMPs) provide central admin console capabilities to discover, manage, automate, optimize, govern and enable SaaS used by employees. SMP also enhances protection of identities and data while using SaaS.

Concept note-2: -Other types of SaaS applications include billing and invoice software, collaboration software, web hosting software and Human Resources software. With all the SaaS options available, you can pick and choose which cloud products make the most sense for your business.

Concept note-3: -SaaS testing helps identify areas where the application needs improvement. The three main types of tests are functional, integration, and acceptance. Functional tests check whether the application performs the required functions correctly.

Concept note-4: -The essential SaaS components are: CRM system. Marketing automation.
