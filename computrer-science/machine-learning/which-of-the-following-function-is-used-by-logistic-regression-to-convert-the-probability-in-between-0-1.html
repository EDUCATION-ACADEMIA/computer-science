Which of the following function is used by Logistic Regression to convert the probability in between [0, 1]

(A) ** Sigmoid

(B) polynomial

(C) square

(D) rbf

EXPLANATIONS BELOW

Concept note-1: -Logistic regression uses a logistic function called a sigmoid function to map predictions and their probabilities. The sigmoid function refers to an S-shaped curve that converts any real value to a range between 0 and 1.

Concept note-2: -Logit function estimates probabilities between 0 and 1, and hence logistic regression is a non-linear transformation that looks like S-function shown below.

Concept note-3: -What is the Sigmoid Function? In order to map predicted values to probabilities, we use the Sigmoid function. The function maps any real value into another value between 0 and 1. In machine learning, we use sigmoid to map predictions to probabilities.

Concept note-4: -Conversion rule To convert a logit ( glm output) to probability, follow these 3 steps: Take glm output coefficient (logit) compute e-function on the logit using exp() “de-logarithimize” (you’ll get odds then) convert odds to probability using this formula prob = odds / (1 + odds) .

Concept note-5: -Such a logistic model is called a log-odds model. Hence, in statistics, Logistic Regression is sometimes called the logistic model or logit model. It is used for predicting the probability of the occurrence of a specific event by fitting data to a logit Logistic Function curve.
