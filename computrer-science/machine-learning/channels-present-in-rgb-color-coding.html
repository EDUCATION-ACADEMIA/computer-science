Channels present in RGB color coding

(A) ** Red

(B) Grey

(C) ** Blue

(D) ** Green

EXPLANATIONS BELOW

Concept note-1: -RGB Color Values Another example, rgb(0, 255, 0) is displayed as green, because green is set to its highest value (255), and the other two (red and blue) are set to 0.

Concept note-2: -An RGB image has three channels: red, green, and blue. An HSL image has three channels: hue, saturation, and lightness. Although you cannot create or edit an image using the CMYK model, you can still split the image into four channels: cyan, magenta, yellow, and black.

Concept note-3: -A CMYK image has four channels: cyan, magenta, yellow, and black.

Concept note-4: -Green channel is alone extracted from this concatenated RGB and CLAHE (Contrast Limited Adaptive Histogram Equalization) is done on that in order to improve the contrast on tile level. Adapthisteq calculates the contrast transform function for each tile individually.
