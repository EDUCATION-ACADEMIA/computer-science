COMPUTRER SCIENCE

MACHINE LEARNING

[SOURCES]
Machine learning is different from “traditional” computer programs because it learns from seeing examples e.g. examples of a cat, or examples of a scan

(A) ** true

(B) false

EXPLANATIONS BELOW

Concept note-1: -Traditional programming is a manual process-meaning a person (programmer) creates the program. But without anyone programming the logic, one has to manually formulate or code rules. In machine learning, on the other hand, the algorithm automatically formulates the rules from the data.

Concept note-2: -In traditional programs, a developer designs logic or algorithms to solve a problem. The program applies this logic to input and computes the output. But in Machine Learning, a model is built from the data, and that model is the logic.

Concept note-3: -Traditional Programming: We feed in DATA (Input) + PROGRAM (logic), run it on the machine, and get the output. Machine Learning: We feed in DATA(Input) + Output, run it on the machine during training and the machine creates its own program(logic), which can be evaluated while testing.

Concept note-4: -Image recognition. Image recognition is a well-known and widespread example of machine learning in the real world. It can identify an object as a digital image, based on the intensity of the pixels in black and white images or colour images.
