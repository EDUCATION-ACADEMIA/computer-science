(A) TP/TN-FN

(B) TP/TP-FN

(D) TP/FN+TP-TN

EXPLANATIONS BELOW

Concept note-1: -False negative (FN), type II error, miss, underestimation. True positive rate (TPR), recall, sensitivity (SEN), probability of detection, hit rate, power.

Concept note-2: -Abbreviations: PPV, Positive predicted value; NPV, Negative predicted value; TP, True Positive; FP, False Positive; FN, False Negative; TN, True Negative.
