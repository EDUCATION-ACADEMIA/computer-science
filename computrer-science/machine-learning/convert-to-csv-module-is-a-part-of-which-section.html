‘Convert to CSV’ module is a part of which section?

(A) Data Input and Output

(B) ** Data Transformation

(C) Model Training

(D) None of the above

EXPLANATIONS BELOW

Concept note-1: -In Python, the csv. reader() module is used to read the csv file. It takes each row of the file and makes a list of all the columns.

Concept note-2: -Python provides a CSV module to handle CSV files. To read/write data, you need to loop through rows of the CSV. You need to use the split method to get data from specified columns.

Concept note-3: -The csv module was incorporated in Python’s standard library as a result of PEP 305. It presents classes and methods to perform read/write operations on CSV file as per recommendations of PEP 305. CSV is a preferred export data format by Microsoft’s Excel spreadsheet software.

Concept note-4: -Pandas DataFrame to csv() function converts DataFrame into CSV data.
