HTTP is a markup language.

(A) True

(B) ** False

EXPLANATIONS BELOW

Concept note-1: -HTML is a type of markup language. It encapsulates, or “marks up” data within HTML tags, which define the data and describe its purpose on the webpage.

Concept note-2: -HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. Other technologies besides HTML are generally used to describe a web page’s appearance/presentation (CSS) or functionality/behavior (JavaScript).

Concept note-3: -HyperText Markup Language (HTML) is the set of markup symbols or codes inserted into a file intended for display on the Internet. The markup tells web browsers how to display a web page’s words and images.

Concept note-4: -Explanation: As HTTP is stateless application-level protocol and it requires a reliable network transport connection to exchange data between client and server.

Concept note-5: -YAML (YAML Ain’t Markup Language) is a data serialization language used as the input format for diverse software applications.
