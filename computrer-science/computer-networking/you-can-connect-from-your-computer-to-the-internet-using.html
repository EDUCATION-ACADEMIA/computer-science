(A) software

(B) a router

(C) ** a web browser

EXPLANATIONS BELOW

Concept note-1: -Internet Connection Types: WiFi, Broadband, DSL, Cable.

Concept note-2: -The browser sends an HTTP request message to the server, asking it to send a copy of the website to the client (you go to the shop and order your goods). This message, and all other data sent between the client and the server, is sent across your internet connection using TCP/IP.
