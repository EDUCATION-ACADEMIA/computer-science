COMPUTER NETWORKING

[SOURCES]
In this network, each node is connected to a central device or hub.

(A) bus network

(B) ring network

(C) ** star network

(D) wireless network

EXPLANATIONS BELOW

Concept note-1: -Star topology is a network topology where each individual piece of a network is attached to a central node (often called a hub or switch). The attachment of these network pieces to the central component is visually represented in a form similar to a star. Star topology is also known as a star network.

Concept note-2: -A star network is a local area network (LAN) topology in which all nodes–personal computers (PCs), workstations or other devices–are directly connected to a common central computer that is often referred to as a hub.

Concept note-3: -Star topology is a type of network topology in which every device in the network is individually connected to a central node, known as the switch or hub.

Concept note-4: -In a star network, every host is connected to a central hub. In its simplest form, one central hub acts as a conduit to transmit messages.
