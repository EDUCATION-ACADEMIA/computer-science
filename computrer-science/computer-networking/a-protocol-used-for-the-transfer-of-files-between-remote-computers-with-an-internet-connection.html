A protocol used for the transfer of files between remote computers with an internet connection

(A) FPT

(B) PTF

(C) TPF

(D) ** FTP

EXPLANATIONS BELOW

Concept note-1: -FTP (File Transfer Protocol) is a network protocol for transmitting files between computers over Transmission Control Protocol/Internet Protocol (TCP/IP) connections. Within the TCP/IP suite, FTP is considered an application layer protocol.

Concept note-2: -File Transfer Protocol (FTP) FTP is an application layer protocol that is used for transferring both text and binary files over the Internet.

Concept note-3: -FTP works on a client-server model. The FTP client is a program that runs on the userâ€™s computer to enable the user to talk to and get files from remote computers. It is a set of commands that establishes the connection between two hosts, helps to transfer the files, and then closes the connection.

Concept note-4: -FTP is a file transfer protocol, which means it is used to transfer files between a client and a server, whereas TCP is a communication protocol used to exchange data between networks. They come together when FTP sends data over a TCP/IP connection.

Concept note-5: -FTP refers to the set of rules that basically allows the process of uploading and downloading files from a computer to the internet. HTTP refers to a set of rules that determines the process of transfer of various web pages over various computers present on the internet.
