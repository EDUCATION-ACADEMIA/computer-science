An IP address is ____

(A) A selection of letters that help computers to communicate with each other.

(B) ** A selection of numbers that help computers to communicate with each other.

(C) A way of finding your local computer shop.

(D) A type of code used to program a computer.

EXPLANATIONS BELOW

Concept note-1: -An IP address, or Internet Protocol address, is a series of numbers that identifies any device on a network. Computers use IP addresses to communicate with each other both over the internet as well as on other networks.

Concept note-2: -On the taskbar, select Wi-Fi network > the Wi-Fi network you’re connected to > Properties. Under Properties, look for your IP address listed next to IPv4 address.

Concept note-3: -An IP address is a long string of numbers assigned to every device connected to a network that uses Internet Protocol as the medium for communication; it’s the digital world’s equivalent of the mailing address associated with your home or workplace.

Concept note-4: -IP address stands for internet protocol address; it is an identifying number that is associated with a specific computer or computer network. When connected to the internet, the IP address allows the computers to send and receive information.

Concept note-5: -Computers connected to the Internet must speak the “Internet language” called the “Internet Protocol” or simply “IP.” Each computer is assigned a unique address somewhat similar to a street address or telephone number. Under the current system there are four numbers that range from 0 to 255 (Example: 206.156. 18.122).
