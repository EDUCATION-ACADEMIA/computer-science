COMPUTER NETWORKING

[SOURCES]
A WNIC card provides a computer with wired network access.

(A) True

(B) ** False

EXPLANATIONS BELOW

Concept note-1: -A wireless network interface controller (WNIC) is a network interface controller which connects to a wireless network, such as Wi-Fi or Bluetooth, rather than a wired network, such as a Token Ring or Ethernet.

Concept note-2: -A network interface card (NIC) is a hardware component, typically a circuit board or chip, which is installed on a computer so it can connect to a network.

Concept note-3: -NICs allow you to connect a computer to a network. There are two types of NIC: wired and wireless. Wired NICs allow you to connect to a network over Wi-Fi. Wireless NICs allow you to connect a computer to a network over ethernet.

Concept note-4: -A modem is not a network interface. A modem has 2 network interfaces. The interface is where the cables plug into the modem. So you have a cable coming from the cable company connecting to it.
