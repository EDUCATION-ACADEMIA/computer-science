What is TCP / IP?

(A) Transmission Caption Protocol / Internet Protocol

(B) Terminal Connection Protocol / Internet Protocol

(C) Transmission Control Protocol / Inside Protocol

(D) ** Transmission Control Protocol / Internet Protocol

EXPLANATIONS BELOW

Concept note-1: -Transmission Control Protocol (TCP) is a standard that defines how to establish and maintain a network conversation by which applications can exchange data. TCP works with the Internet Protocol (IP), which defines how computers send packets of data to each other.

Concept note-2: -Leam McArdle. TCP/IP stands for Transmission Control Protocol / Internet Protocol. It defines how electronic devices (like computers) should be connected over the Internet, and how data should be transmitted between them.
