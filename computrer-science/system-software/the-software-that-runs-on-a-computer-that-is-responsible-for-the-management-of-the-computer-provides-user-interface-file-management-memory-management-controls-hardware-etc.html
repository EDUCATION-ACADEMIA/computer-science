SYSTEM SOFTWARE

[SOURCES]
The software that runs on a computer that is responsible for the management of the computer. Provides user interface, file management, memory management, controls hardware, etc

(A) Application software

(B) ** Operating system

(C) system software

(D) control unit

EXPLANATIONS BELOW

Concept note-1: -An Operating System (OS) is an interface between a computer user and computer hardware. An operating system is a software which performs all the basic tasks like file management, memory management, process management, handling input and output, and controlling peripheral devices such as disk drives and printers.

Concept note-2: -Operating system (OS) – a software that controls and coordinates the computer hardware devices and runs other software and applications on a computer. It is the main part of system software and a computer will not function without it.

Concept note-3: -What is an operating system? An operating system is the most important software that runs on a computer. It manages the computer’s memory and processes, as well as all of its software and hardware.

Concept note-4: -An operating system (OS) is the software component of a computer system that is responsible for the management and coordination of activities and the sharing of the resources of the computer. The OS acts as a host for application programs that are run on the machine.
