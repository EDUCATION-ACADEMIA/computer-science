Which of the following is NOT a function of Operating System?

(A) Booting the computer

(B) Managing resources and jobs

(C) Interfacing with users (typically via a GUI)

(D) ** Disk management

EXPLANATIONS BELOW

Concept note-1: -The correct answer is Execution of arithmetic operations. Execution of arithmetic operations is not a function of the operating system.

Concept note-2: -job control is not included in basic functions of operating system. we can use job scheduling, memory management and data management. operating systems help people interface with computers and provide network. job control is not a function opearting systemcapabilities.

Concept note-3: -Which is not a function of operating system? (a) Memory management (b) Virus protection (c) Disk management (d) File management. Virus protection is not a function of the operating system.

Concept note-4: -Disk Management is an important functionality provided by the Operating System which can be used to create, delete, format disk partitions, and much more. Disk Management enables users to view, create, delete and shrink the partitions associated with the disk drives.
