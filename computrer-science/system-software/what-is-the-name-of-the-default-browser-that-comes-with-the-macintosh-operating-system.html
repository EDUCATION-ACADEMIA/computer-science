What is the name of the default browser that comes with the Macintosh Operating System?

(A) Opera

(B) ** Safari

(C) Spotlight

(D) Firefox

EXPLANATIONS BELOW

Concept note-1: -Google Chrome is a cross-platform web browser developed by Google. It was first released in 2008 for Microsoft Windows and was later ported to Linux, macOS, iOS, and Android where it is the default browser built into the OS.

Concept note-2: -Safari is the default browser when you first set up your Mac. If you accidentally change your default browser or just want to change it back to Safari, you can easily adjust your default. On your Mac, choose Apple menu > System Settings, then click Desktop & Dock in the sidebar. (You may need to scroll down.)

Concept note-3: -With macOS, the default browser that comes bundled with the operating system is none other than Apple’s Safari. Safari is an excellent browser, especially if you use multiple Mac computers and/or iOS devices, because it will be able to sync your data such as browsing history across them.

Concept note-4: -In macOS, Apple’s Safari is the default browser for accessing the web, but you don’t have to keep it that way. This article shows you how to change your default web browser to an alternative option.
