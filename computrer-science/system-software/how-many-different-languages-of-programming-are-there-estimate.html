How many different languages of programming are there? (estimate)

(A) All programming is in one

(B) 100

(C) 500

(D) ** 700

EXPLANATIONS BELOW

Concept note-1: -According to the Online Historical Encyclopaedia of Programming Languages, people have created about 8, 945 coding languages. Today, various sources report anywhere from 250-2, 500 coding languages, although far fewer rank as top contenders in the commonly used group.

Concept note-2: -There are three types of programming languages: machine language, assembly language, and high-level language. Machine language is easier for the computer to understand but harder for the programmer to understand.

Concept note-3: -These programming languages include: C, C++, JAVA, JAVASCRIPT, PHP, RUBY, Objective C, and Python. C is used as the foundational programming language that many of these newer languages are based on.
