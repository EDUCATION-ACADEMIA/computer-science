It is the two or three or four letters after the dot in a filename. It also indicates what kind of file you are dealing with.

(A) File name

(B) ** File extension

(C) File size

(D) File management

EXPLANATIONS BELOW

Concept note-1: -The extension is a three-or four-letter abbreviation that signifies the file type. For example, in letter. docx the filename is letter and the extension is docx. Extensions are important because they tell your computer what icon to use for the file, and what application can open the file.

Concept note-2: -Extensions tell your computer which application created or can open the file and which icon to use for the file. For example, the docx extension tells your computer that Microsoft Word can open the file and to display a Word icon when you view it in File Explorer.

Concept note-3: -System Files Common system file extensions include . DLL, . DRV, . SYS, and .

Concept note-4: -More than one extension usually represents nested transformations, such as files. tar. gz (the . tar indicates that the file is a tar archive of one or more files, and the . gz indicates that the tar archive file is compressed with gzip).
