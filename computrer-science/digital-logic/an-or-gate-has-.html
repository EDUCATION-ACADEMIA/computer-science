An OR gate has ____

(A) ** Two inputs and one input

(B) One input and one output

(C) One input and two outputs

(D) Two inputs and two outputs

EXPLANATIONS BELOW

Concept note-1: -The 74LS32 is a Dual Input OR Gate with Quad package. It contains four independent gates each of which performs the logic OR function. Each gate has two inputs that’s why it is named Quad 2-Input OR Gate.

Concept note-2: -A logical inverter, sometimes called a NOT gate to differentiate it from other types of electronic inverter devices, has only one input.
