DIGITAL LOGIC

[SOURCES]
A NOT gate has ____

(A) Two inputs and one input

(B) ** One input and one output

(C) One input and two outputs

(D) Two inputs and two outputs

EXPLANATIONS BELOW

Concept note-1: -A logical inverter, sometimes called a NOT gate to differentiate it from other types of electronic inverter devices, has only one input. It reverses the logic state. If the input is 1, then the output is 0. If the input is 0, then the output is 1.

Concept note-2: -Rule: NOT Gates A NOT gate is a logic gate with one binary input and one binary output. The function of a NOT gate is to invert a value so that the input value is not the output.

Concept note-3: -NOT gate. The simplest gate is the NOT gate, also known as an inverter. It accepts a single input and outputs the opposite value.

Concept note-5: -For the NOT gate there is only one input (hence one input column, A), which can only have two possible values (0 and 1), so there are only two rows. As there is only one output, Y, there is only one output column.
