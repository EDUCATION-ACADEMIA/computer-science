Choose the correct word to complete this sentence:“Using ____ addressing, the operand holds the memory address of the value to be operated on”

(A) Immediate

(B) ** Direct

(C) Indexed

(D) Indirect

EXPLANATIONS BELOW

Concept note-1: -Direct Addressing Mode-In this addressing mode, The address field of the instruction contains the effective address of the operand. Only one reference to memory is required to fetch the operand. It is also called as absolute addressing mode.

Concept note-2: -Addressing Modes– The term addressing modes refers to the way in which the operand of an instruction is specified. The addressing mode specifies a rule for interpreting or modifying the address field of the instruction before the operand is actually executed.
