Tape units have ____ access.

(A) Sequential

(B) Random

(C) Associative

(D) ** Direct

EXPLANATIONS BELOW

Concept note-1: -A tape drive provides sequential access storage, unlike a hard disk drive, which provides direct access storage.

Concept note-2: -Sequential Access. The standard method for accessing files and devices is the sequential method. It can be like a cassette tape that rewinds itself after every use.

Concept note-3: -A magnetic tape is a direct access storage device.
