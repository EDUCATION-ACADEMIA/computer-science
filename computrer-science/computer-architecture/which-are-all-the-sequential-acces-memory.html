(A) Hard disk

(B) ** tape

(C) CD

EXPLANATIONS BELOW

Concept note-1: -Applications of this sequential memory access are magnetic tapes, magnetic disk and optical memories.

Concept note-2: -In computing, sequential access memory (SAM) is a class of data storage devices that read stored data in a sequence. This is in contrast to random access memory (RAM) where data can be accessed in any order. Sequential access devices are usually a form of magnetic storage or optical storage.

Concept note-3: -Explanation: Magnetic tape is exclusively a sequential access storage device.

Concept note-4: -Magnetic tape is the common sequential access storage device.
