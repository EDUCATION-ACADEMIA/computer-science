Sends data between the processor, the memory unit and the I/O devices

(A) address bus

(B) ** data bus

(C) MAR

(D) MDR

EXPLANATIONS BELOW

Concept note-1: -Detailed Solution. Statement I: Data bus sends data between the processor, memory unit, and the I/O devices. Data Bus: Data Bus carries data between the processor, the memory unit, and the input/output devices.

Concept note-2: -A data bus is used to transfer data between the memory and the CPU. The data bus is bidirectional and the address bus is unidirectional.

Concept note-3: -The system level bus (as distinct from a CPUâ€™s internal datapath busses) connects the CPU to memory and I/O devices. Typically a system level bus is designed for use as a backplane.

Concept note-4: -DMA Data Transfer Scheme Transferring bulk data using a microprocessor consumes more time. Therefore, the microprocessor performs the data transfer between an I/O device and memory using this DMA technique. For a DMA transfer, I/O devices must also contain electronic circuitry to generate control signals.

Concept note-5: -Hence the correct answer is Single BUS structure.
