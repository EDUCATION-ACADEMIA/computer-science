Which of the following is not a form of memory?

(A) Instruction Cache

(B) Instruction Register

(C) ** Instruction Opcode

(D) Both (A) and (B)

EXPLANATIONS BELOW

Concept note-1: -Here you can find the meaning of Which of the following is not a form of memory? a)instruction cacheb)instruction registerc)instruction opcoded)translation lookaside bufferCorrect answer is option ‘C’.

Concept note-2: -Correct Option: C Instruction register stores instruction, look-a-side buffer & instruction cache are also memory. But instruction opcodes are the opcodes related to an instruction which are not part of memory hierarchy. Hence (c) is correct option.

Concept note-3: -3. Which of the following is not a type of memory? Explanation: EEPROM (Electrical Erasable Programmable ROM) is not a type of memory because it is used for erasing purpose only.

Concept note-4: -The IR is used to store the instruction word. When the CPU fetches an instruction from memory, it is temporarily stored in the IR. The instruction is a binary word or code that defines a specific operation to be performed. The instruction word is also called the op code or operation code.

Concept note-5: -In computing, the instruction register (IR) or current instruction register (CIR) is the part of a CPU’s control unit that holds the instruction currently being executed or decoded.
