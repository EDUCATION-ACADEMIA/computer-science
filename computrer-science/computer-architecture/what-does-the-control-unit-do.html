What does the control unit do?

(A) ** It controls and monitors communications between the computer and any hardware attached

(B) It holds the data and programs that the CPU currently needs

(C) It holds the data and programs that the CPU currently needs

(D) All of the above

EXPLANATIONS BELOW

Concept note-1: -The control unit controls and monitors communications between the hardware attached to the computer. It controls the input and output of data, checks that signals have been delivered successfully, and makes sure that data goes to the correct place at the correct time.

Concept note-2: -A control unit, or CU, is circuitry within a computer’s processor that directs operations. It instructs the memory, logic unit, and both output and input devices of the computer on how to respond to the program’s instructions.

Concept note-3: -The large program that controls how the CPU communicates with other hardware components is the operating system.

Concept note-4: -A CU typically uses a binary decoder to convert coded instructions into timing and control signals that direct the operation of the other units (memory, arithmetic logic unit and input and output devices, etc.). Most computer resources are managed by the CU.

Concept note-5: -The Control Unit is a processor which is a hardware device.
