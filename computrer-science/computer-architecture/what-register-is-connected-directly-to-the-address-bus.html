What register is connected directly to the address bus?

(A) ** Memory Address Register

(B) Memory Data Register

(C) Program Counter

(D) Current Instruction Register

EXPLANATIONS BELOW

Concept note-1: -The MAR or address buffer also stores the address that references memory. This register directly drives the address bus and the memory address decoder in RAM or ROM.

Concept note-2: -Memory Address Register (MAR) : Contents of MAR are directly connected to the address bus.

Concept note-3: -In a computer, the memory address register (MAR) is the CPU register that either stores the memory address from which data will be fetched to the CPU registers, or the address to which data will be sent and stored via system bus.
