Which one is an example of flash memory?

(A) ** SD Card

(B) RAM

(C) ROM

(D) Cache

EXPLANATIONS BELOW

Concept note-1: -Common examples of Flash memory include: Multi-Media Card (MMC)-a Flash-based memory card standard used for solid-state storage in smartphones, digital cameras, music players, video camcorders, and personal computers. These cards store digital information such as text, pictures, audio, and video.

Concept note-2: -Flash memory is a solid-state electronic memory storage medium. It enables easy access for users to erase and reprogram data, and is used in a wide range of devices including SD cards and USB sticks, as well as more specialised industrial applications. There are two main types of flash memory – NAND and NOR.

Concept note-3: -Types of memory card formats: Consumer devices These devices include the Secure Digital card (SD card) and its smaller variant, the microSD card; Secure Digital High Capacity (SDHC) card; CompactFlash card (CF card); SmartMedia card; Memory Stick; MultiMediaCard (MMC); xD-Picture card; and USB card.

Concept note-4: -There are two types of flash memory: NOR and NAND.

Concept note-5: -A memory card is a type of storage device that is used for storing media and data files. It provides a permanent and non-volatile medium to store data and files from the attached device. Memory cards are commonly used in small, portable devices, such as cameras and phones. A memory card is also known as a flash card.
