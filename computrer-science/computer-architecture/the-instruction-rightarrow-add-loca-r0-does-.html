The instruction → Add LOCA, R0 does ____

(A) Adds the value of LOCA to R0 and stores in the temp register

(B) Adds the value of R0 to the address of LOCA

(C) ** Adds the values of both LOCA and R0 and stores it in R0

(D) Adds the value of LOCA with a value in accumulator and stores it in R0

EXPLANATIONS BELOW

Concept note-1: -The instruction-> Add LOCA, R0 does a)Adds the value of LOCA to R0 and stores in the temp registerb)Adds the value of R0 to the address of LOCAc)Adds the values of both LOCA and R0 and stores it in R0d)Adds the value of LOCA with a value in accumulator and stores it in R0Correct answer is option ‘C’.

Concept note-2: -The ADD instruction adds the values in Rn and Operand2 . The SUB instruction subtracts the value of Operand2 from the value in Rn . The RSB (Reverse SuBtract) instruction subtracts the value in Rn from the value of Operand2 . This is useful because of the wide range of options for Operand2 .
