Keyboard is an example of ____ component

(A) Display

(B) Data

(C) ** Hardware

(D) Software

EXPLANATIONS BELOW

Concept note-1: -Hardware refers to the physical elements of a computer. Also referred to as the machinery or the equipment of the computer. Examples of hardware in a computer are the keyboard, the monitor, the mouse and the processing unit However, most of a computer’s hardware cannot be seen; It’s inside the computer case.

Concept note-2: -Computer hardware includes the physical parts of a computer, such as the case, central processing unit (CPU), random access memory (RAM), monitor, mouse, keyboard, computer data storage, graphics card, sound card, speakers and motherboard.

Concept note-3: -For example, hardware components such as motherboard, RAM and CPU are internal. Other examples of hardware include output devices such as printer and monitor. Input devices such as the keyboard and mouse. Also, secondary storage devices such as CD, DVD, hard disk, etc.

Concept note-4: -Input Devices The keyboard is probably the most commonly used input device. Below are just some other types of input devices. A microphone is an example of a sound input device. Mouse, touchpad, touchscreen, multi-touch screen, pen input, motion sensor, graphics tablet, interactive smartboard, and fingerprint scanner.

Concept note-5: -When it comes to computers, the hardware is the physical and tangible part, like the monitor, keyboard, and speakers. Software is the programs and operating systems that need to be installed.
