The register which holds the address of the location to or from which data are to be transferred is called

(A) Index Register

(B) Instruction Register

(C) ** Memory Address Register

(D) Memory Data Register

EXPLANATIONS BELOW

Concept note-1: -In a computer, the memory address register (MAR) is the CPU register that either stores the memory address from which data will be fetched to the CPU registers, or the address to which data will be sent and stored via system bus.

Concept note-2: -A memory buffer register (MBR) or memory data register (MDR) is the register in a computerâ€™s CPU that stores the data being transferred to and from the immediate access storage. It contains a copy of the value in the memory location specified by the memory address register.

Concept note-3: -The MAR is a special type of register that contains the memory address of the data and instruction. The main task of the MAR is to access instruction and data from memory in the execution phase. The MAR stores the address of the memory location where the data is to be read or to be stored by the CPU.

Concept note-4: -The register which stores the address of the next instruction is called the program counter.

Concept note-5: -A processor register (CPU register) is one of a small set of data holding places that are part of the computer processor. A register may hold an instruction, a storage address, or any kind of data (such as a bit sequence or individual characters).
